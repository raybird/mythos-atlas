# 奧貢開闢人間之路 (Ogun Opens the Path to Humanity)

- **文化：** 約魯巴神話

## 故事背景

在天地創造之初，人類居住的世界被茂密的原始森林覆蓋。Obatala 已經以金鍊與白雞創造了陸地，Olodumare 也賦予了人類靈魂，但人類無法在世界上自由行動——枝葉交錯、荊棘密布，沒有道路，沒有村落，沒有可通行的路徑。眾神從天界觀看人類在密林中迷失、受苦，卻沒有哪位奧里沙願意承擔為人類開路的工作。這項任務既耗費體力又充滿危險——森林中居住著惡靈與野獸，從未有神祇涉足過那片黑暗的領域。

## 情節

**第一幕：奧貢的承諾**

奧貢 (Ogun) 是眾神中最強壯也最擅長使用鐵器的戰士。當眾神在議會中爭論誰應該降臨人間解決道路問題時，奧貢站了出來。他手持自己的砍刀 (Àdá)，腰間繫著鐵製工具，向 Olodumare 承諾他將為人類開闢一條可以安全行走的道路。其他神祇嘲笑他——一個以殺戮為樂的戰神，如何能擔當開路先鋒？但奧貢不為所動，他從天界一躍而下，墜入最深的叢林之中。

**第二幕：砍刀開路**

奧貢踏入叢林後，眼前是從未有人見過的原始森林——樹木高聳入雲，藤蔓交織如網，荊棘叢生如牆。他揮舞砍刀，一刀斬斷粗壯的樹幹，一刀劈開密集的藤蔓。森林中的惡靈與野獸被他的鐵器聲嚇退——沒有一個敢靠近這位從天而降的鐵之神。奧貢不分晝夜地砍伐開路，他的砍刀從未鈍化，因為鐵是他自身的本質。七天七夜後，奧貢在叢林中開出了一條寬闊的道路。陽光第一次穿透樹冠落入地面，人類沿著這條路走出了森林，找到了可以定居的空地。

**第三幕：道路的守護者**

道路完成後，奧貢沒有回到天界。他決定留在人間繼續保護這條道路以及依賴它的人類。他教導人類如何從礦石中提煉鐵，如何打造工具、武器與農具。人類用他教的方法砍伐樹木建造房屋，用鐵犁耕種土地，從狩獵採集進入了農耕定居的時代。奧貢因此被稱為「道路的開闢者」(Onílé Oníta) 與「文明的奠基者」。

**第四幕：尚戈的嫉妒**

奧貢的成功引起了尚戈 (Shango) 的嫉妒。尚戈認為自己是更強大的神祇，卻沒有獲得奧貢那樣的人類崇敬。兩位神祇之間的競爭幾乎爆發為萬神殿內的戰爭。奧貢以砍刀劈山示威，尚戈以雷電擊碎大地回應。最終 Olodumare 介入，將權責劃分：尚戈掌天空與正義，奧貢掌大地與創造。從此兩位神祇各自被賦予人類的不同領域——但他們的競爭關係在約魯巴神話中從未真正平息。

## 跨文化平行

- **希臘普羅米修斯 (Prometheus)**：盜火給人類，使人類獲得文明——奧貢以鐵（而非火）賦予人類文明工具；普羅米修斯因盜火受罰，奧貢則自願留下保護人類
- **中國神農氏**：教導人類耕種、醫藥與使用工具——與奧貢的「文明賦予者」角色高度重合；神農以草藥和農耕著稱，奧貢以鐵器和開路聞名
- **埃及托特 (Thoth)**：文字與文明的賦予者——兩者都承擔了「將神聖知識轉化為人類可用技術」的橋樑角色
- **北歐奧丁 (Odin)**：將符文知識帶給人類——但奧丁是智慧之神，奧貢是力量之神；兩者對人類的貢獻不同，但其「神→人」的下凡模式一致
