# 奧倫米拉與 Ifá 占卜的起源 (Orunmila and the Origin of Ifá Divination)

- **文化：** 約魯巴神話

## 故事背景

在世界創造之後，人類面對著疾病、乾旱、衝突與死亡——他們不知道如何取悅眾神，不知道什麼行為會帶來祝福或懲罰。眾神從天界俯視人類受苦，但人類缺乏與神界溝通的方法。宇宙中的奧秘本應由 Olodumare 獨享，但至高神決定賜予人類一種途徑以了解祂的意志。這個重責大任交給了奧倫米拉 (Orunmila)——他是唯一在創世時見證了全部過程的奧里沙，因此擁有知曉過去、現在與未來的全部知識。

## 情節

**第一幕：知識的守護者**

奧倫米拉最初將 Ifá 的知識完全保留在天界。他的理由是：人類還沒有準備好接受如此深奧的智慧——若他們知道未來的全貌，將無法自由地做出道德選擇。因此他將 256 個奧杜 (Odù) 封印在棕櫚堅果 (Ikin) 之中，只在天界的神聖議會中使用。人類在地面上受苦，卻沒有任何神聖指引。眾神屢次請求奧倫米拉下凡傳授知識，但他始終拒絕。

**第二幕：Oshun 的策略**

河流女神 Oshun 決定介入。她贊成人類應該獲得 Ifá 的指引，但她也明白強迫奧倫米拉是沒有用的——必須用智慧與耐心。Oshun 穿上她的黃銅飾物，前往奧倫米拉的居所。她敲響銅鈴，跳起優美的舞蹈，以蜂蜜般的聲音吟唱讚美詩。奧倫米拉被她的美麗與優雅所吸引。Oshun 不使用武力或脅迫，而是以魅力和溫柔說服奧倫米拉：知識如果不用於幫助他人是沒有意義的。

**第三幕：Ifá 降臨人間**

奧倫米拉被 Oshun 的智慧之言打動。他同意將 Ifá 知識傳授給人類，但要求人類必須通過儀式與禁忌來維持這條知識通道的純淨。他選擇了第一個 Babalawo（占卜祭司），教導他如何使用棕櫚堅果 (Ikin) 與占卜鍊 (Opele) 來與奧倫米拉溝通。每一次 Ifá 占卜都包含以下步驟：(1) 求卜者提出問題；(2) Babalawo 擲出占卜鍊或棕櫚堅果；(3) 根據落點生成一個奧杜（即某個神聖詩篇）；(4) 吟誦該奧杜的詩文，解釋眾神對問題的回應。Oshun 從此也被視為 Ifá 的共同守護者——她的黃銅物品在所有 Ifá 儀式中都會出現。

**第四幕：Ifá 的擴展**

隨著時間推移，Ifá 占卜從約魯巴地區擴散至整個西非及非裔美洲世界。在古巴的薩泰里阿 (Santería) 中，Ifá 系統與天主教聖人崇拜融合；在巴西的康東布雷 (Candomblé) 中，Ifá 的智慧被整合進非洲傳統傳承。至今 Ifá 不僅是占卜工具，更是約魯巴文化的身分象徵——2005 年聯合國教科文組織將 Ifá 占卜系統列入人類非物質文化遺產代表作名錄。每一次占卜都是人類與奧倫米拉之間的對話，也是一次對約魯巴宇宙觀的重申：命運雖然已被書寫，但人可以透過智慧與獻祭改變它的指向。

## 跨文化平行

- **中國《易經》**：以 64 卦象為基礎的占卜系統——Ifá 的 256 奧杜與易經 64 卦的功能高度相似；兩者皆將宇宙奧秘編碼為符號系統，讓人類透過隨機性與神聖意志溝通
- **希臘德爾斐神諭**：阿波羅透過女祭司皮媞亞傳達預言——與 Ifá 占卜的「神祇→祭司→求卜者」三層結構相同，但德爾斐是固定的聖地，Ifá 則不限地點
- **西藏摩拉占卜**：以骰子與經文進行的佛教占卜——與 Ifá 共享「機率與神聖文本結合」的方法論
- **伊特魯里亞肝卜 (Haruspicy)**：以動物肝臟紋理預測未來——雖然技術完全不同，但兩者都反映了「宇宙的訊息隱藏在具體物質中」的思維
