# 烏利庫米之歌：來自大海的石巨人

- **文化：** 赫梯／西臺神話（Hittite/Hurrian Mythology）

## 故事背景

烏利庫米之歌（Song of Ullikummi）是庫馬爾比神話循環（Kumarbi Cycle）的最後也是最具戲劇性的一章，記載於西臺—胡里安雙語泥板（CTH 345）中。此神話講述被推翻的父神庫馬爾比如何聯合大海之力創造一個石巨神，企圖終結風暴神 Teshub 的王權。烏利庫米的故事不僅是希臘神話中泰豐（Typhon）挑戰宙斯的直接原型，更以其「以石為身、自海而生」的獨特意象，成為古代近東神話中最令人難忘的怪物形象之一。

## 情節

### 第一幕：庫馬爾比的陰謀

庫馬爾比被其子 Teshub 推翻後，心中充滿復仇之念。他離開天界，前往深淵中的大海女神 Šertapšuruḫi 的居所。庫馬爾比向大海女神傾訴怨恨，二人合謀創造一個新的存在——一個不受眾神控制的絕對武器。庫馬爾比以石頭為材料，塑造了一個巨大的人形。他將其命名為「烏利庫米」（Ullikummi），在胡里安語中意為「庫馬爾比的毀滅者」。

> 「讓它像石柱一樣矗立，讓它像山一樣生長！讓它永遠不再被任何神祇擊敗！」

庫馬爾比將烏利庫米藏於大海的底部，放置在巨神 Upelluri（類似希臘的 Atlas，肩扛宇宙的巨人）的肩膀上。

### 第二幕：石巨人的生長

烏利庫米在大海深處以驚人的速度生長。短短十五天，他已從海底長出海面——腰部露出水面時已經高聳如山峰。他繼續生長，迅速地朝天空延伸。太陽神 Utu 在每日的天空巡行中看見了烏利庫米，急忙回報 Teshub：

> 「那個石頭怪物已經長得高不可測！它的頭觸及天界，它的陰影覆蓋了大地！」

烏利庫米生長至天界後，開始攻擊眾神的神廟與居所。諸神四處逃竄，無法抵擋這個沒有血肉、沒有感官的石頭怪物。Teshub 試圖以雷電與風暴攻擊烏利庫米，但石巨人毫無反應——他沒有耳朵可聽雷聲，沒有皮膚可感電流。

### 第三幕：Teshub 的絕望

Teshub 在連續的失敗後陷入絕望。他離開天界，孤身坐於荒野中哭泣。他的妹妹兼戰友 Šauška（沙烏什卡女神）前來安慰，試圖以舞樂引誘烏利庫米，但石巨人沒有知覺，看不見也聽不到任何誘惑——他只是一塊沒有生命的巨大岩石。

西臺泥板在此處用了一段極為感人的經文描寫 Teshub 的挫敗：

> 「風暴神 Teshub 握緊拳頭站立……他的淚水如河流般落下。『誰能抵抗這石頭怪物？誰能拯救眾神？』」

### 第四幕：伊亞的智慧

眾神之中，智慧之神伊亞（Ea，美索不達米亞起源）前往巨神 Upelluri 之處。他問 Upelluri 是否感覺到肩上有什麼重量，Upelluri 回答說右肩上有些微的刺痛感。伊亞意識到烏利庫米是依附在 Upelluri 的肩膀上（如藤壺附著於船的底部）才能從海底向上生長。

伊亞返回天界，從古代寶庫中取出一把名為「當年分離天地時所用」的切割工具——這把刀象徵創造本身的工具，具有切斷任何連結的力量。伊亞飛到海底，以這把刀將烏利庫米的腳踝與 Upelluri 的肩膀分離。石巨人從基岩上被切開後，立刻失去了持續增長的力量，頹然傾斜。

### 第五幕：最後的決戰

伊亞對 Teshub 喊道：「現在，去與烏利庫米戰鬥吧！他不再與大地相連！」Teshub 駕馭雷電戰車，率眾神向傾倒的石巨人發動總攻擊。諸神以閃電、洪水與風暴襲擊烏利庫米，最終將其擊碎——碎片落入大海形成島嶼與暗礁。庫馬爾比的最後一次造反被徹底擊敗。

## 故事結構分析

烏利庫米之歌體現了混沌戰鬥神話（Chaoskampf）的後期發展階段：

1. **舊神不滿** → 被推翻者策劃復仇
2. **創造怪物** → 怪物由非自然材料（石頭）構成
3. **怪物威脅宇宙秩序** → 生長跨越海陸空三界
4. **英雄神失敗** → 英雄陷入頹喪
5. **智慧之神提供關鍵解法** → 非蠻力，而是知識
6. **分離怪物與根基** → 宇宙創世工具再現
7. **最終勝利** → 秩序恢復

此結構與赫西俄德《神譜》中宙斯戰泰豐的敘事完全對應：泰豐從地下（Tartarus）生長而出，頭頂星辰；諸神逃往埃及；宙斯以雷霆戰鬥但被泰豐的蛇足纏繞倒下；赫爾墨斯與潘神幫助宙斯取回筋腱；最終以火山鎮壓泰豐——每一步都可在西臺版本中找到原型。

## 跨文化平行

| 元素 | 西臺 Ullikummi | 對應文化 |
|---|---|---|
| 從水界生長至天界的怪物 | 泰豐（Typhon）從 Tartarus 攀升 | 希臘 |
| 石頭為身、無感無知 | 石巨人（Golem）傳統 | 猶太 |
| 怪物由父親創造對抗兒子 | 蓋亞創造泰豐對抗宙斯 | 希臘 |
| 智慧之神提供關鍵工具 | Metis 幫助 Zeus | 希臘 |
| 分離怪物與基岩致其敗 | 安泰（Antaeus）離開地面即敗 | 希臘 |
| 巨神肩扛世界對應 | Atlas（阿特拉斯）背天 | 希臘 |
| 怪物碎成島嶼 | 巨人遺骸形成地景 | 多文化 |
