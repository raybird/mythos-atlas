# 庫馬爾比與諸王的天界

- **文化：** 西臺神話／胡里安神話（Hittite-Hurrian Mythology）

## 故事背景

Kumarbi（庫馬爾比）是西臺—胡里安神話中一位古老的神王，屬於神界權力更替敘事「天界諸王之治」（Kingship in Heaven）的核心角色。此神話記載於胡里安語—西臺語雙語泥板中（CTH 344），是地中海東部地區最重要的神系交替神話之一，對希臘的赫西俄德《神譜》產生了深遠影響。

## 情節

### 第一代：Alalu 被推翻

最初的天空之王是 Alalu。他統治九年後，他的兒子 Anu（蒼天神）起而反抗，將 Alalu 趕下王位。Alalu 逃往黑暗的冥界，Anu 登上天界寶座。

### 第二代：Kumarbi 取代 Anu

Anu 統治九年後，其子 Kumarbi 起兵叛亂。在戰鬥中，Kumarbi 咬下 Anu 的生殖器並將其吞入腹中。Anu 因此失去王權，逃往天界。Kumarbi 得勝稱王，但 Anu 向他宣告一個預言：吞下的生殖器將使 Kumarbi 懷上三位新的神靈。

### 第三代的孕育：Tarḫunna 與雙神

Kumarbi 因此懷孕，口中誕出 Teshub（西臺語 Tarḫunna）、Tigris 河神與 Aranzaḫ（底格里斯河之神）。Kumarbi 憤怒之下試圖吃掉新生的 Teshub，但 Teshub 被諸神保護並送入天界。

### 第四代：Tarḫunna 奪權

成年後的 Tarḫunna 率眾神反攻 Kumarbi，天界大戰後 Tarḫunna 擊敗父神 Kumarbi，奪取王權。此後 Tarḫunna 成為西臺萬神殿的至高之主。

### 第五代：Kumarbi 的復仇企圖

Kumarbi 不甘失敗，與大海之神（Šertapšuruḫi）合謀創造新的怪物——石怪 Ullikummi——企圖推翻 Tarḫunna。Ullikummi 是有生命的石山，從海底生長直達天界，威脅神界安全。Tarḫunna 最終借助智慧之神 Ea 的計策，以一把舊時代的切割工具將 Ullikummi 與基岩分離，使其失去能力。

## 神話結構分析

此系列神話展現了「四代神王更替」的固定模式：
1. **神王被推翻** → 2. **繼任者自殘或吞食** → 3. **新一代從殘骸中誕生** → 4. **最終推翻舊神建立新秩序**

這個精確的模式在希臘神話中完全重現：Ouranos → Kronos（割父陽具） → Zeus（推翻 Kronos） → Typhon（Kronos 最後的復仇企圖）。比較神話學者已確立 Kumarbi 神話是赫西俄德《神譜》的直接前身文本（Walcot 1966、West 1997）。

## 跨文化平行

| 西臺—胡里安 | 希臘對應 | 功能 |
|---|---|---|
| Alalu | Ouranos（烏拉諾斯） | 第一代神王 |
| Anu | Kronos（克洛諾斯） | 割父上位 |
| Kumarbi 吞 Anu 陽具 | Kronos 割 Ouranos 陽具 | 陽具暴力與權力傳承 |
| Tarḫunna（Teshub） | Zeus（宙斯） | 第三代風暴神王 |
| Ullikummi | Typhon（泰豐） | 石人蛇尾的最終怪物 |
| Ea 幫助 Teshub | Metis 幫助 Zeus | 智慧協助年輕神王 |
