# 赫達姆之歌：深淵中的海怪

- **文化：** 赫梯／西臺神話（Hittite/Hurrian Mythology）

## 故事背景

赫達姆之歌（Song of Hedammu）是庫馬爾比神話循環的另一重要篇章，記載於西臺泥板（CTH 348）中，屬於胡里安語—西臺語的史詩傳統。此神話發生於庫馬爾比創造烏利庫米之前，是庫馬爾比企圖奪回王權的第一次嘗試——若說烏利庫米是「石頭武器」，赫達姆就是「海洋武器」。此故事與美索不達米亞的提亞瑪特（Tiamat）神話、迦南的洛坦（Lotan/Leviathan）、以及希臘的斯基拉（Scylla）等海怪傳說共享深層的海洋混沌象徵。

## 情節

### 第一幕：庫馬爾比的海洋陰謀

被 Teshub 推翻後，庫馬爾比蟄伏於深淵之中。他深知以單一力量無法擊敗風暴神，決定與大海女神 Šertapšuruḫi（一說為大海之神）結盟。庫馬爾比向大海女神描述了一個計畫：創造一隻前所未有的海洋怪物——赫達姆——它將以海為家，但體型足以威脅陸地與天界。

大海女神被庫馬爾比說服。二者的結合使赫達姆誕生——一隻半蛇半魚的巨型海怪，其身軀佈滿黏液與鱗片，口中噴出毒霧，游動時引發滔天巨浪。赫達姆被安置在大海深處，以吞食魚類與海獸成長。

### 第二幕：赫達姆的肆虐

赫達姆快速成長，體型大到可以一口吞下船隻。牠從深海開始向淺海移動，所到之處引發海嘯淹沒沿岸村莊。人類的牲畜被海水沖走，田地化為鹽沼。人類向諸神獻祭求救，但赫達姆的混沌力量強大到連眾神也感到不安。

Teshub 站在山頂眺望大海，看到赫達姆攪動海洋的恐怖景象——海水變成黑色，浪頭高達天界。Teshub 知道直接的雷電攻擊對水中的赫達姆無效（水導電，反傷自身），眾神陷入口角爭執，無法決定對策。

### 第三幕：沙烏什卡的出使

正當眾神爭論不休時，智慧之神伊亞（Ea）提出一個非傳統的方案：不要以暴力對抗赫達姆，而是以魅力馴服它。他建議派遣愛與戰爭女神沙烏什卡（Šauška）前去說服赫達姆。

沙烏什卡沐浴更衣，盛裝打扮。她帶上雙侍女 Ninatta 與 Kulitta——二人手持樂器與舞衣。沙烏什卡來到赫達姆所在的深海區域，在水面上方開始跳舞。她的歌聲穿過海水，她的舞姿讓海面波光粼粼。

> 「沙烏什卡展示了她的美麗……她在赫達姆面前散發了女性的魅力。赫達姆從深海抬頭，注視著她——他的憤怒緩和了。」

### 第四幕：馴服或毀滅

泥板此處嚴重破損，現存線索顯示沙烏什卡的誘惑策略至少暫時使赫達姆平靜下來。然而赫達姆的混沌本質終究無法被長久約束——牠最終恢復破壞性，迫使 Teshub 與眾神發動總攻擊消滅牠。

赫達姆的死亡方式不詳（泥板缺失），但其結果是明確的：庫馬爾比的第一次復仇以失敗告終。然而大海女神警告庫馬爾比——下一次他應該選擇一個「沒有感官、沒有慾望」的武器——這個建議直接導致了後來石巨神烏利庫米的創造。

## 赫達姆與烏利庫米的敘事連貫性

《赫達姆之歌》與《烏利庫米之歌》在庫馬爾比神話循環中構成一個對稱的二部曲：

| 元素 | 赫達姆 | 烏利庫米 |
|---|---|---|
| 材料 | 血肉（海怪） | 石頭（無機物） |
| 弱點 | 有慾望、可被誘惑 | 無感官、無法影響 |
| 對策 | 沙烏什卡以美色誘惑 | 伊亞以智慧切割基岩 |
| 執行者 | 女神魅力 | 古老工具 |
| 結果 | 暫時平靜後終被消滅 | 被切離基岩後擊碎 |

赫達姆代表「生物性的混沌」——與希臘神話中 Zeus 對抗的 Typhon 同類；而烏利庫米代表「無機性的混沌」——在神話傳統中較為罕見，使西臺神話在比較神話學中具有獨特的貢獻。

## 跨文化平行

| 元素 | 西臺 Hedammu | 對應文化 |
|---|---|---|
| 海怪引發洪水氾濫 | Tiamat（提亞瑪特）海龍 | 巴比倫 |
| 多頭海蛇 | Lotan（洛坦／Leviathan） | 迦南 |
| 女神以魅力馴服海怪 | Ishtar 說服怪物 | 美索不達米亞 |
| 色誘作為戰略 | Judith 誘惑 Holofernes | 聖經次經 |
| 海水變黑，巨浪滔天 | Scylla（斯基拉）吞船 | 希臘 |
| 以海洋對付風暴神 | Yam（海神）對抗 Baal | 迦南 |
