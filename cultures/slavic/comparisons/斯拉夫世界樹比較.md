# 斯拉夫世界樹比較：宇宙樺樹 vs Yggdrasil vs 建木

## Comparative Analysis of the World Tree

斯拉夫宇宙論的核心意象是「世界樹」（World Tree / Axis Mundi）——一棵支撐宇宙的巨樹，根系深入冥界、樹幹貫穿人間、樹冠支撐天界。這棵樹在斯拉夫文化中通常被設想為樺樹（白樺）、橡樹或椴樹，在功能上與北歐的Yggdrasil、中國的建木、西伯利亞的薩滿樹平行。

### 結構分析

1. **三界結構的一致性**：斯拉夫世界樹將宇宙分為三層——上界（Prav/規則，眾神居住）、中界（Yav/現實，人類居住）、下界（Nav/冥界，死者居住）。這個「Prav-Yav-Nav」三層模型與Yggdrasil的Asgard-Midgard-Hel結構精確對應，以及與芬蘭的Ylinen-Keskinen-Alainen結構一致——可能反映了原始印歐或烏拉爾宇宙論的共同遺產。

2. **鴨子 vs 龍**：在Yggdrasil的根部，龍Níðhöggr啃噬樹根象徵宇宙的衰敗與循環。在斯拉夫世界樹，創世神話中潛水的鴨子處於樹旁——象徵創造與生命的源泉。北歐的守護者是破壞性的（龍吞噬），斯拉夫的守護者是創造性的（鴨子取土）。

3. **基督教化後的變形**：斯拉夫世界樹的概念在基督教化後並未消失，而是被轉化為「天堂之樹」與「生命之樹」的混合。在俄羅斯民間藝術中，世界樹以「生命樹」（Drevo Zhizni）的圖案保留在刺繡、木雕與彩蛋（Pysanky）上——樹枝對稱伸展、鳥兒棲息其上，是斯拉夫世界樹的直接裝飾化繼承。

4. **樺樹的特殊地位**：不同於Yggdrasil的白蠟樹或中國的建木，斯拉夫世界樹在民間中最常被想像為樺樹——樺樹的白色樹皮被認為是純潔的象徵，且樺樹生長迅速、生命力頑強。斯拉夫人在結婚、生子、播種時種植樺樹。樺樹也是斯拉夫最重要的儀式用樹。
