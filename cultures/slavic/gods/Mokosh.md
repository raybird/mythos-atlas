# 莫科什 (Mokosh)

- **文化：** 斯拉夫神話 (Slavic Mythology)
- **職掌：** 大地女神、紡織、生育、女性命運、潮濕與豐收

## 概述

Mokosh 是斯拉夫萬神殿中唯一確知的主要女神，其名字源自原始斯拉夫語 *mok-（潮濕、濕潤），直接關聯大地之母賦予生命的水分。她是紡織與女性手工藝的守護神，也是羊群與羊毛的保護者。Mokosh 的崇拜遍及所有斯拉夫部落——從波羅的海到巴爾幹，從第聶伯河到易北河，均有其神殿遺址。基督教化後她轉化為聖帕拉斯凱娃（St. Paraskeva Pyatnitsa），星期五成為供奉她的聖日，延續至今。

在基輔羅斯弗拉基米爾大公 980 年建立的官方萬神殿中，Mokosh 是唯一入列的女神，與 Perun、Dazbog、Stribog、Simargl 等男神並列。此地位反映了她在前基督教斯拉夫社會中無可比擬的重要性。

## 神話事蹟

Mokosh 的主要職能體現在農業週期中的角色：她掌管土壤的濕度與地下水源，決定種子是否發芽、作物能否成熟。在民間傳說中，Mokosh 以長髮婦女的形象出現，在夜晚紡織亞麻——紡錘聲被認為預兆豐收或災荒。傳統禁忌規定星期四傍晚後不可紡織或洗滌，以免冒犯女神，否則她會用紡錘刺傷違規者。

Mokosh 同時是命運的編織者，與希臘 Moirai（命運三女神）和北歐 Norns 共享「紡織命運之線」的原型。女性的生產、婚姻、子嗣等生命歷程被認為由 Mokosh 的紡錘決定。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | 蓋亞（Gaia）／命運三女神（Moirai） | 大地母神＋命運紡織者雙重功能 |
| 北歐 | Frigg（紡織雲朵、預知命運）、Norns（編織命運） | 紡織天文與人類命運之線 |
| 印度 | Prithvi（大地之母） | 賦予生命的大地女性本質 |
| 中國 | 后土娘娘 | 土地豐收與女性守護的雙重角色 |
| 埃及 | 伊西斯（Isis） | 作為妻子與母親的保護力量 |
| 羅馬 | Tellus／Ops | 大地豐饒、地下水源的女神 |

## 相關神祇

- 配偶／對應：Perun（雷神，天空原則 vs 大地原則）
- 女兒：Marzanna（死亡與冬季女神，冬季為 Mokosh 休眠期的人格化）
- 孩子：Jarilo（春季豐收之神，暗示她與 Jarilo 有母子關係）
- 盟友：Dazbog（太陽提供光照使 Mokosh 的大地可孕育作物）
- 北歐對應：Frigg 作為 Odin 之妻，Mokosh 在部分地區被視為 Perun 之妻

## 出現在

- 基輔羅斯編年史《往年紀事》（Primary Chronicle）980 年萬神殿列表
- 民間咒語與婚禮祝禱（19 世紀烏克蘭與白俄羅斯田野記錄）
- 刺繡圖案中的女性形象（雙手高舉，象徵降雨祈禱，延續至 20 世紀民俗裝飾）
- 北俄羅斯木雕上的 Mokosh 神像（常與馬匹同時出現，象徵她對農畜的雙重管轄）
