# 瑪爾札娜 (Marzanna)

- **文化：** 斯拉夫神話 (Slavic Mythology)
- **職掌：** 死亡女神、冬季嚴寒、腐朽與重生循環

## 概述

Marzanna（也作 Morana、Mara、Morena）是斯拉夫神話中掌管死亡、冬季與黑暗的女神。她的名字與原始印歐語詞根 *mer-（死亡、消逝）同源，亦與拉丁語 mors（死亡）、梵語 māra（毀滅）、立陶宛語 mirti（死亡）相通。Marzanna 在斯拉夫季節循環神話中代表黑暗的一半——從秋分到春分期間大地沉睡、生命消退的力量。

與一般「邪惡」死亡女神不同，Marzanna 的作用是辯證的：她的統治（冬季）是年度必經階段，沒有死亡就沒有重生。她的毀滅力量是 Jarilo（春季豐收之神）復興的必要前提，形成死亡→復活→再死亡的自然循環。

## 神話事蹟

Marzanna 每年與 Jarilo 結婚（象徵春夏季節的豐饒），但 Jarilo 終將死亡或返回冥界，Marzanna 恢復為哀悼與毀滅的女神。此聯姻對應 Jaanilõ（春季婚姻）儀式，在波蘭、捷克與斯洛伐克民俗中仍可追溯。Jarilo 死後 Marzanna 的淚水化為秋季的雨水與冬季的霜雪。

最重要的 Marzanna 相關儀式是波蘭、捷克、斯洛伐克與東德地區的「Marzanna 溺斃」或「Marzanna 焚燒」傳統——在春分時節，村民以稻草紮成 Marzanna 人偶，穿上傳統服飾，遊行後將之焚燒或投入河水中淹沒，象徵冬季的死亡與春天的回歸。該儀式在 Christianization 後仍頑強存續至 20 世紀，是歐洲最古老的連續性民間儀式之一。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Persephone（春季回歸）、Thanatos（死亡） | 冥界女王；冬季回歸冥界後大地枯竭 |
| 北歐 | Hel（冥界女王） | 死者領域的統治者 |
| 埃及 | Nephthys（死亡儀式守護） | 冬季、腐朽與過渡的聯想 |
| 印度 | Kālī（時母，毀滅與重生） | 毀滅力量作為創造循環的必要元素 |
| 日本 | 黃泉津大神（Yomotsu-Ōkami，伊邪那美死後化身） | 死亡女神的雙重性（生者→死者統治者） |
| 中國 | 泰山府君（冥界主宰） | 死亡權力的轉變 |
| 立陶宛 | Giltinė（死神，以白布覆面） | 共享波羅的-斯拉夫死亡概念 |

## 相關神祇

- 配偶／兒子：Jarilo（春季豐收之神，為其子或配偶）
- 母親：Mokosh（冬季為 Mokosh 休眠階段，Marzanna 為其隱晦面向）
- 對立：Jarilo／Dazbog（太陽與生命原則 vs 死亡與冬季原則）
- 相似：Perun（雷神常喚醒春雷打破冬季韁鎖）
- 北歐對應：Skaði（冬季與狩獵女神）＋ Hel（冥界女王）

## 出現在

- 波蘭中世紀編年史 *Gesta principum Polonorum*（波蘭王公事蹟，12 世紀）
- Jan Długosz 編年史（15 世紀）— 將 Marzanna 等同於羅馬女神 Ceres
- 16–19 世紀民族誌記錄（波蘭、捷克、斯洛伐克、東德、烏克蘭）
- 當代波蘭民俗節慶「Marzanna 溺斃日」（春分前後）持續流傳至今
