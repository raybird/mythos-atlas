# 雅里洛 (Jarilo)

- **文化：** 斯拉夫神話 (Slavic Mythology)
- **職掌：** 春季、豐收、男性生育力、戰爭與自然復甦

## 概述

Jarilo（也作 Jaryło、Jarovit、Gerovit）是斯拉夫神話中的春季與豐收之神，其名字來自原始斯拉夫語 *jarъ（春季、年輕、旺盛生命力），與「春季穀物」（jar）同源。Jarilo 代表年度循環中的光與熱原則——從春分開始大地回暖至秋分收割完成的全部農業週期。

Jarilo 的雙重性是其最獨特的特徵：他是和平的豐收之神（帶來穀物生長、牛羊繁殖），同時也是戰爭之神（在波拉比亞斯拉夫地區的 Jarovit 神廟中，手持金盾為戰神形象）。此雙重功能反映前基督教農業社會中「保衛收成」的現實——豐收需要武力守護。

## 神話事蹟

Jarilo 神話的核心是其出生→消失→回歸的循環敘事：

Jarilo 為 Mokosh 或 Perun 之子，出生後被帶往冥界，由 Veles（地下世界之神）撫養長大。春季時他從地下世界甦醒（象徵種子在土壤中發芽），來到人間與 Marzanna 結婚。此婚姻象徵春季的生命力量征服冬季的死亡力量。婚後大地豐饒，作物豐收。但到了秋季，Jarilo 返回冥界（或死亡），Marzanna 恢復為冬季的死亡女神，直到隔年春季循環重新開始。

此故事與 Persephone 神話形成鏡像：在希臘神話中，Persephone 每年春季從冥界返回；在斯拉夫神話中，Jarilo 從冥界返回——性別相反但結構完全一致。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Dionysus（死而復生的豐收神）、Adonis（春季回歸） | 年輕男神死亡→重生，與女性情緒哭泣相關 |
| 埃及 | Osiris（冥界之神，死而復生） | 收割後死亡→來年復甦的穀物神面貌 |
| 美索不達米亞 | Tammuz／Dumuzi（牧神，Inanna 之配偶） | 每年回歸冥界的年輕配偶 |
| 西亞 | Attis（Cybele 之配偶，春季復活） | 春季儀式中的植物神 |
| 北歐 | Freyr（豐收與生育之神） | 和平豐收之神，春季儀式中的地位 |
| 印度 | Soma（月神與植物之神） | 植物生長與生命力的神格化 |

## 相關神祇

- 母親：Mokosh（一說）；Perun（另一說，Perun 為天空原則之父）
- 養父：Veles（冥界之神，將 Jarilo 撫養長大）
- 配偶／母親：Marzanna（春季聯姻→冬季分離的循環）
- 師尊：Svarog（授予金屬盾牌，對應其戰神面向）
- 衝突對象：Negod（夏季火熱之神，部分斯拉夫傳統中與 Jarilo 爭奪季節主導權）

## 出現在

- Polabian Slavs 編年史 *Chronica Slavorum*（12 世紀 Helmold 記）— Gerovit 神殿
- Jan Długosz 編年史（15 世紀）— 標註 Jarilo 對應羅馬 Mars
- 白俄羅斯與俄羅斯田野儀式歌謠（19 世紀記錄，含 Jarilo 婚禮讚歌）
- 塞爾維亞與克羅埃西亞春季慶典「Jarilo」
