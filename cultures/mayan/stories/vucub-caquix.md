# 七金剛鸚鵡的隕落——英雄雙子之戰 (The Defeat of Vucub Caquix / Seven Macaw)

- **文化：** 馬雅神話 (Mayan Mythology)

## 故事背景

在《波波爾·烏》（*Popol Vuh*）的編年結構中，Vucub Caquix（七金剛鸚鵡 / Seven Macaw）的故事發生於創世洪水毀滅了木頭人之後、英雄雙子 Hunahpu 與 Xbalanque 誕生之前。Vucub Caquix 是創世後第一個自稱為神的生物——一隻巨大的金剛鸚鵡，其翅膀閃耀如七彩寶石，嘴如磨石般堅硬，棲息在宇宙中心的一棵大樹上。

Vucub Caquix 向世界宣稱：「我是太陽！我是月亮！我的光芒照耀一切！我是眾生的偉大創造者！」然而他不過是一個凡間的造物——一個在洪水後的虛空中趁機竊取神格的妄自尊大者。此舉激怒了天之心（Huracan）與天上的真正眾神。

## 情節

### 第一幕：英雄雙子的啟程

故事的核心是 Hunahpu 與 Xbalanque——此時的他們尚未成為後世的太陽與月亮，而是「兩位年輕的魔術師」，以吹箭（blowgun / cerbatana）為主要武器。Huracan 派遣他們去懲罰 Vucub Caquix 的狂妄。

雙子發現 Vucub Caquix 每天棲息在名為「納安·胡庫」（Nac Xuc）的大樹上，以金剛鸚鵡特有的姿態炫耀羽毛。雙子潛伏於樹下，以吹箭瞄準——Hunahpu 的精準一擊正中 Vucub Caquix 的下頜，使他從樹上墜落。

### 第二幕：斷臂之戰

Vucub Caquix 雖然身受重傷，但力量依然驚人。他猛力掙扎時抓住了 Hunahpu 的手臂，用力拉扯，竟將 Hunahpu 的整條手臂從肩膀處撕裂扯斷！

Vucub Caquix 帶著這條斷臂回到巢穴，將它掛在爐灶上燒烤，宣稱要以英雄的臂膀為食。Hunahpu 痛苦不堪，Xbalanque 不得不帶著受傷的兄弟撤退，商議對策。

### 第三幕：老人的計謀

雙子化身為兩名白髮老人——一名是骨頭醫生，一名是拔牙師——出現在 Vucub Caquix 的巢穴前。他們對疼痛難耐的 Vucub Caquix 說：「我們是專門治療痛苦的人。您的牙齒在折磨您嗎？讓我們幫您拔掉吧。」

Vucub Caquix 在劇痛中同意了。雙子先拔掉他嘴中的磨石般的牙齒——這些牙齒正是他力量與驕傲的來源。接著，他們取下他的金屬鼻子裝飾，挖去他的眼睛（如寶石般閃耀的「鏡子」），拔光他所有象徵神聖的羽毛。

當 Vucub Caquix 失去了一切外顯的華美裝飾之後，他終於認出眼前的老人就是英雄雙子。但他已無力反抗——他不再是光芒萬丈的偽神，而是一隻普通的金剛鸚鵡。Vucub Caquix 在短暫的咒罵後死去。

### 第四幕：Hunahpu 的復原

Xbalanque 找回被烤在爐灶上的斷臂，對著手臂念誦復活咒語，將骨頭重新接合回 Hunahpu 的肩膀。Hunahpu 的手臂完好如初，彷彿從未被撕裂過。

兩位英雄雙子的名聲從此遠播。他們用 Vucub Caquix 的羽毛裝飾自己的服裝——這些七彩羽毛成為他們勝利的徽章與魔力之物。

## 跨文化平行

| 文化 | 對應故事 | 共享主題 |
|------|---------|---------|
| 美索不達米亞 | Marduk 戰 Tiamat | 神以智慧（而非純力量）擊敗自稱至高的混沌怪物 |
| 埃及 | Ra／Horus 戰 Set | 偽神竊取王位，最終被真正的繼承者擊敗 |
| 希臘 | 奧德修斯 vs 波呂斐摩斯（獨眼巨人） | 英雄化身為弱者，以欺騙與偽裝擊敗強大的傲慢者 |
| 北歐 | Thor 戰 Hrungnir（霜巨人） | 英雄以武器（類似吹箭的雷神之鎚）遠距離攻擊巨人 |
| 印度 | Indra 戰 Vritra（旱魔） | 宇宙秩序中的 Chaos 偽裝為神格，被真正的神王懲罰 |
| 中國 | 后羿射日 | 十日並出（虛假太陽）被英雄以弓箭（遠距投射武器）射下 |

Vucub Caquix 的故事呈現了馬雅神話中一個關鍵的宇宙原則——真正的權力不需要自我宣稱；驕傲與自大是通向毀滅的捷徑。與西方神話中普羅米修斯式的高傲（hubris）主題類似，Vucub Caquix 的隕落警示人類與神靈都必須承認自身在宇宙秩序中的恰當位置。

## 相關主題

- 虛假的太陽（false sun）——跨文化母題：虛假的太陽或月亮必須被真正的天體取代（如中國后羿射日、埃及 Ra 的太陽船神話）
- 英雄以欺騙而非蠻力取勝——智慧勝於力量的「低弱者勝利」原型（the Trickster / the Underdog）
- 羽毛與寶石作為偽王權的象徵——外顯的華麗與內在空虛的對比
