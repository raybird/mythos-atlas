# Alux 精靈傳說

- **文化：** 馬雅神話 (Mayan Mythology)

## 故事背景

Alux（複數 Aluxo'ob）是尤卡坦半島馬雅民間信仰中的小型土地精靈，類似於世界各文化中的土地神或家宅守護精靈。Alux 的形象在馬雅傳說中非常一致：矮小（不超過膝蓋的高度）、赤身裸體、頭戴寬邊帽、皮膚古銅色、面貌如同年老的馬雅農夫。Alux 不是神祇，而是介於神靈與人類之間的中介存在——祂們是未受洗的兒童靈魂或古代馬雅人的遺留靈體。傳統上，馬雅農夫在開墾新田（milpa）時必須先建造一座微型石屋（稱為 kahtal Alux）給 Alux 居住，並在每個種植季節初期以蜂蜜、玉米或 balché 酒獻祭，以換取 Alux 的保護與土地的豐收。

## 情節

### Alux 的起源傳說

根據尤卡坦半島的口傳，Alux 是馬雅創世神在創造世界時遺留的「初稿」——祂們是介於木頭人與玉米人之間的一種過渡存在。因為不符預期，神祇沒有給 Alux 完全的靈魂，但也沒有毀滅祂們，而是允許祂們以精靈的形式存在於森林邊緣與田野之間。此傳說解釋了 Alux 的矛盾本質：祂們足夠聰明以理解人類的語言和需求，但缺乏完整的靈魂，因此無法與神祇直接溝通，必須依賴人類的獻祭為中介。

### 善意的 Alux

大多數 Alux 故事屬於幫助人類的類型。一個典型的傳說講述一個貧困的馬雅農夫在田裡工作時發現了一個被遺棄的 Alux 石屋。他將石屋清理乾淨並放置了新鮮的玉米與蜂蜜。隔天早晨，他發現自己的田地被完美翻鬆——Alux 在夜間替他完成了工作。從此農夫的收成連年豐收，成為村中最富裕的人。但他向鄰居炫耀了自己的秘密，說自己再也不需要工作——夜間他聽到 Alux 在屋外哭泣。隔天石屋空了，農夫的田恢復了貧瘠。此故事的寓意：Alux 的幫助是禮物而非權利——人類不可傲慢地依賴於祂們。

### 惡作劇的 Alux

Alux 也有搗蛋的一面——迷路、夜晚的怪聲、物品的移動、牛奶突然變酸，都被歸因於 Alux 的惡作劇。一個常見的故事：旅人在夜晚的玉米田中看到小型的燈光跳動——他跟著燈光走，結果在原地繞圈直到天亮（Alux 製造的迷路魔法）。不同於歐洲將此類現象歸咎於鬼魂或惡魔，馬雅人對 Alux 的惡作劇抱持輕鬆的態度——祂們只是「調皮但不是惡意」，如同頑皮的孩子。制衡 Alux 惡作劇的方式很簡單：在石屋門口放置十字架或將上衣翻面穿著，Alux 便會覺得迷惑而離去。

### 被遺忘的 Alux 的憤怒

若是人類忘記為 Alux 建造居所或停止獻祭，Alux 會轉向敵意——打翻農具、讓牛隻生病、使婦女在磨玉米時不斷失手。在此情況下，必須由 H-men（馬雅薩滿巫師）執行簡化的淨化儀式：以龍舌蘭酒、蜂蜜與新鮮玉米煮成的混合物噴灑在田地的四個角落，同時朗誦一段祈禱，請求 Alux 原諒人類的疏忽並恢復保護。此儀式在現代尤卡坦的馬雅社群中仍被實踐——2023 年的民族誌田野記錄在 Yaxcabá 地區仍有 H-men 執行 Alux 求和儀式。

## 跨文化平行

- **愛爾蘭／蘇格蘭：** Leprechaun — 矮小精靈、以惡作劇聞名、喜歡隱藏財寶；與 Alux 的體型與行為模式高度相似
- **日本：** 座敷童子 (Zashiki-warashi) — 家宅守護精靈，為家族帶來好運；若被忽視則離去導致衰落
- **北歐：** Tomte / Nisse — 農場守護小精靈，需要冬至獻祭（聖誕粥）；被忽視時會搗蛋
- **阿拉伯：** Jinn — 火靈，同樣具有友善、惡作劇與敵意三種態度取決於人類的對待
- **菲律賓：** Diwata — 自然精靈，掌管森林與田地，需要獻祭以維持好感
