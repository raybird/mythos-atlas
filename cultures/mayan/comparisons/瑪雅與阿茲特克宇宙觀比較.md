# 瑪雅與阿茲特克宇宙觀比較

瑪雅與阿茲特克文明同屬中美洲文化圈，共享許多宇宙觀的基本結構——如世界樹、多層宇宙、神聖曆法與獻祭核心地位——卻在具體宇宙模型、創世序列與末日論上存在顯著差異。以下從宇宙結構、創世神話與時間哲學三個層面進行比較。

## 宇宙結構比較

### 垂直宇宙

**瑪雅模型**：宇宙由十三層天堂（Oxlahuntiku）與九層冥界（Bolontiku）構成。世界樹（Yaax Cheel Cab／木棉樹）貫穿三界，樹根深入冥界，樹冠支撐天堂。四方各有色對應的 Bacab（四方神）托舉天空。

**阿茲特克模型**：同樣有十三天九地，但阿茲特克更加強調水平方位——世界由五方構成（中央＋四方位），各方由顏色、樹木、鳥類與神祇明確對應。中央為 Xihuatl（臍帶之地），是宇宙能量的交匯點。

### 水的角色

瑪雅：原水（原始之海）存在於創世之前，Tepeu 與 Gucumatz 在水中對話創造世界。阿茲特克：Tlalocan（雨神之域）是富饒的地下水世界，而 Chalchiuhtlicue（玉裙女神）掌管地表水體。兩者都視水為混沌與秩序之間的界質，但瑪雅強調水的創造性對話力量，阿茲特克則偏重水的毀滅威脅（四次太陽紀中兩次以水終結）。

## 創世敘事比較

### 太陽紀（世界時代）概念

| 維度 | 瑪雅 | 阿茲特克 |
|---|---|---|
| 時間體系 | Long Count 長紀曆（線性＋循環） | 太陽紀（五紀循環毀滅） |
| 人類之前世界 | 三次造人失敗（泥／木／玉米） | 四次太陽紀人類（巨人／不敬者／矮子） |
| 目前時代 | 第四或第五創造 | 第五太陽紀（Nahui Ollin） |
| 終結方式 | 世界持續循環，無固定末日 | 第五太陽紀將以地震終結 |
| 最初人類材料 | 玉米團混合神血 | 骨頭混合玉米與神血 |

### 造人神話

瑪雅《Popol Vuh》中，眾神三次嘗試造人——先以泥土失敗，再以木頭造出不敬人類（以洪水毀滅，變為猴子），最後以玉米團混合祖先神靈的血成功。這反映了玉米在瑪雅文明中的絕對中心地位。

阿茲特克神話中，Quetzalcoatl 前往 Mictlan（冥界）取回前世界人類的骨頭，磨成粉後混合自己的血與玉米，塑造成新人類。此版本強調死亡與犧牲在創造中的必要性——神必須流血才能賦予人類生命，直接呼應阿茲特克人祭宇宙觀的核心。

## 時間哲學比較

瑪雅時間觀本質上是「任務導向」——時間不會毀滅世界，而是通過曆法循環（260 天 Tzolk'in 與 365 天 Haab' 的 52 年曆法輪）不斷重置，宇宙在循環中持續存在。瑪雅的創世文句提到「世界在此終結」，但語意是指一個週期結束、新週期開始，而非絕對毀滅。

阿茲特克時間觀則充滿強烈的「末日焦慮」——第五太陽紀本身就是在前四次毀滅的陰影下誕生的，Cosmic 的五太陽紀神話直接告訴人們：「這個世界也將被毀滅」。這種焦慮是阿茲特克大規模人祭背後的思想基礎——必須以人類的心臟與鮮血持續供養太陽（Huitzilopochtli），才能延緩末日的到來。

## 跨文化對照表

| 比較項 | 瑪雅 | 阿茲特克 | 其他文化平行 |
|---|---|---|---|
| 宇宙層級 | 十三天九地 | 十三天九地 | 北歐九界／佛教三界 |
| 世界樹 | 木棉樹 Yaax Cheel Cab | 阿茲特克世界樹（各方向不同樹種） | 北歐 Yggdrasil／中國建木 |
| 創世材料 | 玉米＋神血 | 骨頭＋神血＋玉米 | 希臘泥土造人／中國女媧泥土 |
| 曆法循環 | Tzolk'in + Haab'（52年輪） | Tonalpohualli + Xiuhpohualli（52年輪） | 中國干支（60年輪） |
| 末日 | 週期輪迴（無毀滅性末日） | 第五太陽紀終以地震毀滅 | 北歐諸神黃昏／印度 Kali Yuga |
