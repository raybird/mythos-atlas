# 查克切爾 (Chac Chel / Goddess O)

- **文化：** 馬雅神話 (Mayan Mythology)
- **職掌：** 醫藥、助產、編織、洪水、衰老與重生

## 概述

查克切爾（Chac Chel，馬雅語意為「紅色虹彩」或「大虹」）是馬雅神話中的老年女神，在德勒斯登手抄本（Dresden Codex）與馬德里手抄本（Madrid Codex）中以女神 O（Goddess O）的身份出現。她與伊克切爾（Ix Chel，年輕月神）構成月神的雙重面向——少女與老嫗、創造與毀滅、醫治與洪水。在馬雅宇宙論中，查克切爾司掌女性生命週期中的關鍵轉折：分娩的危險、疾病的療癒、織布機的韻律，以及洪水毀滅的不可預測性。

查克切爾的外貌特徵極為鮮明——她通常被描繪為面部有皺紋、下頷鬆垂的年長女性，身著帶有交叉骨頭的裙子，手持有毒的水罐或紡錘。她頭頂常有蛇形頭飾，雙腳有時呈現鳥爪的形態——這些特徵共同傳達她跨越不同界域的力量。

## 神話事蹟

**洪水之罐**：在德勒斯登手抄本中，查克切爾傾倒手中的水罐，降下毀滅性的洪水。此洪水分別在四個方向以顏色標記——東方紅色、北方白色、西方黑色、南方黃色，對應馬雅宇宙的四象限結構。她傾倒的水罐象徵天空的裂口，雨水不再溫柔降下而是化為毀滅性的瀑布。查克切爾的洪水並非懲罰人類的道德譴責，而是在宇宙週期終結時必然發生的自然重整——此概念與北歐諸神黃昏、印度宇宙循環（Yuga）中的劫末洪水共享深層結構。

**編織與命運**：查克切爾是有名的編織女神，以紡錘與織布機交織出人類的命運之線。她編織的不僅是衣物與布匹，更是世界秩序本身的紋理。此角色使她與伊斯蘭教的時運女神曼娜特（Manat）、北歐的諾倫三女神（Norns）——烏爾德、薇兒丹蒂、詩蔻蒂——以及希臘的摩伊賴（Moirai）形成跨文化平行。編織作為命運的象徵出現在從北歐到東亞的眾多文明中——查克切爾以老年女性的身份承擔此角色，強調了命運的不可逆與接受性。

**醫藥與助產**：作為醫藥女神，查克切爾擁有草藥知識與接生的技能。馬雅產婦在分娩時向她祈禱——查克切爾被認為能讓胎兒在子宮中正確轉向，也同時掌管產後熱病的治療。她與中美洲的天然井（cenote）深度關聯——天然井中的紅色沉積物被認為是她的經血，具有治療與淨化的力量。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | 赫拉（Hera，老年面向） | 掌管婦女生育、老年女性守護者 |
| 北歐 | 諾倫三女神（Norns） | 紡織命運之線、世界樹下的編織者 |
| 印度 | 卡莉（Kali，時間與毀滅） | 毀滅與重生雙重角色、不可阻擋的力量 |
| 埃及 | 塞赫麥特（Sekhmet，戰爭與醫藥） | 同時掌管醫治與毀滅 |
| 美索不達米亞 | 提亞瑪特（Tiamat，原始鹹水） | 原始洪水女神、宇宙創新的混沌力量 |
| 中國 | 女媧（補天/洪水控制） | 女性創世力量、水界象徵中的控制與釋放 |

## 相關神祇

- **Ix Chel（伊克切爾）**：年輕月神，與 Chac Chel 為同一女神的雙重面向
- **Itzamna（伊察姆納）**：創世之神，常與 Ix Chel／Chac Chel 成對出現
- **Chaac（查克）**：雨神，與 Chac Chel 的洪水力量共同構成馬雅水循環的整體
- **Bacab（巴卡布）**：四位撐天神，象徵宇宙四方向與四季，與 Chac Chel 的四色洪水相呼應

## 出現在

- 德勒斯登手抄本（Dresden Codex）頁 40–43（洪水場景）
- 馬德里手抄本（Madrid Codex）頁 70–75（醫藥與編織場景）
- 馬雅古典期陶器繪畫（典型的神 O 形象）
- 帕倫克（Palenque）的神廟浮雕中作為月神的老年顯現
