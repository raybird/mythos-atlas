# 玉米神——尤姆·卡克斯 (Yum Kaax / God E / Tonsured Maize God)

- **文化：** 馬雅神話 (Mayan Mythology)
- **職掌：** 玉米、農業、生育、豐收、生命循環

## 概述

尤姆·卡克斯（Yum Kaax，馬雅語意為「森林之主」或「玉米之主」）是馬雅神話中最古老且最重要的神祇之一，在馬雅古典時期（約公元250–900年）的陶器、碑文與壁畫中頻繁出現。他常被稱為「剃髮玉米神」（Tonsured Maize God）或「少年玉米神」（Young Maize God），以別於阿茲特克神話中的同類神祇。

在馬雅創世史詩《波波爾·烏》（*Popol Vuh*）中，玉米不僅是主食，更是人類生命的本質——眾神以玉米麵團塑造了人類的肉體與靈魂。尤姆·卡克斯因此處於馬雅宇宙觀的核心位置：他象徵死亡與重生的永恆循環，對應玉米的播種、發芽、成熟、收割與來年再生的農業節奏。

## 神話事蹟

### 玉米的起源

在《波波爾·烏》的創世敘事中，動物們（狐狸、郊狼、鸚鵡、烏鴉）指引眾神前往「分裂之水」（Split Place / Paxil）與「黃色玉米堆積之處」（Yellow Corn Accumulation Place / Cayalá），那裡隱藏著玉米之山。眾神從此山中取來黃色與白色的玉米穗，由創世祖母 Xmucane 研磨成麵團，注入了神之血，以此塑造了第一批真正的人類——四名馬雅先祖。

尤姆·卡克斯即為此玉米之山的守護者與化身。他的形象常以玉米穗為頭飾，頭部後方延伸出玉米葉，如同待收割的玉米苞葉。他的面容年輕俊美，象徵玉米在生長季中的朝氣。

### 死亡與重生

馬雅古典時期的神話場景中，尤姆·卡克斯經常經歷死亡、被魚或龜吞噬、最終從裂開的龜殼中重生的敘事。這一組象徵直接對應玉米的農作循環——種子埋入土中（死亡）、經過黑暗的地下階段（冥界考驗）、最後破土而出（重生）。烏拉克斯圖倫（Uaxactún）的壁畫和科潘（Copán）的雕刻中反覆出現玉米神從龜殼中升起的場景，學者將其解讀為農業重生與宇宙復甦的雙重隱喻。

### 球賽中的玉米神

尤姆·卡克斯與馬雅球賽有密切關聯。在《波波爾·烏》中，英雄雙子的父親 Hun-Hunahpu 的頭顱被懸掛在葫蘆樹上，其形象常與玉米神的頭顱符號重疊。古典時期球場浮雕中，玉米神被斬首的場景直接連結到球賽的犧牲儀式——球賽的勝負隱喻農業年度的豐歉。

## 跨文化對應

| 文化 | 對應神祇／概念 | 共享特徵 |
|------|-------------|---------|
| 阿茲特克 | Centeotl（玉米神）／Chicomecoatl（七蛇女神） | 玉米之神/女神主掌農業豐收，以玉米穗為標誌 |
| 印加 | Saramama（玉米之母）／Pachamama（地母） | 玉米擬人化神靈，與大地母神共生 |
| 埃及 | Osiris（冥神/重生之神） | 被殺後復活的神，象徵尼羅河岸的植物循環 |
| 希臘 | Demeter（農耕女神）／Persephone（春夏女神） | 母女輪迴對應季節循環——Persephone 每年死而復生 |
| 美索不達米亞 | Dumuzi/Tammuz（牧神/植物神） | 每年死亡入冥界又回歸，對應植物枯萎與再生 |
| 北歐 | Freyr（豐收/和平之神） | 主掌農業、生殖與和平，以穀穗為象徵 |
| 日本 | Ukemochi（保食神） | 被殺後身體化為五穀 |

## 相關神祇

- 協作者：Xmucane（創世祖母，研磨玉米造人）、Huracan（創世風暴，指引尋玉米之山）
- 被殺者／化身：Hun-Hunahpu（英雄雙子之父，被斬首後頭顱掛於葫蘆樹，象徵玉米種子）
- 關係者：Chaac（雨神，為玉米生長提供雨水）、Kukulkan（羽蛇神，農業與知識之神）
- 敵人：Xibalba 死亡領主（以疾病與死亡威脅玉米的豐收）

## 出現在
