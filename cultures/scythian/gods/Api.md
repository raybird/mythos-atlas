# Api — 斯基泰地母

- **文化：** 斯基泰神話 (Scythian Mythology)
- **職掌：** 大地、水、生育
- **希臘對應：** Gaia
- **梵文對應：** Prithvi

## 概述

Api（或 Apia）是斯基泰神話中的地母神，同時也掌管原始水域。希羅多德將其與希臘的 Gaia（大地女神）等同，並指出斯基泰人認為 Api 是 Papaios 的妻子。在創世序列中，Api 與 Papaios 居於第二層級，僅次於 Tabiti。她既是大地之母，也是 Borysthenes（第聶伯河）河神的女兒——這使她在斯基泰人的世界觀中同時連結土地與水域。

## 神話事蹟

Api 與 Papaios 的聖婚是斯基泰創世神話的核心情節。兩者結合生育了始祖英雄 Targitaus，後者成為斯基泰人的祖先。在第一個起源神話版本中，Targitaus 被描述為「Zeus（Papaios）與 Borysthenes 河神之女（即 Api）的兒子」，將大地、河流與天父三者串聯為一體。

在第二個起源神話（蛇足女神版本）中，半蛇半人的女神可能為 Api 的另一化身——蛇足女神被稱為「這片土地的女主人」，她與 Heracles（即 Targitaus 的希臘對應）結合生了三位斯基泰分支民族的祖先。蛇足女神的蛇尾形象象徵她與大地的深層連結（蛇為地底生物），同時呼應希臘的 Echidna。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘 | Gaia | 地母、眾神之母 |
| 印度 | Prithvi | 大地女神、與天父配對 |
| 蘇美 | Ki（或 Ninhursag） | 地母、生育 |
| 埃及 | Nut（或 Geb） | 天地配對 |
| 中國 | 后土 | 大地之神 |
| 北歐 | Jörð | 大地女神、索爾之母 |

Api 的名稱在斯基泰語中與「水」同源，暗示她與原始水域的深層關聯。這呼應了斯基泰創世中水元素的核心地位——大地自水中浮現，Api 作為地母同時是水之化身。
