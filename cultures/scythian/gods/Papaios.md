# Papaios — 斯基泰天父

- **文化：** 斯基泰神話 (Scythian Mythology)
- **職掌：** 天空、天父、眾神之父
- **希臘對應：** Zeus
- **梵文對應：** Dyaus Pitr

## 概述

Papaios（古斯基泰語意為「守護者／父親」）是斯基泰神話中的天父神，地位僅次於 Tabiti。希羅多德將其與希臘的 Zeus 等同。Papaios 與地母 Api 結合，生育了眾神與人類始祖。在創世序列中，Tabiti 居首，Papaios 與 Api 居次，構成斯基泰神話的三層宇宙結構：火（聖性）→ 天父／地母（宇宙父母）→ 眾功能神。

## 神話事蹟

Papaios 最核心的神話在於他與 Api 的聖婚（hieros gamos）。天地結合生出 Targitaus——斯基泰人的始祖英雄。希羅多德記載的第一個起源神話版本中，Targitaus 為「Zeus（即 Papaios）與 Borysthenes 河神之女」所生。Idanthyrsos 王稱 Papaios 為「斯基泰人的先祖」。Papaios 亦與鳥類和格里芬（griffin）有關聯，這些象徵常見於斯基泰金器與刺青藝術中。

Papaios 也掌管誓言的神聖性——在 Tabiti 的爐火之後，對 Papaios 所發的誓言同樣具有束縛力。斯基泰人相信 Papaios 以雷電懲罰背誓者，其形象可能與草原上的雷暴相關。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘 | Zeus | 天父、雷電、眾神之王 |
| 印度 | Dyaus Pitr | 天父、與地母配對 |
| 羅馬 | Jupiter | 天父、最高神 |
| 北歐 | Odin | 眾神之父（功能對應） |
| 中國 | 昊天上帝 | 上天之主宰 |
| 波斯 | Ahura Mazda | 天界至高神 |

Papaios 的名稱詞根與印歐語系的「父親」稱謂同源，對應梵文 *pitr*、希臘文 *pater*、拉丁文 *pater*，呈現了印歐神話天父神的一致命名傳統。
