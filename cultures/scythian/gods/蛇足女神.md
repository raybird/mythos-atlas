# 蛇足女神 — 斯基泰始祖母

- **文化：** 斯基泰神話 (Scythian Mythology)
- **職掌：** 大地、始祖母、地下世界
- **希臘對應：** Echidna／Cybele

## 概述

蛇足女神（Snake-Legged Goddess，希臘文稱 *mixoparthenos*「半處女半蛇」或 *echidna diphyea*「雙形蛇妖」）是斯基泰神話中的始祖母之一，在第二個起源神話版本中佔據核心位置。她上半身為女性，下半身為蛇尾，居住在 Hylaea（林地）的洞穴中。希羅多德記載的 Heracles 版本中，她擄走 Heracles 的馬匹，要求與之結合後才歸還。三子出生後，Heracles 留下弓與腰帶考驗誰能繼承王位——僅幼子 Skythes 通過考驗，成為斯基泰人的始祖。

## 神話事蹟

蛇足女神與 Heracles 的故事是斯基泰與希臘神話融合的經典案例。Heracles 在趕往 Geryon 處執行第十項功業的途中，經過斯基泰土地，因嚴寒而披上獅皮入睡。醒來時發現馬匹失蹤，四處搜尋後在 Hylaea 的洞穴中發現了蛇足女神。她歸還馬匹的條件是與 Heracles 同寢。結合後生了三個兒子：Agathyrsos、Gelonos 與 Skythes。

Heracles 臨走前留下一把弓與一條腰帶，交代：「誰能拉開此弓、繫上腰帶，誰就留在這裡統治；做不到的則被送走。」老大老二失敗，僅 Skythes 成功，成為斯基泰人的始祖國王。這個故事的寓意在於解釋斯基泰人何以是「善射的民族」，並將希臘英雄 Heracles 納入斯基泰的血統譜系。

蛇足女神也可能等同於 Argimpasa 或 Api 的化身。在斯基泰金器與飾物中，蛇足女神的形象大量出現，是斯基泰藝術最具辨識度的神話母題。
