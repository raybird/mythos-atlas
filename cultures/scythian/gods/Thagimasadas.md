# Thagimasadas — 斯基泰海神與馬神

- **文化：** 斯基泰神話 (Scythian Mythology)
- **職掌：** 水、海、馬匹、王室祖先
- **希臘對應：** Poseidon

## 概述

Thagimasadas 是斯基泰神話中的海神與馬神，僅由「王族斯基泰人」(Royal Scythians) 所崇拜。希羅多德將其等同於希臘的 Poseidon。不同於斯基泰萬神殿中的其他七位主要神祇，Thagimasadas 是第八位、僅屬於王室部落的特殊神祇。他的存在反映了斯基泰社會的階層分化——王族斯基泰人自認擁有獨特的保護神。

## 神話事蹟

Thagimasadas 與馬的關聯對草原遊牧民族而言極具文化意義。馬是斯基泰人的命脈——提供交通、戰鬥、食物與儀式犧牲。斯基泰人以大量馬匹殉葬，帕茲里克墓葬中發現了數百匹冷凍保存的戰馬。Thagimasadas 作為馬神，可能掌管馬群的繁殖與戰馬的勇猛。

在斯基泰藝術中常見的帶翼馬圖像，可能與 Thagimasadas 有關——這與希臘的 Pegasus 意象相呼應。一些學者認為 Thagimasadas 的名字可能包含 *thag-* 詞根，與馬有關，但他同時被認為是王族祖先——王族斯基泰人自稱「Scolotoi」，傳說其祖先為 Thagimasadas 與地母之子。

|------|----------|--------|
| 希臘 | Poseidon | 海神、馬神、震地者 |

| 北歐 | Njörðr | 海神、豐饒 |
| 波斯 | Apam Napat | 水域之神 |
| 中國 | 河伯 | 河神 |
