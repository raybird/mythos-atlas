# Argimpasa — 斯基泰愛神與生育女神

- **文化：** 斯基泰神話 (Scythian Mythology)
- **職掌：** 愛、生育、王權、薩滿
- **希臘對應：** Aphrodite Urania（天上阿芙蘿黛蒂）
- **伊朗對應：** Arti（豐裕女神）

## 概述

Argimpasa（亦作 Artimpasa）是斯基泰神話中極為複雜的女神，身兼愛神、生育女神、王權守護者與薩滿信仰的主宰者。希羅多德將其等同於希臘的 Aphrodite Urania（天上阿芙蘿黛蒂），但她的功能遠超單純的愛與美——她掌管王室的正統性與薩滿祭司（Enarees）的靈性力量。她的名字可能與伊朗的豐裕女神 Arti 同源。

## 神話事蹟

Argimpasa 是蛇足女神的主要候選人之一。在斯基泰金器與刺青藝術中，經常出現一位上半身為女性、下半身為蛇尾的存在，學者普遍認為這即是 Argimpasa（或她與蛇足女神的融合）。她與 Targitaus／Heracles 結合，生了三位斯基泰分支的祖先。她的蛇尾形象象徵地下世界與豐饒的雙重力量。

Argimpasa 同時是 Enarees（意為「非男性者」）的主神。這些特殊的薩滿祭司穿著女性服裝、從事女性工作、以柳樹皮占卜——被認為是受到 Argimpasa 的感召而性別轉換。這在印歐宗教中極不尋常，反映了斯基泰社會對性別流動性的獨特態度。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘 | Aphrodite Urania | 天上之愛、生育 |
| 伊朗 | Arti | 豐裕、王權 |
| 蘇美 | Inanna/Ishtar | 愛與戰爭、性別轉換 |
| 印度 | Lakshmi | 豐饒、王權 |
| 腓尼基 | Astarte | 愛與生育女神 |

Argimpasa 的蛇足形象在斯基泰藝術中極為常見——從金梳、馬具裝飾到刺青，蛇足女神是斯基泰視覺文化中最具代表性的神話形象。
