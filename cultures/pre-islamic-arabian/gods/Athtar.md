# Athtar (عثتر)

- **文化：** 前伊斯蘭阿拉伯神話 (Pre-Islamic Arabian Mythology)
- **職掌：** 金星、灌溉、雷雨、豐收

## 概述

Athtar 是南阿拉伯（今葉門）萬神殿中最重要的一位神祇，其崇拜遍布沙巴（Saba）、卡塔班（Qataban）、哈德拉毛（Hadramawt）與馬因（Ma'in）等王國。他是金星之神——晨星與昏星均被視為其顯現。與美索不達米亞的 Ishtar 不同（女性），Athtar 為男性神，但同樣與「流動」和「灌溉」相關。他的名字在銘文中與灌溉渠道（mṣrʿ）並列出現——農民開鑿運河或修復水壩時必先向 Athtar 獻祭。南阿拉伯的王室銘文中國王經常自稱「Athtar 之僕」，顯示他與王權的緊密連結。Athtar 同時為雷雨之神——他的到來伴隨閃電與暴雨，滋潤乾涸的葉門高地。

## 神話事蹟

Athtar 神話的核心主題是「水的爭奪」。南阿拉伯創世傳說中，Athtar 曾試圖取代月神 Ilmaqah 的地位但失敗——被貶至天空成為流動的星辰，永遠不得安歇。民間傳說 Athtar 以金矛刺穿烏雲，使雨水降臨大地。他與灌溉季節的關聯最為具體——沙巴王國的大規模水利工程（如 Marib 大壩）的每一次修繕均以 Athtar 之名舉行奠基儀式。Athtar 的聖殿經常建於水壩旁，祭司在旱季進行「喚雨儀式」——手持金矛指向天空，吟唱特定咒語。南阿拉伯碑文記載，每當農作物歉收，部落長老即認定 Athtar 離棄了該地——必須以少年獻祭重新喚回他的眷顧。

## 跨文化對應

| 文化 | 對應神祇 | 關係說明 |
|------|---------|---------|
| 美索不達米亞 | Ishtar | 金星神（但 Athtar 為男性） |
| 希臘 | Zeus | 雷雨與王權 |
| 迦南 | Baal | 雷雨、豐收與季節之神 |
| 埃及 | Set | 暴風雨的破壞與創造力 |
| 印度 | Indra | 雷雨與戰爭之神、金星相關 |
