# Epona（埃波娜）

- **文化：** 凱爾特神話 (Celtic Mythology)
- **職掌：** 馬、生育、豐饒、冥界引導、騎士與旅者守護
- **象徵：** 馬（尤其母馬）、豐裕之角、穀物、小馬駒
- **聖地：** 高盧馬場、阿爾卑斯山道的聖祠

## 概述

Epona 是凱爾特世界中傳播最廣泛的女神之一——從北不列顛到多瑙河流域、從西班牙到萊茵河的廣闊領土上都有她的崇拜痕跡。她是唯一一位被羅馬人正式納入官方曆法的凱爾特神祇（每年12月18日為其節日），也是高盧-羅馬宗教融合的最典型範例。

她的名字直接來自高盧語 *epos*（馬），而印歐語系中的 *ekwo-* 同源——與希臘的 *hippos*、拉丁的 *equus*、梵文的 *aśva* 相對應。這使她在**馬女神**的跨文化語境中站在最古老的神話傳統交匯點。騎兵與馬夫特別崇拜 Epona——她的壁畫與祭壇遍布羅馬騎兵軍營。

## 神話事蹟

凱爾特神話中無 Epona 的書面敘事保存——她的崇拜主要是考古與題銘證據：

**騎士女神：** 在幾乎所有圖像中，Epona 側坐於馬背或站立於兩匹馬之間，手中持豐裕之角或穀物。她的形象直接向觀者傳達——她是**馬的馴化者與保護者**，呼應凱爾特人以馬為核心機動戰力的社會結構（高盧騎兵讓羅馬人頭疼不已）。羅馬騎兵軍團（equites）直接在軍營內設立 Epona 祭壇，請求她在征戰中保護馬匹與騎士。

**冥界引導者：** Epona 的雕像頻繁出現於墓葬——馬是凱爾特人的冥界動物，在葬禮中隨死者入葬。Epona 因此也被理解為**靈魂的引渡者**，她騎著馬帶領死者穿越邊界進入來世。此功能與希臘的赫爾墨斯（Hermes Psychopompos）引導靈魂渡冥河、埃及的阿努比斯（Anubis）引領死者參加審判平行對應。Epona 的引魂功能使得她的崇拜延伸至生死兩個世界的交界。

**馬的祖先：** 部分學者認為 Epona 的崇拜源自更古老的馬圖騰信仰——某些高盧部落（如 Aedui 與 Remi）以馬為氏族象徵，Epona 可能代表了這些部族的共同神話祖先。在此脈絡下，Epona 不只是馬的保護者，也**象徵部落的統一與認同**。

## 跨文化對應

| 凱爾特 Epona | 對應神祇 | 文化 |
|---|---|---|
| 馬女神 | 德墨忒爾（Demeter，「母馬」為其聖物） | 希臘 |
| 馬守護者 | 卡斯托與波魯克斯（Dioscuri，馴馬者） | 希臘/羅馬 |
| 靈魂引渡者 | 赫爾墨斯（Hermes Psychopompos） | 希臘 |
| 冥界引導 | 阿努比斯（Anubis） | 埃及 |
| 豐饒/穀物 | 刻瑞斯（Ceres） | 羅馬 |
| 馬與戰士守護 | 米瑟拉斯（Mithras，騎兵相關） | 波斯/羅馬 |
| Ašvieniai 神馬 | 立陶宛的雙神馬 | 波羅的海 |

## 相關神祇

- **Cernunnos：** 同為動物主宰，高盧時期並列崇拜
- **Macha：** 愛爾蘭馬女神，與 Epona 功能性疊合
- **Great Stag / 鹿神：** 與馬相對應的野生動物之神
- **Tarvos Trigaranus：** 三鶴公牛神，同列高盧動物崇拜

## 出現在

- 數百處高盧-羅馬時期的 Epona 題銘與雕像（法國、德國、英國、匈牙利、羅馬尼亞）
- 羅馬騎兵軍營中的祭壇與壁畫
- 普林尼《自然史》間接記錄
- 尤維納爾《諷刺詩》中提及 Epona 節日
