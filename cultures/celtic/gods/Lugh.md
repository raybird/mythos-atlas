# Lugh（盧格）

- **文化：** 凱爾特神話 (Celtic Mythology)
- **職掌：** 光之神、太陽、技藝、戰爭、王權、才藝
- **象徵：** 長矛（Gae Assail / 彩虹之矛）、光輝面容、渡鴉、獵犬
- **聖地：** Tara 山丘、Lughnasadh 慶典（收穫祭）

## 概述

Lugh（拉丁化：Lugus）是凱爾特神話中最多才多藝的神祇——他被稱為 Samhildánach（「所有技藝集於一身者」）。他的名號反映其全能：同時為戰士、詩人、鐵匠、醫師、巫師與工匠。他的節日 Lughnasadh（8月1日）是凱爾特四大火節之一，慶祝收穫的開始。

Lugh 的父親 Cian 來自 Tuatha Dé Danann，母親 Ethniu 來自敵對的 Fomoire 巨人族——他的跨血統身份使他成為調和兩個世界的橋樑。此模式對比希臘的赫拉克勒斯（神人混血）與波斯的魯斯塔姆（混血英雄）。

## 神話事蹟

**進入 Tara 宮廷：** Lugh 來到 Tara 的眾神宮廷尋求加入 Tuatha Dé Danann。守門人問他有何技能——他依次列出：木匠、鐵匠、戰士、豎琴師、詩人、史學家、巫師、醫師、斟酒者。守門人說「我們已有各具專精者」，Lugh 反問：「但誰能同時具備所有技藝？」寡言的國王 Nuada 聞言說：「Samhildánach——全才者——你正是我們所需要的。」Lugh 由此成為 Tuatha Dé Danann 的領袖之一。

**以彩虹之矛擊敗 Balor：** 第二次莫伊圖拉之戰中，Lugh 面對其祖父——Fomoire 的邪眼之王 Balor。Balor 的獨眼能發出毀滅一切的目光，需要四名戰士以槓桿抬起他的眼皮。Lugh 趁 Balor 眼皮抬起的瞬間以彩虹長矛（或說彈弓）射入他的邪眼，貫穿頭部擊殺。此戰奠定了 Tuatha Dé Danann 在愛爾蘭的統治。此主題對比希臘的珀爾修斯以鏡盾反射美杜莎石化之眼——兩者均為「以邪眼克制邪眼」。

**永恆之矛：** Lugh 的名矛 Gae Assail（或 Lugh's Spear）傳說中具有自己的意識，渴求戰鬥。此矛必須浸泡在罌粟安眠藥水中以防它自行出戰。在北歐神話中奧丁的永恆之槍岡格尼爾（Gungnir）擲出必中，而印度神話中因陀羅的金剛杵（Vajra）也是對應的不可抵擋之器。

**Luigh 作為眾技之神：** Lugh 在愛爾蘭神話中也是工藝與發明的守護者——他被認為發明了 Fidchell（凱爾特棋類遊戲）、馴馬、以及多種農業技術。作為太陽神，他照耀收割的時令，Lughnasadh 慶典包含賽馬、摔角與婚姻配對儀式。

## 跨文化對應

| 凱爾特 Lugh | 對應神祇 | 文化 |
|---|---|---|
| 光之神、全能技藝 | 阿波羅（Apollo） | 希臘 |
| 全才者（Samhildánach） | 奧丁（多面向智慧） | 北歐 |
| 擊殺祖父 Balor | 珀爾修斯殺美杜莎 | 希臘 |
| 長矛不可抵擋 | 岡格尼爾 / 金剛杵 | 北歐 / 印度 |
| 收穫祭 Lughnasadh | 豐收節慶跨文化對應 | — |
| 戰神與文化英雄結合 | 奎札爾科亞特爾 | 阿茲特克 |

## 相關神祇

- **Dagda：** 眾神之父，Lugh 的同陣營領袖
- **Nuada：** 銀臂國王，在前次戰爭中失去手臂
- **Balor：** Fomoire 統治者，Lugh 的祖父（兼宿敵）
- **Cian：** Lugh 之父
- **Ethniu：** Lugh 之母，Balor 之女
- **Goibniu：** 鐵匠神，為 Lugh 鍛造武器
- **Morrígan：** 戰爭女神，預言 Lugh 的勝利
