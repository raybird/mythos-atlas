# Dagda (達格達)

- **文化：** 凱爾特神話 (Celtic Mythology)
- **職掌：** 眾神之父、大地、豐饒、智慧、魔法與契約
- **象徵：** 魔法大鍋（Undry）、巨棒（Lorg Mór）、豎琴（Uaithne）
- **聖地：** Brú na Bóinne（紐格蘭奇墓）、Tara 山丘

## 概述

Dagda（古愛爾蘭語："善神"或"全能之神"）是 Tuatha Dé Danann（丹努女神之民）的領袖，也是凱爾特神話中最具權威與複雜性的神祇。他兼具戰士、德魯伊、酋長與小丑的矛盾形象——腰繫過大的連帽斗篷、手持重到需以輪車拖行的巨棒，卻又能以一根手指舉起。

Dagda 的名字本義為 *Dag(o)-dēvos*（善神），但他的「善」並非道德良善，而是指**無所不能**——他是眾神之父、大地之主、魔法與誓約的守護者。他與希臘的宙斯、北歐的奧丁、印度的因陀羅在眾神之父位置上平行對應，但其粗獷幽默的氣質更接近北歐的索爾。

## 三件神器

Dagda 擁有三件神力之物，分別對應生命、死亡與藝術：

**魔法大鍋（Undry / Coire Ansic）：** 永不空竭的大鍋，取之不盡的食物。此鍋象徵大地的豐饒與生命之源的永不枯竭——在凱爾特神話中，大鍋同時也是重生與靈感的隱喻。後世的聖杯傳說直接繼承了凱爾特魔法大鍋的意象。

**巨棒（Lorg Mór / Lorg anfaid）：** 一頭可殺死九人，另一頭可讓死者復生。這把雙面武器象徵 Dagda 生死掌管的雙重權能——他既是戰爭的主宰，也是生命的賜予者。對比印度濕婆的毀滅與創造雙面性、埃及歐西里斯的死而復生。

**豎琴（Uaithne / "四角音樂"）：** 精雕的橡木豎琴，彈奏時能改變季節的順序——「悲傷之音」使聽者哭泣、「歡笑之音」使眾人喜悅、「睡眠之音」讓世界陷入沉睡。Dagda 以此琴在戰爭前使敵人沉睡，不戰而勝。

## 神話事蹟

**與 Fomoire 的戰爭：** Tuatha Dé Danann 登陸愛爾蘭後，與原住民的 Fomoire（冥界/海洋巨人族）爆發衝突。Dagda 前往 Fomoire 陣營協議休戰，Fomoire 故意給他吃下巨量粥食羞辱他——他卻以巨棒將粥地挖出一條溝渠。此故事展示 Dagda 的力量與粗獷幽默。

**與 Morrígan 的結合：** 戰前 Dagda 在河邊遇見 Morrígan（戰爭女神）正在沐浴，二人結合後 Morrígan 預言 Tuatha Dé Danann 將戰勝 Fomoire。此結合象徵王權（Dagda）與土地/戰爭（Morrígan）的神聖婚姻。

**豎琴奪回：** Fomoire 偷走 Dagda 的豎琴藏在宴會廳的牆中。Dagda 前往呼喚：「橡木！橡木！橡木！」豎琴應聲飛出，擊殺九名 Fomoire 戰士後回到主人手中。

## 跨文化對應

| 凱爾特 Dagda | 對應神祇 | 文化 |
|---|---|---|
| 眾神之父、魔法大鍋 | 宙斯（Zeus） | 希臘 |
| 智慧犧牲、魔法 | 奧丁（Odin） | 北歐 |
| 生死雙面權能 | 濕婆（Shiva） | 印度 |
| 豐饒、大地之父 | 普里阿帕斯（Priapus）/薩圖恩（Saturn） | 羅馬 |
| 戰鎚、力量 | 索爾（Thor） | 北歐 |
| 永不空竭的大鍋 | 聚寶盆 / Amalthea 羊角 | 跨文化豐饒象徵 |

## 相關神祇

- **Morrígan：** 戰爭女神，與 Dagda 結合預言戰爭結局
- **Lugh：** 光之神，同為 Tuatha Dé Danann 領袖，Dagda 的年輕對應
- **Brigid：** 詩歌/療癒女神，Dagda 之女
- **Aengus Óg（安格斯·奥格）：** 青春/愛神，Dagda 之子
- **Ogmios：** 雄辯/文字之神，Dagda 的兄弟或對應
