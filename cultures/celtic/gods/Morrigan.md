# Morrígan（莫瑞甘）

- **文化：** 凱爾特神話 (Celtic Mythology)
- **職掌：** 戰爭、命運、死亡、主權、預言、變形
- **象徵：** 烏鴉/渡鴉、狼、洗血衣、魔法之矛
- **聖地：** 阿爾斯特戰場、River Unshin

## 概述

Morrígan（古愛爾蘭語：「幻影女王」或「恐怖女王」）是凱爾特神話中最令人畏懼的神祇——她既是戰爭女神，也是命運與死亡的化身。她不是戰爭的旁觀者，而是**戰爭本身的超自然顯現**：她以烏鴉之姿在戰場上空盤旋，選擇死者，決定勝負。

Morrígan 同時以三位一體的形象出現——與她的姐妹 Badb（暴怒/戰爭之鴉）、Macha（主權/馬）和（有時）Nemain（恐怖/戰爭狂亂）組成三重女神。這種三位一體結構與希臘的命運三女神（Moirai）和北歐的諾恩（Norns）平行對應，但在 Morrígan 的版本中，三位一體服務於戰爭與死亡的面向。

## 神話事蹟

**在河邊遇見 Dagda：** 第二次莫伊圖拉之戰前，Morrígan 在河中沐浴時被 Dagda 撞見。二人結合後，Morrígan 向 Dagda 預言 Tuatha Dé Danann 將戰勝 Fomoire，並承諾以魔法協助戰爭。她隨後在戰場上散播恐懼，削弱 Fomoire 軍隊的士氣。此為凱爾特版的「聖婚」（Hieros Gamos）——王權之神與土地/戰爭女神結合以確保勝利。

**預言 Cú Chulainn 之死：** Morrígan 曾化為少女向英雄 Cú Chulainn 求愛被拒，惱羞成怒後多次化身動物阻撓他（以鱔魚絆腳、以母狼襲擊）。最終，她預言 Cú Chulainn 的死期——在他最後一戰中，她化身為烏鴉落在他的肩頭，象徵死亡的必然降臨。此故事展示 Morrígan 性格中的執念與因果報應。

**三種化身：** Morrígan 最著名的變形是烏鴉/渡鴉，但她也能化身為：
- **蒼鷺**（象徵不祥）
- **白色母馬**（誘使戰士落馬）
- **老婦**（如 Badb 的形象，在河邊洗衣——洗的正是戰死者的血衣）
- **年輕少女**（如求愛 Cú Chulainn 時的容貌）

**洗血衣（Washer at the Ford）：** 在阿爾斯特傳說中，Morrígan 常以河邊洗衣婦的形象出現——所洗的衣物沾滿鮮血，預示即將戰死者的命運。此意象與北歐的諾恩修剪命運絲線、希臘的命運三女神紡織生命線平行對應，但 Morrígan 的版本更具鮮明的暴力美學——她不是在紡織命運，而是在清洗即將死亡者的血衣。

## 跨文化對應

| 凱爾特 Morrígan | 對應神祇 | 文化 |
|---|---|---|
| 戰爭/死亡女神、三位一體 | 命運三女神（Moirai） | 希臘 |
| 烏鴉化身、選擇死者 | 奧丁的渡鴉 / 瓦爾基麗 | 北歐 |
| 戰爭預言 | 米涅瓦（Minerva） | 羅馬 |
| 死亡與轉化 | 卡莉（Kali） | 印度 |
| 烏鴉/渡鴉戰爭象徵 | Rudra（魯陀羅） | 吠陀 |
| 河邊洗血衣 | 諾恩紡織命運線 | 北歐 |

## 相關神祇

- **Dagda：** 結合並預言戰爭結局
- **Badb：** 姐妹，暴怒女神，同樣化身烏鴉
- **Macha：** 姐妹，主權與馬女神
- **Cú Chulainn：** 被她詛咒並預言死亡的英雄
- **Aengus Óg：** 曾代 Dagda 向 Morrígan 求愛
