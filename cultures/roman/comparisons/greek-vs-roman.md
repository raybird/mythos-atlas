# 跨文化比較：羅馬與希臘神話的 Interpretatio Romana

## 羅馬—希臘對應表

| 羅馬 | 希臘 | 功能 | 關鍵差異 |
|------|------|------|---------|
| Jupiter | Zeus | 天空/雷神 | Jupiter 更國家化、較少人性弱點 |
| Juno | Hera | 婚姻/家庭 | Juno 更具政治性（Juno Regina） |
| Minerva | Athena | 智慧/工藝 | Minerva 失去戰爭面向 |
| Mars | Ares | 戰爭 | Mars 地位更高（羅馬之父） |
| Venus | Aphrodite | 愛/美 | Venus 為羅馬祖先（Venus Genetrix） |
| Diana | Artemis | 狩獵/月 | Diana 為平民崇拜核心 |
| Apollo | Apollo | 預言/藝術 | 羅馬未改名，直接引進 |
| Ceres | Demeter | 農業 | 有自己的祕密儀式（Ceres 祭） |
| Vulcan | Hephaestus | 火/鍛造 | 地位低於希臘版本 |
| Mercury | Hermes | 商業/旅行 | Mercury 強調商業面 |
| Neptune | Poseidon | 海 | Neptune 常與淡水泉祭祀 |
| Pluto | Hades | 冥界 | 羅馬 Pluto 兼具財富神格 |

## 關鍵差異

1. **國家宗教取向**：羅馬神祇更具政治與軍事功能，缺乏希臘神話的敘事性
2. **Interpretatio Romana**：羅馬征服一地後將當地神祇等同於羅馬神祇，此宗教政策促進了帝國整合
3. **祖先神話**：Venus 為 Aeneas 之母、Mars 為 Romulus 之父，將羅馬政治家族與神直接連結
