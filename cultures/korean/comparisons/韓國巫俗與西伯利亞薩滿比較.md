# 韓國巫俗與西伯利亞薩滿比較

- **文化：** 韓國神話 (Korean Mythology) / 西伯利亞神話 (Siberian Mythology)
- **類型：** 跨文化比較研究

## 前言

韓國巫俗（무속／Musok）是全球範圍內保留最完整的薩滿傳統之一——在現代韓國，仍有超過 30 萬名巫堂（무당）活躍，每年舉行數萬場「굿」（Gut／巫祭）。西伯利亞薩滿（Siberian Shamanism）則是學術界公認的薩滿原型（prototype），通古斯語系的「samān」一詞成爲全球通用的「Shaman」稱呼。兩者共享深厚的阿爾泰語系文化底蘊，但在歷史進程中走出截然不同的路徑。

下表系統比較二者的核心特徵。

## 核心比較表

| 比較維度 | 韓國巫俗 (Musok) | 西伯利亞薩滿 (Siberian) |
|---|---|---|
| 神職名稱 | 무당 (Mudang / 巫堂) | Саман (Saman / 薩滿) |
| 性別比例 | 99% 女性巫堂 | 男女兼有，依區域而異 |
| 傳召方式 | 神病 (신병) —— 被神選中後經歷精神崩潰→康復→成巫 | 多爲世襲或「被選中者」，也需經歷大病與精神試煉 |
| 宇宙觀 | 三界（天上／人間／地下） | 三界或九界（依民族而異），宇宙樹連接各層 |
| 法器 | 鈴鐺（방울）、巫刀（신칼）、扇子（부채）、銅鏡（경） | 鼓（덴그르／Shaman drum）、鈴鐺、杖 |
| 神靈溝通模式 | 神體（강신）：神靈降臨至巫堂身上 | 靈魂出竅（靈魂旅程）：薩滿靈魂離體前往精靈世界 |
| 集體儀式 | 굿（Gut）—— 包含歌舞、占卜、解怨、祈福 | Камлание (Kamlanie) —— 鼓聲吟唱，靈魂旅程 |
| 文本傳承 | 巫歌（무가）—— 口傳長篇敘事詩，部分長達數萬句 | 多爲口傳韻文，較少長篇史詩 |

## 關鍵差異分析

### 1. 性別與社會角色

韓國巫俗幾乎爲女性專有——男性巫堂稱爲「박수」（Baksoo），比例極低。這與朝鮮王朝儒敎社會將女性排除在正式宗教體系之外有關——女性轉向巫俗作爲精神出口，巫堂同時擔任心理治療師、諮商顧問與社區領袖。西伯利亞薩滿沒有明顯的性別偏好，部分民族（如楚科奇）甚至有跨性別薩滿傳統。

### 2. 降神 vs 出竅

這是兩者最根本的技術差異。韓國巫堂在굿中讓神靈「借用身體」——巫堂的意識退居次位，神靈以其口說話、跳舞、唱歌。西伯利亞薩滿則相反——薩滿的靈魂離開身體前往精靈世界（上山、入海或穿過宇宙樹），身體在此期間保持恍惚中的靜止。這兩種技術分別爲「possession trance」（附體）與「visionary trance」（幻象型恍惚）的典型代表。

### 3. 원한（恨）的化解

韓國巫俗有一個西伯利亞薩滿沒有的獨特概念——「원한」（恨／Han）——積壓未解的冤屈與遺憾。굿中的關鍵環節「풀이」（Puli／解怨）即是將亡者或生者的「恨」說出來、唱出來、化解掉。這是韓國巫俗的心理治療功能遠重於西伯利亞薩滿的原因之一。

## 共享的宇宙觀元素

兩者均信仰「三層宇宙」結構：

- **上界**（韓國：하늘／天空；西伯利亞：上天層）——居住天神與善良精靈
- **中界**（韓國：이승／此岸人間；西伯利亞：地面層）——人類與動物
- **下界**（韓國：저승／彼岸；西伯利亞：地層／死亡王國）——亡靈與地神

兩者均將宇宙軸心視爲「世界樹」或「天地柱」，薩滿／巫堂通過此軸穿越三界。韓國굿中豎立的「신대」（神竿）即爲宇宙軸的具體表現。
