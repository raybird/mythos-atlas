# 山神 (Sanshin / 山神)

- **文化：** 韓國神話 (Korean Mythology)
- **領域：** 山嶽、狩獵、守護、自然靈力
- **相關信仰：** 韓國巫俗／韓國佛敎民俗化

## 概述

山神（산신／Sanshin）是韓國最普遍且最具代表性的自然神之一，掌管山嶽、野生動物與山林的靈力。其典型形象爲一位白髮長鬚的慈祥老人，頭戴儒生斗笠，手持枴杖，身旁伴隨一隻老虎。這一形象融合了韓國薩滿信仰、本土民俗與佛敎—儒敎的層層影響。

在韓國各地，山神信仰至今仍非常活躍——幾乎每一座寺院內部或附近都設有「산신각」（山神閣），供奉單獨的山神畫像（산신도）。與中國山神多爲朝廷冊封的官僚神不同，韓國的산신保留了原始薩滿信仰的直觀性：山神是「在此山中」的精靈，而非遙遠的行政官員。山神同時也被視爲韓國國土守護者，象徵白頭山（백두산）的靈脈精神。

## 神話事蹟

### 山神與獵人

忠清道傳說：一名獵人在深山追捕老虎時迷路，天色漸暗，寒風刺骨。一位白髮老人突然出現，以枴杖敲擊岩石，岩石裂開現出溫暖洞穴。老人招待獵人過夜，次日引導他下山。臨別時老人說：「我是這座山的守護者，這隻老虎是我的坐騎。從今以後不要獵殺超過所需之物。」獵人從此只取適量獵物，並在每年秋季上山供奉。此類傳說反映韓國山神信仰中的「生態平衡」倫理——山神並非被動的守護者，而是主動懲罰貪婪的監督者。

### 山神與王室

高麗王朝時期，太祖王建在開城建國前夢見一位白髮老人騎虎而來，指示他前往松嶽（송악）建都。王建醒後遵循指示，果然在該地點發現龍脈與水源。王建在松嶽建立「산신사」（山神祠），此後每位國王即位時必須親往祭祀，延續至朝鮮王朝。

### 山神的老虎伴侶

韓國山神的經典配圖中，老虎不是坐騎而是伴侶。一則口傳解釋：遠古時期山神與老虎打賭——比誰先爬上一千座山。老虎飛奔跳躍卻因急躁錯過山頭，山神以智慧與耐心逐步攀登，最終勝出。老虎心服口服，從此追隨山神左右。此故事巧妙詮釋了山神作爲「智慧長者」而非「武力征服者」的定位。

## 跨文化對應

- **中國 山神（五嶽之神）**：中國山神被納入道教天廷體系，華山、泰山、衡山、恆山、嵩山各有專神管轄，冊封色彩濃厚。韓國山神更接近民間精靈信仰，無天廷官僚體系。
- **日本 山の神（Yama-no-Kami）**：日本山神同樣以老人形象出現，但更多與農耕循環（春下山秋上山）相關，而非永久駐山。
- **希臘 潘（Pan）**：半羊半人的山林之神，掌管野生動物與自然之聲，與山神共享「山中的守護者」的原型。
- **美洲原住民 動物之主（Gamekeepers）**：北美原住民信仰中，每種動物有一位精靈守護者規定狩獵規則，與山神維持山林平衡的角色相似。
