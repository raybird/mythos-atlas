# 三神할머니 (Samshin Halmeoni / 三神 grandmother)

- **文化：** 韓國神話 (Korean Mythology)
- **領域：** 生育、孩童保護、命運
- **相關信仰：** 韓國巫俗 (Muism / Korean Shamanism)

## 概述

三神할머니（Samshin Halmeoni，直譯爲「三位奶奶神」）是韓國巫俗信仰中掌管生育、分娩與嬰兒命運的女性神祇。與中國送子觀音或希臘生育女神不同，三神할머니並非單一神，而是三位一體的老嫗神——在韓國家庭信仰中，她們被供奉於「삼신각」（三神閣）或竈房角落的「삼신단지」（三神罈）中。

韓國傳統中，孕婦分娩時會由家中女性長輩或巫堂（무당）主持「삼신기도」（三神祈禱），在產房擺放米、海帶湯與清水作爲供品。產後嬰兒的第一件衣服稱爲「삼신저고리」（三神褂），象徵三神的守護。滿百日與週歲的「돌잔치」（週歲宴）上仍保留向三神할머니感恩的習俗。

## 神話事蹟

三神할머니的明確敍事在現存文獻中較少，其信仰主要透過儀式與口傳保留。在韓國巫俗的「부정」（不淨）觀念中，三神할머니是極度潔淨的神祇——產房必須保持絕對清潔，否則三神會離去，導致嬰兒夭折。

一則濟州島傳說記載：三位老嫗在某村莊的溪邊洗衣時，聽見嬰兒哭聲。她們沿着聲音找到一間茅草屋，屋中產婦正在痛苦分娩。三位老嫗以草藥、熱水與咒語協助產婦平安產子，從此成爲所有產婦的守護神。

另一則口傳提到：三神할머니原本是人間的助產婆，因一生接生無數卻無一失手，死後被天帝封爲生育之神。她們在每年冬至返回人間，檢查各家各戶是否保持潔淨，並在嬰兒的額頭上留下看不見的祝福印記。

## 跨文化對應

- **希臘 Moirai（命運三女神）**：三位一體掌控生命線，紡織、丈量與剪斷，與三神할머니決定嬰兒命運的功能高度對應。
- **北歐 Norns（諾倫三女神）**：在世界樹 Yggdrasil 根部分別紡織過去、現在與未來的命運之線。
- **羅馬 Parcae（帕爾凱）**：三位生育與命運女神——Nona（第九月）、Decima（第十月）、Morta（死亡）。
- **斯拉夫 Rozhanitsy（分娩女神）**：多位女性命運神在嬰兒出生時決定其一生命運。
- **中國 註生娘娘**：掌管生育與子嗣的女神，但通常爲單一神或二神組合而非三神結構。
