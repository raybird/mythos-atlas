# 蘇美創世與巴比倫創世比較 (Sumerian vs. Babylonian Creation Myths)

- **文化：** 蘇美神話 (Sumerian Mythology)

## 比較概觀

蘇美創世神話與巴比倫創世史詩《埃努瑪·埃利什》（Enūma Eliš，意為「當在高處時」）雖然共享同一地理文化脈絡，卻呈現了截然不同的創世模型與神學思想。蘇美版本以水的融合與分離為核心隱喻，巴比倫版本則以神戰、分裂屍體與王權鬥爭為主題。二者的差異不僅反映了宗教思想的演進，更揭示了蘇美與阿卡德—巴比倫政治權力結構的轉移——從城邦聯盟到帝國中央集權。

## 創世神話比較表

| 層面 | 蘇美創世 | 巴比倫《埃努瑪·埃利什》 |
|---|---|---|
| 原始狀態 | 原水 Nammu（鹹水女神）中生天地 | Apsû（淡水）與 Tiāmat（鹹水）交織 |
| 創世方式 | 自然生成：An（天）與 Ki（地）從 Nammu 生出 | 暴力分裂：Marduk 以弓箭擊殺 Tiāmat，屍體分為天地 |
| 首代神祇 | Nammu → An+Ki → Enlil（分離天地） | Apsû+Tiāmat → Lahmu+Lahamu → Anshar+Kishar → Anu → Ea → Marduk |
| 人類創造 | Enki 以黏土與神血造人取代諸神勞役 | Marduk 以叛神 Kingu 的血混黏土造人 |
| 創世動機 | 自然分化繁衍 | 秩序（Marduk）戰勝混沌（Tiāmat） |
| 至高神 | An（天神）→ Enlil（風神） | Marduk（巴比倫主神） |
| 時間 | 無特定創世時間（永恆過程） | 諸神戰爭後創世，有明確敘事序列 |
| 人類地位 | 替代諸神勞役的僕人 | 解放諸神、服務神祇的奴僕 |

## 關鍵差異分析

### 1. 創世模式：生成 vs 戰鬥

蘇美創世是典型的「生成型」（emanation）神話：世界從原水中自然地分化為天地，層層遞進如同生物繁衍。而巴比倫版本是「戰鬥型」（combat）神話：創世是暴力征服的結果。Tiāmat 代表了海洋的混沌力量，Marduk 象徵年輕一代的文明秩序——創世本身就是秩序戰勝混沌的軍事勝利。這種模式轉變與巴比倫作為帝國首都的政治崛起有直接關聯。

### 2. 神系權力更替

蘇美三部權力交替從 Nammu（原水）→ An（天）→ Enlil（大氣）的過程相對和平，近乎宇宙的自然分化。巴比倫四代權力更替（Apsû/Tiāmat → Anshar/Kishar → Anu → Ea → Marduk）則充滿陰謀與暴力——Apsû 被 Ea 殺死，Tiāmat 被 Marduk 殺死。這反映了巴比倫帝國建立過程中消滅前朝勢力的政治現實。

### 3. 人類創造的目的

兩個版本皆以黏土混合神血造人，但蘇美版本中人類是為了諸神免於勞役而被創造（較和平的替代需求），巴比倫版本則直接連結叛神 Kingu 的鮮血——人類的創造同時是對叛亂的處罰與對眾神的解放。人性中「造反」的種子來源自 Kingu 的血，為後續人類不服從神意的主題鋪路。

## 結構比較

蘇美創世的結構較為鬆散，是在多個神話片段（《Eridu 創世》、《Enki 與 Ninmah》等）中拼合。巴比倫版本則有嚴密的統一結構：第一塊泥板：諸神起源與衝突；第二至四塊：Marduk 的登場、挑戰與武器製造；第四至五塊：Marduk 擊敗 Tiāmat 後以屍體創世；第六塊：人類的創造；第七塊：Marduk 的五十個聖名讚頌。這種緊密的敘事結構使《埃努瑪·埃利什》不僅是宗教文本，更是政治宣傳。
