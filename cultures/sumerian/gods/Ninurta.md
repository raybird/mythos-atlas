# 尼努爾塔 (Ninurta) — 戰士與農耕之神

- **文化：** 蘇美神話 (Sumerian Mythology)
- **職掌：** 戰爭、農業、狩獵、洪水控制
- **象徵：** 狼牙棒 Sharur、獅首、犁
- **聖城：** 尼普爾（Nippur）的 Ešum eša 神廟
- **對應：** 阿卡德 Ninurta（同名）、巴比倫 Nabu（部分重疊）、希臘 Ares/Heracles、北歐 Thor

## 概述

尼努爾塔（Ninurta，蘇美語「大地之主」）是蘇美與美索不達米亞神話中最古老的複合神祇之一，同時掌管戰爭與農業這兩種看似衝突的領域——因為在美索不達米亞世界觀中，灌溉工程的維護如同戰鬥，農耕本身就是對自然的征服。他是恩利爾（Enlil，大氣與王權之神）的兒子，在蘇美萬神殿中地位顯赫，僅次於三大主神。他的神話經歷了從農神到戰神的轉變，反映了美索不達米亞文明從城邦到帝國的歷史進程。

## 神話事蹟

### 擊敗阿薩格（Asag）

尼努爾塔最偉大的功績是擊敗岩石與混沌之魔阿薩格（Asag）及其率領的石頭大軍。阿薩格是從地下深處崛起的原始邪惡力量，其身體為熔岩與巨石。尼努爾塔以恩利爾賜予的聖武裝——七頭狼牙棒薩魯爾（Sharur）與會說話的武器——與阿薩格在底格里斯河上游展開大戰。擊敗阿薩格後，尼努爾塔將巨量的石頭堆積成底格里斯河與幼發拉底河之間的屏障，將混沌力量封印於地下世界——此舉象徵文明灌溉系統對自然蠻荒的馴服。

### 灌溉與文明功績

〈尼努爾塔的功績〉（Lugale）史詩描述他在戰勝阿薩格後，將原來四散奔流的山洪集結成有規律的河流，修築堤壩、興建灌溉渠道，使美索不達米亞農業從被動依賴洪水轉向主動水利調控。他同時教導人類犁耕與播種技藝，因此被農民尊為保護神——一隻腳在戰場，一隻腳在田埂。

### 尼努爾塔與神諭

尼努爾塔的聖武裝薩魯爾擁有說話與預言能力，在戰鬥前會向主人提出建議。在〈Lugale〉中，薩魯爾說服尼努爾塔不要輕視阿薩格，後來又鼓勵他在戰鬥最艱苦時堅持下去——這使尼努爾塔不僅是蠻力的象徵，也代表了策略與智慧的重要。

## 跨文化對應

| 蘇美尼努爾塔 | 對應神祇 | 文化 |
|---|---|---|
| 戰神與農神 | Mars（戰）與 Saturn/Saturnus（農） | 羅馬 |
| 擊敗石魔 | Thor 戰巨人的 Mjölnir | 北歐 |
| 馴服山洪建立灌溉 | 禹疏九河治水 | 中國 |
| 神聖武器具意識 | Zeus 的雷霆/Thor 的雷錘 | 印歐 |
| 雙重領域（破壞與養育） | Shiva 毀滅與再生 | 印度 |

## 相關神祇

- **恩利爾（Enlil）**：父親，王權之神，傳授武藝
- **寧利爾（Ninlil）**：母親（或養母）
- **恩基（Enki）**：叔父，知識之神，部分功績相互呼應
- **阿薩格（Asag）**：宿敵，混沌石魔
- **伊南娜（Inanna）**：戰神的競爭者與盟友
