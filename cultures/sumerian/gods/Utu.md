# 烏圖 (Utu / Babbar / Shamash)

- **文化：** 蘇美爾神話 (Sumerian Mythology)
- **職掌：** 太陽神、正義與真理之神、占卜與預言之神、冥界夜間統治者

## 概述

Utu（蘇美爾語，阿卡德／巴比倫語稱 Shamash，意為「太陽」）是蘇美爾萬神殿中的太陽神與正義之神。他是月神 Nanna（Sin）之子、Inanna（伊南娜）的孿生兄弟。Utu 每日乘坐戰車穿越天空，從東方之門（Mount Mashu）升起，到西方之門降下。但 Utu 的獨特處在於：他降入地下後，繼續在冥界（Kur）擔任審判者與光明提供者——因此他既是日間的正義執行者，也是死者的冥界法官。

## 神話事蹟

### 每日旅程與正義之眼

Utu 每日從 Mashu 山開啟天空之門，駕著由火焰散發的戰車（由騾子或馬拉動）穿越天空。他手持「鋸齒刀」（šāšaru，象徵切割真理與謊言的光線）和「丈量繩」（象徵公平的度量）。

Utu 被稱為「看見一切者」——因為太陽的光芒照亮了人類所有的行為，沒有人可以在 Utu 面前隱藏。因此 Utu 是正義之神、契約之神、邊界之神。國王們在 Utu 面前訂立條約，受壓迫者向 Utu 呼籲尋求正義。

### Gilgamesh 的夥伴

Utu 在吉爾伽美什史詩中扮演關鍵角色——他是英雄 Gilgamesh 與 Enkidu 的保護者與盟友。當 Gilgamesh 與 Enkidu 決定前往雪松森林挑戰 Huwawa（芬巴巴）時，Gilgamesh 向 Utu 祈禱。Utu 以八個風的力量幫助他們擊敗 Huwawa。在 Enkidu 死亡後，Gilgamesh 再次向 Utu 祈禱，Utu 在冥界之門引導 Gilgamesh 前進。

### 冥界的太陽

Utu 的獨特性在於：他不僅是白天的太陽，也是夜晚的冥界統治者。當太陽在地球另一側時，Utu 在冥界（Kur）照亮死者，主持審判。死者向他祈求光明與正義。因此 Utu 具有雙重身分：日間的裁判者與夜晚的冥界之神。

### 主要崇拜中心

Utu 的兩大神廟位於 Sippar（稱為 Ebabbar，意為「白色之家」）與 Larsa。Sippar 的 Shamash 神廟是巴比倫時期最重要的正義象徵——漢摩拉比法典在此神廟中供奉。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 埃及 | Ra（太陽神，每日穿越天空與地下世界） | 每日穿越天空與冥界的旅程結構完全一致 |
| 希臘 | Helios（太陽神，全觀之眼） | 看見一切、正義之眼、駕馭戰車 |
| 印度 | Sūrya（太陽神，消除黑暗與疾病） | 正義與真理的守護者 |
| 北歐 | Sól（太陽女神——性別反轉） | 被狼追逐的太陽旅程 |
| 日本 | 天照大神（太陽女神） | 太陽隱居與回歸的神話（但性別不同） |
| 伊朗 | Mithra（密特拉，光明與契約之神） | 契約見證者、正義執行者——密特拉與 Utu/Shamash 有大量宗教傳承關係 |

## 相關神祇

- 父母：Nanna（月神）與 Ningal（月之女神）
- 姐妹：Inanna（伊南娜，金星女神）
- 配偶：有時為 Šerida（Aya，黎明女神）
- 祭品：每日祭儀包括麵包、啤酒、肉類與香料
- 象徵：太陽圓盤、鋸齒刀、丈量繩、戰車
- 崇拜城市：Sippar（北）、Larsa（南）
