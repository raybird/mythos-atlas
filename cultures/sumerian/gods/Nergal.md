# 尼爾加爾 (Nergal / Erra / Meslamtaea)

- **文化：** 蘇美爾神話 (Sumerian Mythology)
- **職掌：** 冥界之王、戰爭與瘟疫之神、毀滅與治癒的雙重力量

## 概述

Nergal（尼爾加爾，阿卡德語也作 Erra、Meslamtaea）是蘇美爾-阿卡德萬神殿中的冥界之王（後期取代 Ereshkigal 成為共同統治者）、戰爭與瘟疫之神。他的名字可能意為「偉大居所的君主」——meslam 為冥界神廟之名。Nergal 是 Enlil 之子（也有版本為 Enki 之子），在不同時期擁有截然不同甚至矛盾的形象：他是帶來瘟疫毀滅城邦的恐怖之神，也是治癒者與保護者。此矛盾體現了美索不達米亞文明對死亡與毀滅的複雜態度——毀滅者也是使生命持續的力量。

## 神話事蹟

### Nergal 與 Ereshkigal

Nergal 最著名的神話是「Nergal 與 Ereshkigal」——一個關於神祇如何成為冥界之王的故事。諸神舉辦宴會，Ereshkigal（冥后）因無法離開冥界而派使者 Namtar 代她領取食物。所有神祇向 Namtar 鞠躬——除了 Nergal。Ereshkigal 大怒，要求諸神將 Nergal 送到冥界接受懲罰。

Nergal 在 Enki 的指示下攜帶 14 個護身符前往冥界。他在冥界被 Ereshkigal 的美麗折服，兩人發生了關係。Nergal 原本計劃逃回天界，但 Ereshkigal 派遣使者威脅諸神：如果不送回 Nergal，她將釋放亡靈吞噬生者。

最終 Nergal 同意返回冥界與 Ereshkigal 結婚。Ereshkigal 說：「你是我的丈夫——我是你的妻子。我將讓你在這廣大土地上擁有統治權。」Nergal 從此成為冥界的共同統治者。

### Erra 史詩中的瘟疫之神

在巴比倫時期的 Erra 史詩中，Nergal（以 Erra 之名）是一個近乎失控的毀滅力量。他厭倦了和平，決定以瘟疫懲罰人類——即使智慧之神 Marduk 也無法阻止他。此史詩反映了美索不達米亞人對無法解釋的大規模死亡（瘟疫）的神話化——Nergal 是這些災難的人格化。

### 作為治癒之神

儘管 Nergal 是毀滅之神，他的崇拜者同時向他祈求治癒——因為賜予瘟疫者也可以收回瘟疫。在拉爾薩與庫塔（Kutha）的 Nergal 神廟中，患者前來進行治療儀式，祈求 Nergal 撤銷他們的病痛。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Hades／Ares 混合體 | 冥界之王 + 戰爭與暴力的破壞性 |
| 埃及 | Sekhmet（獅頭女神，降下瘟疫的毀滅力量） | 以疾病/瘟疫懲罰人類的兇猛女神 |
| 印度 | Rudra（樓陀羅，毀滅與治癒的雙重角色） | 同時是降下疾病與治癒疾病的神 |
| 迦南 | Mot（摩特，死亡之神與冥界之神） | 死亡與荒蕪的破壞人性質 |
| 中國 | 瘟神（如趙公明部分面向）或閻羅王 | 瘟疫的降臨者與冥界的審判者 |

## 相關神祇

- 父親：Enlil（眾神之王）或少數版本為 Enki
- 配偶：Ereshkigal（冥后，Nergal 以強制方式迎娶）
- 兒子：Tadmushtum（較次要的冥界神）
- 下屬：Namtar（命運之神，Ereshkigal 的使者）
- 象徵：獅頭、刀劍、火焰、瘟疫的隱形力量
- 崇拜中心：Kutha（庫塔，Meslam 神廟）、拉爾薩
- 星座與火星關聯：Nergal 與火星（Ṣalbatānu）綁定——兇星
