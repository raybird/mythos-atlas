# 埃雷什基伽爾 (Ereshkigal) — 冥界之后

- **文化：** 蘇美神話 (Sumerian Mythology)
- **職掌：** 冥界女王、死亡審判、黑暗主權
- **象徵：** 黑夜、冥界之眼、權杖
- **對應：** 阿卡德 Allatu、赫梯 Lelwani、希臘 Persephone、埃及 Isis/Osiris 冥界位

## 概述

埃雷什基伽爾（Ereshkigal，蘇美語「偉大地下的女士」）是蘇美神話中統治冥界庫爾（Kur）的女神。她既是死亡的化身，也是地下世界秩序的維持者——與其妹妹伊南娜（Inanna，愛情與戰爭女神）形成光明與黑暗的對比。她是安（An，天神）的女兒，伊南娜的姐姐，也是冥界七法官之首。埃雷什基伽爾的神格具有雙重性：她的領域令眾神畏懼，但她也體現了生命必須依循的循環法則——無死亡即無重生。

## 神話事蹟

### 伊南娜下冥界

埃雷什基伽爾最著名的登場是在〈伊南娜下冥界〉史詩中。當伊南娜以華服盛裝抵達冥界時，埃雷什基伽爾命守門人涅提（Neti）引導伊南娜穿過七道門，每過一門剝奪一件權力象徵。最後赤裸的伊南娜跪於埃雷什基伽爾面前，冥界七法官將伊南娜判死，屍體懸掛於壁上。然而當恩基創作的無性生物對埃雷什基伽爾的分娩之苦表示同情時，埃雷什基伽爾卸下防備，同意歸還伊南娜的屍體——此細節展現了冥后亦為承受苦難的存在，不單是冷酷的審判者。

### 婚姻與子嗣

埃雷什基伽爾最初獨自統治冥界，後與戰神涅伽爾（Nergal）結為夫妻。根據〈涅伽爾與埃雷什基伽爾〉神話，涅伽爾因侮辱冥后特使而被傳喚至冥界，二人經過戰鬥與妥協後結為夫妻。涅伽爾從此每年在人間與冥界各居半年，呼應季節神話的結構。他們的子嗣包括冥界神祇與諸多守護冥界入口的惡靈。

## 跨文化對應

| 蘇美埃雷什基伽爾 | 對應神祇 | 文化 |
|---|---|---|
| 冥界女王 | Persephone / Hekate | 希臘 |
| 死亡審判者 | Osiris（冥王） | 埃及 |
| 黑暗領域統治者 | Hel / Hela | 北歐 |
| 冥界之主（配偶） | Nergal / Hades / Pluto | 近東—歐洲 |
| 與愛情女神為姐妹 | 生命與死亡之雙生原型 | 全球神話 |

埃雷什基伽爾最獨特之處在於她是「冥界原生」的女神——她並非被貶入或劫入冥界，而是生為冥界之主。這與劫持冥界的希臘 Hades-Persephone 敘事形成根本差異。

## 相關神祇

- **伊南娜（Inanna）**：妹妹，光明與生命極致，與埃雷什基伽爾互為鏡像
- **涅伽爾（Nergal）**：配偶，戰神兼瘟疫神，共同統治冥界
- **涅提（Neti）**：冥界首席守門人
- **阿努納奇（Anunnaki）**：冥界七法官
