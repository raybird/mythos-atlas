# 恩基與寧瑪赫造人 (Enki and Ninmah Create Humanity)

- **文化：** 蘇美神話 (Sumerian Mythology)

## 故事背景

在蘇美神話中，神話故事經常處理一個根本問題：人類為何存在？〈恩基與寧瑪赫造人〉是現存最完整的蘇美人類起源神話之一，記載於烏爾第三王朝（約公元前2100年）的楔形文字泥板中。這個故事不僅解釋了人類被創造的原因（諸神的勞役替代），也藉由造人過程中的失誤與挑戰，探討了人的不完美、殘疾、社會分工與命運的多樣性。故事中恩基（Enki，水與智慧之神）與寧瑪赫（Ninmah，或寧胡爾薩格 Ninhursag，大地母神）聯手以黏土造人，過程中的細節反映了古代美索不達米亞人對人類狀況的深刻哲學思辨。

## 情節

### 第一幕：諸神的苦役

故事以一段失落的開場開始。在人類被創造之前，較低階的諸神（可能是伊吉吉 Igigi）承擔了所有體力勞動——挖掘運河、耕種土地、建造神廟。經過數千年後，這些神祇疲憊不堪，發動抗議與罷工。他們圍攻恩利爾的神廟，拒絕繼續勞動。恩利爾召集眾神會議，尋求解決方案。恩基在此時提出一個革命性的建議：創造一種生物來承擔諸神的勞役。

### 第二幕：黏土與神血的造人實驗

恩基使用智慧神恩基（自我參照般的敘事技巧）宣布他將以深淵（Abzu）中的黏土混合神之血肉來塑造人類。他邀請母神寧瑪赫協助這個造人計畫。眾神殺死一名低階神祇（在阿卡德版本中為 We-ila 或 Kingu），以其血肉與黏土混合——這樣人類體內將帶有神聖的元素，但本質上為塵土之軀，使其既有勞動能力又不會超越神界。寧瑪赫將混合好的黏土搓揉為人形，恩基賦予其生命與氣息。

### 第三幕：缺陷人類的創造

寧瑪赫與恩基在慶祝造人成功的宴會上展開挑戰：看誰能創造出有缺陷的人類並為其找到適合的社會角色。寧瑪赫首先創造了六種有殘缺的人類——一名眼盲者（成為吟遊詩人）、一名癱瘓者（成為銀匠）、一名天生無生殖能力者（成為太監宮臣）、一名無性別者、一名身患水腫者等。每個人恩基都賦予了生存的社會功能——這顯示了蘇美社會對「每個人都有其位置」的高度秩序觀念。

### 第四幕：恩基的反挑戰

輪到恩基時，他創造了一個極度虛弱且心智不成熟的生物——稱為「烏穆爾」（Umul，已枯萎者）。這個生物無法站立、無法言語、無法進食。恩基要求寧瑪赫找到這個生物在社會中的位置。寧瑪赫無論如何努力都無法使其改善，最終承認失敗。恩基的訊息很明確：有些缺陷是人類智慧無法彌補的——亦即挑戰神的權限有其邊界。

## 跨文化平行

- **女媧造人（中國）**：女媧以黃土捏人，富裕者為黃土人，貧賤者為泥漿人——同樣以泥土造人、人的差異來自造人過程的細節。
- **普羅米修斯造人（希臘）**：普羅米修斯以泥土塑造人類，雅典娜賦予氣息。普羅米修斯亦如恩基般為人類利益而與高階神對抗。
- **聖經創世（希伯來）**：耶和華以地上的塵土造亞當——同為黏土/塵土造人的基本意象。
- **Atrahasis 版本（巴比倫）**：後續的巴比倫版本將造人細節更強化：神血混合黏土，並建立人類死亡與繁衍率來控制人口。
