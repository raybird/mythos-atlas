# 渡鴉盜光與造人

- **文化：** 西伯利亞神話（楚科奇 / 科里亞克 / 勘察加）

## 故事背景

渡鴉是西伯利亞東北部神話中最活躍的角色。他不像 Buga 那般遙不可及——他就在人類身邊，貪吃、狡猾、幽默，卻又為人類帶來了文明的一切基礎。楚科奇與科里亞克的渡鴉傳說有數百個變體，構成一部「渡鴉傳奇循環」，其中盜光與造人是最核心的兩段故事。

## 情節

**盜光篇：**

在時間的起點，世界沒有光。楚科奇人的祖先在永恆的黑暗中摸索——他們狩獵靠觸覺，烹食靠直覺，寒冷刺骨。

渡鴉問人類：「為何你們生活在黑暗之中？」

人類回答：「因為光在上界，被天神的家族藏了起來。」

渡鴉展翅飛入最高層天。他變形成一片松針，落在一口井邊——天神的女兒每日來此汲水。渡鴉（松針）落入她的水桶，被她喝下腹中——渡鴉在她體內重生為嬰兒。天神的女兒撫養這個「孫子」長大——他擁有幼童的可愛外貌和渡鴉的狡猾心智。

老天神將太陽與月亮鎖在鐵箱中——每當他打開箱子取光，渡鴉（化身為孫子）便在一旁觀看。一日，老天神外出，小童打開鐵箱——刺眼光芒噴湧而出。小童叼起太陽與月亮，衝出帳篷，現出渡鴉原形，展翅飛回人間。

從此，世界有了晝夜。

**造人篇：**

渡鴉飛過新造的大地——他看見山川、森林、河流，卻不見人類。他降落在海邊，用喙拾起一枝柳條，在沙灘上劃出人的輪廓。他以泥土填入輪廓中，吹入自己的氣息——第一對人類站了起來。

渡鴉教他們：「用石片刮獸皮，用乾苔引火，用骨針縫衣。」

但最初的人類不會說話——他們只能發出咯咯聲。渡鴉又飛上天，從雷神那裡偷來一截閃電——放入人類喉嚨後，語言便誕生了。雅庫特版本的變體中，渡鴉以 Spruce 松木造男人、以 Larch 落葉松造女人——因為材料不同，男女性情有別。

## 跨文化平行

| 西伯利亞渡鴉盜光 | 對應故事 | 文化 |
|---|---|---|
| 渡鴉變形盜光 | Raven Steals the Sun | 北美特林吉特/海達 |
| 英雄藏身容器接近天神 | Maui 藏在 Hina 的圍裙中 | 波利尼西亞 |
| 以氣息造人 | Prometheus 以泥土造人 | 希臘 |
| 從天神處盜火 | Prometheus 盜火 | 希臘 |
| 偷取閃電放入喉嚨造語言 | 語言由天所賜 | 多種文化 |
| 動物造人 | 女媧以黃土造人 | 中國 |

此故事中的「藏身→變形→降生→盜取」序列在波利尼西亞的 Maui 與北美西北海岸的 Raven 傳說中幾乎完全一致，強烈暗示白令海峽兩側的共用神話基底。
