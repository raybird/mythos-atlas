# 薩滿入冥界取回病人靈魂

- **文化：** 西伯利亞神話（通古斯 / 雅庫特 / 布裏亞特共通）

## 故事背景

在西伯利亞薩滿世界觀中，疾病的主要成因並非生理問題——而是病人的靈魂被惡靈或 Erlik 扣押。薩滿（Evenki 語：saman / 雅庫特語：oyuun）的終極醫療方式是進行一場靈魂旅程——進入冥界下層，與 Erlik 或 Abasy 談判，取回被囚的靈魂。這套儀式構成西伯利亞薩滿信仰的核心實踐，影響遍及全球的薩滿式醫療傳統。

## 情節

村中一位年輕獵人突然倒下——他的身體仍然溫暖，但雙眼空洞、不認親人、不吃不喝。族人請來大薩滿（Buga 的祭司）。薩滿點燃杜松枝，以煙淨化自身與空間。他穿上綴滿鐵片與骨飾的薩滿袍——每件鐵片代表一個靈魂援助者。他戴上鹿角冠（象徵穿越三界的宇宙鹿），拿起鼓與槌。

鼓聲開始——緩慢、沉重，像心跳。薩滿在帳篷中央旋轉，靈魂漸漸脫離身體。他看見宇宙樹 Aal Luuk Mas——樹根處有三條路：左路通向 Erlik 的鐵宮、中路通向祖先之地、右路通向迷失者的迷途。

薩滿選擇左路。他沿樹根下降——越過七層地下世界，每層都有守門者。他必須唱誦每一層的通行密語——這些密語只有薩滿傳承中口耳相傳。第五層——Abasy 的狩獵場——惡靈化為狼群追趕他。薩滿以鼓聲震懾狼群，繼續前行。

第七層——鐵門前。Erlik 端坐黑鐵座上，手持鎖鏈。薩滿獻上祭品——白馬的鬃毛、馴鹿的血與九片雲母石。「歸還這個人的靈魂，」薩滿說。

Erlik 冷笑：「他犯了禁忌——在聖樺樹下解手。他的靈魂是我的。」

薩滿以鼓槌擊打鐵門三下——鼓聲穿透冥界。薩滿吟唱他老師的老師傳承下來的史詩——關於 Buga 如何創造世界、如何從水中升起大地。Erlik 終於動容——他從鐵箱中取出獵人的魂靈，一隻小白鳥。

薩滿護送小白鳥穿越七層世界回到人間。在帳篷中，他的肉體睜開眼睛——鼓聲驟停。他將小白鳥推入獵人的胸膛——獵人長嘆一口氣，睜開雙眼，喊渴。

薩滿說：「你曾去過冥界的門口——記住不要對任何人提起你在那裡看見了什麼。」
