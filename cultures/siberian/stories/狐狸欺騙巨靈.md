# 狐狸欺騙巨靈

- **文化：** 西伯利亞神話（Evenki / 雅庫特 / 布里亞特）

## 故事背景

在整個西伯利亞與北亞的口傳傳統中，狐狸（Evenki：soloki；雅庫特：sahyl）是僅次於渡鴉的經典智力型動物角色。與渡鴉的文化英雄功能不同，狐狸的故事更多屬於「弱勢者以智慧勝強權」的類型——巨靈（或森林巨怪、Erlik 的隨從）象徵蠻力與暴力，狐狸以欺騙、偽裝與言語操縱獲得勝利。這些故事也反映了西伯利亞狩獵社會中對「智力勝於武力」的價值認同。

## 情節

### Evenki 版本：狐狸與獨眼巨靈

森林中住著一隻獨眼巨靈（Tartalo 式角色，與巴斯克傳說共享同一類型）。巨靈以狩獵維生，但每次獵殺數量遠超過所需，導致森林中動物大量減少。狐狸決定教訓巨靈。狐狸走到巨靈的洞穴前，假裝跛腳哀嚎。巨靈聽到聲音，以大手抓住狐狸說：「今天的晚餐有著落了。」狐狸低聲說：「偉大的巨靈，我如此瘦小，不夠你塞牙縫。但我可以帶你前往湖邊——那裡有一頭胖馴鹿，足夠你吃一個月。」巨靈半信半疑，跟著狐狸走到湖邊。狐狸指著水中的月亮倒影說：「看見那圓形的白色物體了嗎？那是巨形白鹿的背。你只要伸手進水中就能抓住。」巨靈彎腰以大手撈月——水花四濺，月亮倒影碎裂。巨靈大怒，狐狸早已躲入柳叢中喊道：「你連自己的影子都抓不住，怎麼配當森林之王？」後續版本中，狐狸以同樣計策讓巨靈走入沼澤，巨靈沉入泥中，從此再未出現。

### 雅庫特版本：狐狸偷 Erlik 的火焰

雅庫特版本中，Erlik（冥界之王）將火焰壟斷在地下世界，人類在冰凍的北地瑟瑟發抖。狐狸自願前往冥界盜火。它跑到 Erlik 的宮殿門口，假裝受傷——Erlik 的衛兵（黑鐵犬）可憐它，將其帶入宮中。狐狸在 Erlik 的火爐旁取暖，趁衛兵不注意，將一小塊燃燒的木炭塞入尾巴的絨毛中。狐狸快速逃出冥界——但 Erlik 發現後派黑犬追趕。狐狸將木炭扔入乾草叢——草叢燃起大火，黑犬被火勢阻擋。狐狸帶著火種回到人間，人類從此有了火焰。但狐狸的尾巴尖端被燒焦——這就是為何狐狸尾巴尖端總是黑色的傳說由來。Erlik 因此詛咒狐狸：「你將永遠被人類追趕、剝皮。」這就是為何狐狸被人類獵殺的原因。

## 跨文化平行

| 文化 | 狐狸欺騙故事 | 功能 |
|---|---|---|
| 西伯利亞（Evenki） | 狐狸騙獨眼巨靈落水/沼澤 | 智力勝蠻力 |
| 西伯利亞（雅庫特）| 狐狸盜 Erlik 之火（尾尖燒黑）| 文化英雄式盜火 |
| 中國 | 狐仙／狐狸精（化人、欺騙、智慧）| 智力型變形者 |
| 日本 | きつね（狐）騙人／化人 | 變形與智慧 |
| 北美 | Coyote 騙巨人／盜火 | 騙子型文化英雄 |
| 北美 | 狐狸偷太陽 | 盜物母題 |
| 北歐 | Loki 騙巨人／騙神 | 騙子神 |
| 非洲 | Anansi 蜘蛛騙豹子／騙神 | 小型動物戰勝大型動物 |
| 希臘 | 普羅米修斯盜火（但非狐狸）| 盜火／文明起源 |

狐狸在雅庫特版本中扮演「盜火者」的角色——與普羅米修斯的神話功能一致。但雅庫特版本將盜火歸因於「狡猾而非犧牲」，反映西伯利亞狩獵文化對「智力型成功」的偏好。
