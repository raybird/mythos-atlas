# Erlik / Erlik Khan（冥界之王）

- **文化：** 西伯利亞神話（布裏亞特 / 阿爾泰 / 蒙古系）
- **職掌：** 冥界主宰、靈魂審判者、疾病與苦難之源

## 概述

Erlik（亦作 Erlik Khan、Erleg Khan）是西伯利亞—中亞神話中最強大的黑暗力量。他起源於布裏亞特與阿爾泰薩滿信仰，後經由蒙古與藏傳佛教的影響，成為跨區域的冥界主宰。Erlik 居住在冥界（Tam）的第九層——他的宮殿以黑鐵打造，門口由兩頭巨型黑犬看守。他面如黑鐵，鬍鬚由火焰組成，手持鎖鏈與鐵錘。Erlik 非同於邪惡本身——他更是宇宙二元平衡中的必然一端。薩滿在治療儀式中必須下達 Erlik 的領域，以獻祭或協商取回病人被扣押的靈魂。

## 神話事蹟

Erlik 曾是 Buga 或 Bai-Ulgan（至高神）的兄弟或助手——因驕傲與不服從被貶入下界。他試圖模仿造物主創造人類，但只能造出畸形、有毒的生物——這解釋了疾病與死亡的起源。Erlik 也是洪水神話中的關鍵角色——人類違反薩滿禁忌時，Erlik 以疾病與災難懲罰，迫使薩滿舉行儀式緩解。薩滿鼓面的圖案常繪有 Erlik 的形象——鼓槌敲擊時象徵薩滿穿越三界與 Erlik 談判。在布裏亞特版本中，Erlik 擁有二十七個隨從——每一個掌管一種疾病或不幸。

## 跨文化對應

| 西伯利亞 Erlik | 對應神祇 | 文化 |
|---|---|---|
| 冥界君王 | Hades / Pluto | 希臘 / 羅馬 |
| 靈魂審判者 | Osiris | 埃及 |
| 黑暗對立者 | Angra Mainyu | 波斯 |
| 冥界之王 | Yama | 印度 / 佛教 |
| 墮落天使 | Lucifer / Satan | 基督教 |
| 冥界之主 | Hel | 北歐 |
| 靈魂收割者 | 泰山府君 | 中國 |
| 疾病之神 | 瘟神 / 疫鬼 | 中國 |

Erlik 與印度 Yama 的相似性尤其引人注意——Yama 為第一個死亡的人類，成為冥王；Erlik 也同樣被描述為「第一個被神懲罰的存在」。佛教傳入後，Erlik 與 Yama 在布裏亞特與蒙古民間信仰中發生融合。

## 相關神祇

- **Buga / Bai-Ulgan**：至高神，Erlik 的兄弟與對立者
- **Kalau**：楚科奇的惡靈，與 Erlik 功能平行
- **Abasy**：雅庫特惡靈群，Erlik 體系的外圍
- **Shaman**：穿梭三界與 Erlik 協商者
- **Kere-tkun**：楚科奇造物主，與 Erlik 無直接關聯但為善惡對立系統中的善極

## 出現在

- 布裏亞特薩滿口傳
- 阿爾泰史詩
- 蒙古民間故事與《黃金史》
- 藏傳佛教影響後的冥界描寫
