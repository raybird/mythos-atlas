# Kere-tkun（楚科奇造物主）

- **文化：** 西伯利亞神話（楚科奇 / 科里亞克）
- **職掌：** 至高造物主、宇宙秩序、天界主宰、人類保護者

## 概述

Kere-tkun（亦作 Kere-txun、Kerejtkun）是楚科奇（Chukchi）與科里亞克（Koryak）民族信仰中的至高造物主。其名在楚科奇語中意為「至高存在」或「上面的力量」。與通古斯系統的 Buga 或阿爾泰的 Bai-Ulgan 不同，Kere-tkun 的角色更接近於一個「不干預的造物主」——祂創造了世界與人類之後，便將日常事務交由較低階的精靈與薩滿處理，只在宇宙秩序面臨崩壞時才直接介入。Kere-tkun 被描述為居住在「天上的帳篷」中的存在——祂的帳篷以白色馴鹿皮製成，以星辰為火塘，北極光為祂的神聖篝火。

## 神話事蹟

楚科奇創世神話屬於大地潛水者（Earth-Diver）類型，但主角並非鳥類而是 Kere-tkun 本人。最初只有無邊汪洋與永恆黑暗——Kere-tkun 從天界降下，在海面上行走。祂呼喚一隻潛水鳥（或海鴨）從海底取泥——前兩次失敗後，第三次成功取得少量泥土。Kere-tkun 將泥土握在手中，念誦薩滿咒語，泥土擴張為大地。祂在大地上播種，從積雪中生出馴鹿、從岩石中生出北極狐、從海水中生出海豹與海象。最後祂以泥土與風創造了第一對人類——吹入靈魂後人便有了呼吸與意識。

Kere-tkun 與 Raven（渡鴉）的關係是楚科奇神話中引人注目的母題。在部分楚科奇版本中，Raven 同時是 Kere-tkun 的兒子和對手——Raven 試圖模仿父親的創造行為卻屢屢失敗，祂的惡作劇為人類帶來了死亡與冬天。Kere-tkun 對 Raven 的懲罰與寬容反映了楚科奇人對世界缺陷的哲學態度——世界並非完美，但造物主仍在暗中守護。

科里亞克版本的 Kere-tkun 被稱為「存在於上方者」——相對於楚科奇的版本，科里亞克的 Kere-tkun 更加遙遠且抽象。科里亞克人不直接向 Kere-tkun 祈禱，而是透過自然精靈（如海神、山神與馴鹿守護靈）作為仲介。

## 跨文化對應

| 西伯利亞 Kere-tkun | 對應神祇 | 文化 |
|---|---|---|
| 至高造物主 | Buga | 通古斯 |
| 創造主 | Yryn-Ajy | 雅庫特 |
| 不干預的造物主 | Olorun / Olódùmarè | 約魯巴 |
| 天父 | Bai-Ulgan | 阿爾泰 / 蒙古 |
| 超越的造物主 | 梵天（Brahma） | 印度 |
| 遙遠的天神 | 昊天上帝 | 中國 |
| 不直接干預的至高原型 | Tengri | 蒙古 |

Kere-tkun 與約魯巴的 Olódùmarè 之間的平行最耐人尋味——兩者皆為至高造物主但較少直接介入人間事務，日常儀式以祭祀較低階神靈為主。楚科奇的薩滿在進行重要治療儀式時才直接呼喚 Kere-tkun 之名——這與約魯巴祭司只有在重大場合才誦念 Olódùmarè 的聖名完全一致。

## 相關神祇

- **Raven（渡鴉）**：Kere-tkun 之子／對手，文化英雄／騙子
- **Kalau**：楚科奇惡靈，Kere-tkun 的對立面
- **Buga**：通古斯系統的對應至高神
- **Shaman**：Kere-tkun 與人間的溝通者
- **Kere-tkun 的助手精靈（羣）**：負責執行造物主意願的低階存在
