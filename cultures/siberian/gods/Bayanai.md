# Bayanai（通古斯狩獵神）

- **文化：** 西伯利亞神話（通古斯 / Evenki / Even）
- **職掌：** 狩獵之神、森林之主、動物支配者

## 概述

Bayanai（亦作 Bayanay、Baianai）是通古斯語系民族信仰中最重要的自然神祇之一，職掌狩獵與森林動物。他並非居住在遠不可及的天界，而是直接在泰加林的深處活動。Evenki 獵人在出發狩獵前必須舉行儀式向 Bayanai 祈禱——若 Bayanai 高興，便將馴鹿或貂鼠驅趕至獵人的箭路；若他不悅，森林將空空如也。Bayanai 的形象通常為高大的老年獵人，身穿馴鹿皮衣，手持弓或長矛，有時以巨熊形態出現。他與 Buga（至高神）的關係類似於執行者與立法者——Buga 制定宇宙秩序，Bayanai 在森林中維護此秩序。

## 神話事蹟

Evenki 傳說中，Bayanai 與人類獵人曾有直接互動。一名獵人因連續七天狩獵無獲而瀕臨餓死——他在樺樹林深處祈禱，Bayanai 以馴鹿形態出現，引導獵人至獸群聚集處。獵人獲救後欲獻祭卻發現馴鹿已消失——Bayanai 不接受以自身形態動物的獻祭。另一廣泛流傳的故事中，獵人因貪婪殺死過多動物，Bayanai 化為巨熊摧毀他的帳篷，奪走他的火種與弓，並告誡：「森林取你所需，而非你所要。」從此 Evenki 形成狩獵不可超過所需量的禁忌。Bayanai 也掌管動物的轉世——他將馴鹿的靈魂收集在森林深處的聖樹洞中，待新生命降生時重新釋放。

## 跨文化對應

| 西伯利亞 Bayanai | 對應神祇 | 文化 |
|---|---|---|
| 狩獵神 / 森林之主 | Artio (Arduinna) | 凱爾特 |
| 動物之主 | Lord of the Animals | 北美原住民 |
| 狩獵女神 | Artemis / Diana | 希臘 / 羅馬 |
| 山神 / 狩獵神 | 后土 / 山君 | 中國 |
| 森林守護者 | Basajaun | 巴斯克 |
| 馴鹿支配者 | 無精確對應 | 北歐（接近 Ullr）|

Bayanai 與希臘 Artemis 最顯著的差異在於性別——Bayanai 為男性神祇，反映西伯利亞狩獵社會以男性獵人為主體的結構。他與北美原住民的「動物之主」（Animal Master）概念最為接近——二者都要求獵人遵守禁忌、不可濫殺，且以動物形態直接監督人類的狩獵行為。

## 相關神祇

- **Buga**：至高神，Bayanai 在森林中的行動為 Buga 的體現
- **Umai**：生育女神，Bayani 的對應——後者控制「取」，前者控制「生」
- **Tulujn**：雷神，與 Bayanai 同為 Buga 之下的執行神
- **Erlik**：冥界之王，與 Bayanai 分別掌管生死兩界的動物靈魂
- **Shaman**：薩滿為 Bayanai 與獵人間的中介者
