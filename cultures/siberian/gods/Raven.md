# Raven / 渡鴉（文化英雄與造物者）

- **文化：** 西伯利亞神話（楚科奇 / 科里亞克 / 勘察加）
- **職掌：** 文化英雄、火種盜取者、光之使者、人類創造者

## 概述

渡鴉（楚科奇語：Kurkul / Quurkil；科里亞克語：Quikinna'qu）是西伯利亞東北部（楚科奇、科里亞克、伊捷爾緬、勘察加）神話中最活躍的文化英雄。與通古斯與雅庫特的抽象至高神不同，渡鴉以具體的動物形態出現——他既是巨大烏鴉也是人形（可自由變形），象徵跨越物種界線的轉化能力。西伯利亞的渡鴉與北美西北海岸原住民的 Raven 共享令人震驚的相似主題——以至於學者普遍認為二者在末次冰期白令海陸橋時期曾有共同起源。

## 神話事蹟

渡鴉的事蹟構成楚科奇—科里亞克神話體系的主軸：(1) 盜光——最初世界一片漆黑，渡鴉飛入最高天界偷取太陽與月亮，藏於喙中帶回人間，從此人類有了光明。(2) 造人——渡鴉以柳枝與泥土捏出第一對人類，教他們生火、搭帳篷、狩獵海豹與馴鹿。(3) 盜火——渡鴉從天上偷取火種，藏在鵝卵石中帶回人間。(4) 馴服自然——渡鴉將過大的湖泊壓小、將過高的山脈踩低，使大地適合人類居住。(5) 與 Kalau 對抗——渡鴉在楚科奇版本中與惡靈 Kalau 鬥智鬥力，防止世界被毀滅。渡鴉的故事常帶幽默與惡作劇色彩——他貪吃、好色、說謊，但最終總為人類帶來益處。

## 跨文化對應

| 西伯利亞 Raven | 對應角色 | 文化 |
|---|---|---|
| 渡鴉文化英雄 | Raven | 北美西北海岸（特林吉特/海達） |
| 文化英雄/火盜 | Prometheus | 希臘 |
| 文化英雄/變形 | Coyote | 北美原住民（平原/西南） |
| 盜光者 | Maui | 波利尼西亞 |
| 文化英雄/智者 | Odin（渡鴉為其使者） | 北歐 |
| 動物造人 | Nüwa（女媧） | 中國 |
| 惡作劇神 | Eshu | 約魯巴 |
| 騙子/文化英雄 | Loki | 北歐 |

西伯利亞渡鴉與北美西北海岸 Raven 的對應是神話比較學中最經典的案例——二者共享「盜光」、「造人」、「盜火」、「變形」與「惡作劇」核心母題，語言學與考古學證據支持二者同源於古西伯利亞—古愛斯基摩文化層。

## 相關神祇

- **Kere-tkun**：楚科奇造物主，渡鴉之上仍有最高存在
- **Kalau**：楚科奇惡靈，渡鴉的對手
- **Buga**：通古斯至高神，但功能不同——渡鴉更具體、更擬人化
- **Kotan-kar-kamuy**：愛努創世神，同樣有潛水鳥造地（跨物種造物者）

## 出現在

- 楚科奇與科里亞克薩滿口傳
