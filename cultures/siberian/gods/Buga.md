# Buga（通古斯至高神）

- **文化：** 西伯利亞神話（通古斯 / Evenki / Even）
- **職掌：** 至高造物主、宇宙秩序、天界主宰

## 概述

Buga（布加）是通古斯語系民族（Evenki、Even、Negidal 等）信仰中的至高神，其名稱直接意為「存在」、「宇宙」或「天」。與蒙古的 Tengri、阿爾泰的 Bai-Ulgan、芬蘭的 Ukko 同屬北亞—北歐的「天父—造物主」神話圈。Buga 居住在宇宙樹（世界樺樹）頂端的第三層天，掌管宇宙秩序、季節更替與薩滿傳承。他並非擬人化的具體形象，而是無所不在的宇宙力量——其意志表現為自然規律與薩滿的預言。Evenki 獵人在森林中行獵前必須向 Buga 祈禱，請求允許取用動物。

## 神話事蹟

Buga 最核心的功績為創世。最初宇宙只有無邊汪洋與虛空——Buga 從自身分離出光與暗，命令一隻鵡鴒（或潛水鳥）潛入海底取泥。前兩次均失敗，第三次鳥從嘴中吐出少量泥土——Buga 將其鋪於水面，形成大地。他繼而以泥土與樺木枝條創造人類——吹入靈魂後人便有了生命。Buga 將火賜予人類，教導他們編織獸皮、搭建帳篷（Chum）與辨識可食植物。部分傳說記載 Buga 以雷電懲罰違反薩滿禁忌的惡人。

## 跨文化對應

| 西伯利亞 Buga | 對應神祇 | 文化 |
|---|---|---|
| 至高造物主 | Bai-Ulgan | 阿爾泰 / 蒙古 |
| 天父 | Tengri | 蒙古 / 匈奴 |
| 宇宙秩序 | Ukko | 芬蘭 |
| 創世者 | Dievas | 波羅的 |
| 天父 | Dyaus Pita | 印度吠陀 |
| 天父 / 雷神 | Zeus | 希臘 |
| 天父 | Jupiter | 羅馬 |

Buga 與蒙古的 Tengri 共享「永恆藍天」的抽象概念，但 Evenki 的 Buga 更具有林木與狩獵文化的色彩——其儀式常在樺樹林中的聖地舉行，而非露天祭壇。

## 相關神祇

- **Yryn-Ajy**：雅庫特造物主，功能相近但名稱不同
- **Shaman**：薩滿為 Buga 與人間的溝通者
- **Bayanai**：狩獵神，為 Buga 在森林中的執行者
- **Umai**：生育女神，Buga 的女性補充
- **Kalau**：楚科奇惡靈，Buga 的對立面

## 出現在

- Evenki 與 Even 薩滿口傳 Uliger
- 通古斯比較神話學重建（Vladimir Jochelson、Maria Czaplicka 民族誌）
- 北亞薩滿宇宙樹敘事
