# Umai（通古斯生育女神）

- **文化：** 西伯利亞神話（通古斯 / Evenki / 突厥系）
- **職掌：** 生育、嬰兒靈魂、家族繁榮、爐火保護

## 概述

Umai（亦作 Ymay、Umay）是通古斯與西伯利亞突厥語民族信仰中的生育女神與嬰兒靈魂守護者。她的名字源於古突厥語 *uma*（子宮/胎盤），與蒙古語的「母腹」同源。Umai 居住在宇宙樹根部附近的神聖樹洞中——嬰兒出生前，靈魂在此處等待降生。薩滿治病時若遇兒童疾病，常被診斷為 Umai 將孩子的靈魂召回樹洞——薩滿必須舉行儀式將其請回。Umai 的形象通常為年輕母親，長髮披肩，懷中抱嬰兒或胸前有無數乳房。她也是爐火的守護者——帳篷（Chum）中的火永不熄滅，象徵 Umai 的持續保護。

## 神話事蹟

Umai 最核心的功能為掌管生育與嬰兒靈魂的轉移。Evenki 傳說中，嬰兒出生前，Umai 在宇宙樺樹的樹洞中為靈魂吹入生命氣息——氣息越強，孩子將來越健康。流產或嬰兒夭折被解釋為 Umai 收回靈魂——通常因為父母違反了某項禁忌。另一則流傳於雅庫特的故事：一名婦女連續失去三個孩子，薩滿入冥界向 Umai 求情。Umai 展示三根樹枝——其中兩根已折斷，第三根尚綠。她告訴薩滿：「這位母親的第三個孩子可以存活，但必須終生配戴樺樹皮護符。」從此雅庫特與 Evenki 形成了新生兒佩戴樺樹皮或馴鹿皮護符的習俗，以確保 Umai 不會召回靈魂。Umai 也引導成年女性的經期與懷孕——她以月亮的盈虧週期標記適合受孕的時機。

## 跨文化對應

| 西伯利亞 Umai | 對應神祇 | 文化 |
|---|---|---|
| 生育女神 / 嬰兒靈魂守護者 | Artemis / Eileithyia | 希臘 |
| 生育與爐火 | Vesta / Hestia | 羅馬 / 希臘 |
| 地母 / 生育 | Žemyna / Zemes māte | 波羅的 |
| 生育女神 | Frigg / Freyja | 北歐 |
| 送子娘娘 / 床母 | 註生娘娘 / 床母 | 中國 |
| 子宮 / 生命樹下 | 西王母（崑崙樹下）| 中國 |
| 嬰兒靈魂守護 | コノハナサクヤビメ（木花開耶姬）| 日本 |

Umai 與希臘的 Eileithyia（生育女神）共享「決定出生與否」的核心功能——二者都居住在象徵生命源泉的洞穴（Umai 的樹洞 vs Eileithyia 的阿瑪尼索斯洞穴）。Umai 與中國的註生娘娘功能幾乎完全平行——二者都掌管生育登記與嬰兒靈魂的分配。

## 相關神祇

- **Buga**：至高神，Umai 為其女性面向之一
- **Bayanai**：狩獵神，與 Umai 形成「取生對立」——Bayanai掌管動物的死、Umai 掌管人類的生
- **Erlik**：冥界之王，Umai 的靈魂樹洞與 Erlik 的冥界形成對應（一個待生、一個待審）
- **Yryn-Ajy**：雅庫特造物主，Umai 在雅庫特信仰中地位弱於 Evenki 版本
- **Shaman**：薩滿為 Umai 與人類家庭之間的溝通橋樑

## 出現在

- Evenki 與突厥語民族分娩儀式
- 薩滿兒童治病儀式
- 西伯利亞育兒習俗（樺樹皮護符）
- 阿爾泰與薩彥嶺地區的生育祭祀
