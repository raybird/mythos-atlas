# Tulujn（通古斯雷神）

- **文化：** 西伯利亞神話（通古斯 / Evenki）
- **職掌：** 雷電、降雨、正義審判、天火

## 概述

Tulujn（亦作 Tuliyn、Tolujn）是通古斯 Evenki 信仰中的雷電之神，職掌風暴、降雨與以雷電懲罰違反薩滿禁忌者。他的形象記錄較為殘缺——Evenki 的動物風格藝術與薩滿鼓圖案中保留了他的輪廓：手持石鎚或木槌，身披熊皮，駕乘暴風雲。他在西伯利亞神系中的地位類似於執行正義的神警——Buga（至高神）制定宇宙法則，Tulujn 以雷電確保法則不被違背。Evenki 獵人在夏季雷暴發生時必須停止狩獵，躲入帳篷，不可直視閃電——那被視為 Tulujn 正在執法，凡人不得干預。

## 神話事蹟

Evenki 口傳中最著名的 Tulujn 故事與森林惡靈相關。一隻名為 Chargin（森林吞噬者）的惡靈在泰加林中四處破壞，吞食動物與人類。獵人們向 Buga 祈禱——Buga 派遣 Tulujn 下凡。Tulujn 駕馭黑色雲團降臨，以雷鎚重擊地面——每一次撞擊都產生閃電，將 Chargin 擊退。惡靈逃入湖中，Tulujn 以三道雷電劈開湖水，迫使 Chargin 現身。最後一擊將惡靈化為焦炭。從此 Tulujn 的雷電被視為森林中正義的象徵。另一故事描述 Tulujn 懲罰一名偷竊鄰居馴鹿的獵人——獵人矢口否認，Tulujn 以雷電擊中他面前的大樹，樹皮上浮現出偷竊的圖像，獵人當場承認。Tulujn 還與降雨相關——乾旱時 Evenki 薩滿製作小型雷鎚符號懸掛於樺樹上，向 Tulujn 請求降雨以救活馴鹿的牧草。

## 跨文化對應

| 西伯利亞 Tulujn | 對應神祇 | 文化 |
|---|---|---|
| 雷神 / 正義執行者 | Zeus（雷電+正義）| 希臘 |
| 雷神 | Thor / Mjölnir | 北歐 |
| 雷神 | Perun / Perkūnas | 斯拉夫 / 波羅的 |
| 雷神 | Indra / Vajra | 印度 |
| 雷神 | Shango（雙斧）| 約魯巴 |
| 雷神 | 雷公 | 中國 |
| 雷神 / 天火 | Ukko | 芬蘭 |
| 雷神 | Tarhunt / Teshub | 安納托利亞 |

Tulujn 的雷鎚與北歐 Thor 的 Mjölnir、斯拉夫 Perun 的雷斧群屬於同一個北歐—北亞雷神武器圈——三者皆為「擲出必命中」型武器且回歸持有者手中。Tulujn 的正義審判功能最接近希臘 Zeus（Dike 的執行者）與印度 Indra（以 Vajra 懲罰違反 Ṛta 者），反映雷神跨文化地與社會秩序緊密連結。

## 相關神祇

- **Buga**：至高神，Tulujn 的法則執行者
- **Bayanai**：狩獵神，Tulujn 在森林中的正面力量與 Bayanai 形成互補
- **Erlik**：冥界之王，Tulujn 審判的惡靈最終送至 Erlik 處
- **Kalau**：楚科奇惡靈，Tulujn 式的雷電功能在楚科奇信仰中缺失（Kalau 代表 chaos 而非秩序）
- **Shaman**：薩滿為 Tulujn 與人類的溝通中介

## 出現在

- Evenki 薩滿鼓圖案與雕刻
- 19-20 世紀俄羅斯民族誌（Shirokogoroff、Czaplicka）
