# N'urgun Bootur（雅庫特文化英雄）

- **文化：** 西伯利亞神話（雅庫特 Sakha）
- **職掌：** 戰士英雄、惡靈討伐者、人類守護者

## 概述

N'urgun Bootur（紐爾衮·波圖爾，亦作 Nürgun Bootur）為雅庫特 Olonkho 史詩循環中最著名的英雄。他既是人類又與天界相連——為 Yryn-Ajy（善造物主）派遣下凡的戰士，專責對抗 Abasy（惡靈群）對人類世界的侵擾。N'urgun Bootur 騎乘一匹神話級的白色戰馬（Malju'u），手持三刃長矛與鐵弓，身著以銅、鐵、鋼三層打造的戰甲。他的戰鬥不僅是物理層面的——同時也是薩滿式的靈魂對抗，他能在戰鬥中召喚 Ajy 善靈的援助。

## 神話事蹟

N'urgun Bootur 的傳說構成 Olonkho 史詩的核心部分。他一出生便具備完整的力量與智慧——沒有成長期，直接進入戰鬥。他最重要的戰鬥包括：斬殺三頭 Abasy 巨靈 Kün D'iribine、搗毀下界 Abasy 的鐵城、救回被惡靈擄走的人類靈魂。在最著名的史詩段落中，N'urgun Bootur 下達冥界三層——擊敗守門的九頭蛇怪——將被捕的祖先靈魂釋放回人間。他的戰馬 Malju'u 在戰鬥中能說話，給予英雄智慧與預言。N'urgun Bootur 最終完成所有任務後返回 Yryn-Ajy 的天界居所，成為北極星（象徵宇宙秩序）的守護者。

## 跨文化對應

| 雅庫特 N'urgun Bootur | 對應英雄 | 文化 |
|---|---|---|
| 神派戰士討伐惡靈 | Heracles | 希臘 |
| 降臨拯救人類 | Indra | 印度吠陀 |
| 英雄入冥界救靈魂 | Orpheus / Heracles | 希臘 |
| 騎戰馬征戰 | Cú Chulainn | 凱爾特 |
| 斬多頭蛇怪 | Heracles（九頭蛇） | 希臘 |
| 討伐阿修羅 | Rama | 印度 |
| 文化英雄戰惡魔 | 后羿 | 中國 |

N'urgun Bootur 與印歐譜系的英雄有驚人平行——尤其是希臘 Heracles 的十二試煉、印度 Indra 斬 Vritra、以及中國后羿的上天任務。可能的解釋為北亞與印歐神話英雄原型經由草原絲路的深層交流。

## 相關神祇

- **Yryn-Ajy**：派遣 N'urgun Bootur 的造物主
- **Abasy**：惡靈群，N'urgun Bootur 的敵人
- **Ajy**：善靈群，N'urgun Bootur 的同盟
- **Malju'u**：神話戰馬（可與北歐 Sleipnir 比較）
- **Kün D'iribine**：三頭 Abasy 巨靈
