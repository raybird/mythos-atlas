# 貝伊維 (Beaivi / Beiwe / Biejvve)

- **文化：** 薩米神話 (Sami Mythology)
- **職掌：** 太陽女神、春季、馴鹿繁殖、人類生育

## 概述

Beaivi 是薩米神話中的太陽女神，也是薩米萬神殿中最重要、最廣泛受到崇拜的神祇。在極地漫長冬季的背景下，太陽的回歸不僅是天文事件，更是薩米文化的核心宗教經驗。每年日照消失的極夜時期，薩米人向 Beaivi 獻祭以祈求她早日回歸；冬至（Juhkku 或 Beaivi 節）是薩米年曆中最神聖的時刻之一。

Beaivi 的名字在原始薩米語中直接意為「太陽」，與芬蘭語 päivä（日）、愛沙尼亞語 päev（日）同源，屬於廣泛的芬-烏戈爾語系太陽詞根。

## 神話事蹟

Beaivi 的神話核心是她的年度旅程。在夏季，她掛在天空中，用溫暖的陽光餵養馴鹿與人類；在冬季，她離開人間，留下大地在黑暗中。薩米人相信太陽的強弱與馴鹿的繁殖直接相關——Beaivi 的陽光使母鹿泌乳、馴鹿幼崽健康成長。

Beaivi 被想像為一位穿著馴鹿皮長袍的女性，乘坐由馴鹿拉的太陽車穿越天空。她隨身攜帶一個大袋子，裡面裝著陽光——她在夏季將陽光傾倒在大地上，冬季則將袋子緊閉。在 Beaivi 節期間，薩米家庭將奶油或脂肪塗在門上，讓 Beaivi 可以舔食，並將馴鹿骨頭投入火中以祈求太陽的力量。

她是人類生育與健康的守護神。不孕婦女向 Beaivi 獻祭白色馴鹿，祈求子女；疾病時也向 Beaivi 祈禱康復。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 北歐 | Sól（太陽女神，乘坐馬車穿越天空） | 女性太陽、馬車、被狼追逐的末日預言 |
| 芬蘭 | Päivätär（日女神） | 語源一致（*päivä → 日），編織金色紡織 |
| 希臘 | Helios（太陽神） | 太陽車穿越天空，但性別不同 |
| 印度 | Sūrya（太陽神，戰車） | 戰車穿越天穹，由七馬拉拽 |
| 日本 | 天照大神（Amaterasu，太陽女神） | 女性太陽神、洞穴隱居與回歸的季節隱喻 |
| 埃及 | Ra（太陽神） | 每日穿越天空的旅程，夜間穿越地下世界 |

## 相關神祇

- 女兒：Biejvvene（Beaivi 的女兒，春季融合象徵，與母親共同回歸）
- 對抗：冬季的黑暗力量（Beaivi 離去導致大地冰封）
- 助手：馴鹿（拉動太陽車的聖獸）
- 父親？：Radien（部分傳統中 Radien 為 Beaivi 之父）
- 協作：Bieggagallis（風神，吹散雲層讓 Beaivi 照耀大地）

## 出現在

- 現代薩米 Beaivi 節慶復興運動
