# 霍拉加勒斯 (Horagalles / Tordöm / Thurribom)

- **文化：** 薩米神話 (Sami Mythology)
- **職掌：** 雷神、天氣、狩獵、男性力量

## 概述

Horagalles（也作 Tordöm、Thurribom、Horagalles）是薩米神話中的雷神，其名字直接借自北歐 Thor（Tor → Horagalles，字面即「托爾老人」），是薩米文化與北歐文化長期接觸的明確語言學證據。然而，Horagalles 並非 Thor 的簡單複製——薩米人將北歐雷神的概念重塑為符合自己狩獵採集與馴鹿畜牧生活的神祇形象。

Horagalles 在薩米萬神殿中與 Bieggagallis（風神）共同掌管天氣，他在春天的第一聲雷鳴被視為大地解凍的信號——此時馴鹿開始遷徙、河流解凍、冬季禁獵期結束。

## 神話事蹟

Horagalles 最顯著的神話特徵是他的分類系統。在薩米宇宙觀中，Horagalles 既是一個神祇的名字，也是一類小型木製神像的通稱——每個家庭或營地可能在門前或馴鹿圈附近豎立 Horagalles 木像以祈求保護。這些木像通常手持一把槌子或斧頭。

Horagalles 與 Bieggagallis（風神）是合作者：Horagalles 掌控雷鳴與閃電（垂直的破壞力量），Bieggagallis 掌控風（水平的移動力量）。兩者共同構成薩米天氣神學的雙軸系統——冬季風神與夏季雷神的輪流主導決定了四季的更替。

霍拉加勒斯的武器是雙刃斧或槌子，當他揮舞槌子發出雷聲時，閃電被認為是他斧頭上迸發的火花。他驅逐冬季的惡靈與疾病，確保馴鹿群安全度過最冷的月份。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 北歐 | Thor（雷神） | 名字直接取自 Tor，使用槌子或斧頭，對抗巨人/冬季力量 |
| 芬蘭 | Ukko（雷神，天空之神） | 春季第一聲雷鳴標誌播種季節開始 |
| 斯拉夫 | Perun（雷神） | 斧頭作為雷電武器、春季回歸擊敗冬季 |
| 印度 | Indra（雷雨之神） | 以雷電對抗冬季的冰封力量 |
| 中國 | 雷公 | 雷電的垂直打擊作為破壞與淨化的力量 |

## 相關神祇

- 合作者：Bieggagallis（風神，冬天吹走雲層，夏天帶來雨水）
- 同僚：Beaivi（太陽女神，春季與 Horagalles 的雷聲共同解凍大地）
- 助手：馴鹿精靈（幫助 Horagalles 巡視人類營地）
- 敵對者：Stalo（巨人/強盜，Horagalles 保護人類免受 Stalo 襲擊）
- 北歐對應：名字借自 Tor，但功能細節（狩獵、鹿）為薩米特有

## 出現在

- 薩米太陽鼓（Noel-bearka）上的 Horagalles 槌子符號
- 18 世紀傳教士記錄（Kildal、Tuderus 等）
- 芬蘭北部薩米社區保留的 Horagalles 木像傳統
- 現代薩米說唱與藝術中對雷神的當代詮釋
