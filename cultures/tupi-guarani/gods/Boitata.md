# 波伊塔塔 (Boitatá / M'boi Tatá)

- **文化：** 圖皮-瓜拉尼神話 (Tupi-Guarani Mythology)
- **職掌：** 火蛇、森林與草地的守護者、火焰之靈

## 概述

Boitatá（圖皮語：mboi = 蛇 + tatá = 火，意為「火焰之蛇」）是巴西神話中最古老且最知名的傳說生物之一。它是一條巨大的火蛇（有時被描述為一隻巨大的發光眼睛），在夜晚穿越草地與叢林。Boitatá 既是保護者也是毀滅者——它守護土地不被不當焚燒，但當人類以火焰破壞森林時，它會以火焰反擊。

最早的紀錄出自葡萄牙耶穌會傳教士 José de Anchieta 於 1560 年的書信——他記錄了當地人對 Boitatá 的信仰，描述「一種幽靈般的蛇形光，在夜間照亮整個濕地」。

## 神話事蹟

在古老的傳說中，世界曾經歷一場大洪水。所有動物都在一隻巨大的木筏上避難。當洪水退去時，動物們回到陸地。但一條蛇留在了最後——牠在船底待了太久，幾乎腐爛。這條蛇爬上岸後，飢餓地吃掉了所有在洪水中死亡的動物屍體。

牠吃了太多，身體無法承受——開始燃燒。從牠的身體中迸發火焰。這就是 Boitatá 的誕生——第一條火蛇。

如今 Boitatá 在夜間巡行乾草原與叢林邊緣。它的火焰在夜晚閃爍——有時像一隻巨大的發光眼睛，有時像一條流動的火焰河。Boitatá 攻擊那些以火焚林的人類——但對尊重自然的人無害。

在巴西民間傳說中，Boitatá 的出現被視為天氣變化的徵兆——如果它在旱季出現，火焰特別明亮，表示接下來的雨季極為豐沛。

## 跨文化對應

| 文化 | 對應存在 | 共享特徵 |
|------|---------|---------|
| 澳洲原住民 | Rainbow Serpent（彩虹蛇） | 蛇形、與水與火（彩虹）連結、創世力與毀滅力共存 |
| 北歐 | Jörmungandr（耶夢加得，環繞世界的巨蛇） | 巨大蛇形、與水（海洋）連結、世界毀滅的角色 |
| 北歐 | Fafnir（化為龍的巨人） | 蛇／龍與火焰的連結 |
| 中國 | 燭龍（Zhulong，燭照九陰的鐘山之神） | 人面蛇身、睜眼為白天、閉眼為夜晚、口中銜燭 |
| 馬雅 | Kukulkan（羽蛇神） | 蛇與天象（光）的結合、季節變化與降雨 |
| 希臘 | Hydra（九頭蛇，噴毒而非火） | 蛇形、重生與毀滅力量 |

## 相關神祇

- 相關天空之神：Tupã（雷電之神，Boitatá 的火焰被視為 Tupã 在地上的延伸）
- 同一分類：Mboi（圖皮-瓜拉尼語的蛇，所有蛇類靈體的總稱）
- 對立者：人類縱火者——Boitatá 懲罰不當使用火的人類
- 節日：無特定節日，但巴西農村在燒墾季節會進行「平息 Boitatá」的儀式
- 現代文化：Boitatá 是巴西的神話象徵之一——被視為環境保護的寓言符號
