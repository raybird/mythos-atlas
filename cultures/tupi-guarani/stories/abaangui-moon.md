# 阿巴安圭造月 — 頭顱升天

- **文化：** 圖皮-瓜拉尼神話 (Tupi-Guarani Mythology)

## 故事背景

在圖皮-瓜拉尼神話的宇宙起源敘事中，人類與天體之間有著比希臘或北方神話更近的血緣關係。月亮與星辰最初並非遙不可及的神體，而是由一個凡人——或者說人類英雄——的身體變化而成。阿巴安圭（Abaangui）的故事是所有南美造月神話中最奇異的版本之一：月亮是一位凡人的頭顱。

這個神話屬典型的「斬首升天」母題（Severed-Head-As-Celestial-Body），與中國神話中刑天被斬後以乳為目、以臍為口繼續戰鬥，以及北歐神話中 Mímir 的頭顱被保存後提供智慧的結構不同——阿巴安圭的頭顱不是維持意識或說話能力，而是直接變成天文物體。

## 情節

### 阿巴安圭與兄弟

阿巴安圭與他的兄弟（在某些版本中名為 Sarigoera）同住在一個沒有月亮的世界。每到夜晚，大地上空一片漆黑，只有星辰的微光。兩兄弟是獵人——但夜間狩獵極為困難，狗群無法追逐獵物。阿巴安圭決定改變這個狀況。

有一天，阿巴安圭對他的兄弟說：「我需要用你的頭來做一件事。」

他的兄弟出於信任——或出於恐懼——同意了。阿巴安圭揮刀斬下兄弟的頭，然後高高地將其拋向天空。

### 月亮的誕生

兄弟的頭顱飛入天空後，發出耀眼的光芒——變成了月亮。然而阿巴安圭發現兄弟的頭顱太小，產生的光線微弱且不均勻，不足以照明大地。他需要一個更大的月亮。

於是他決定加入自己的頭顱。阿巴安圭回轉家中，對妻子說：「請你砍下我的頭，將它拋上天去。我的頭比兄弟的更大、更亮，能成爲真正的月亮。」

妻子哭泣抗拒，但阿巴安圭堅持。最終妻子含淚照辦——阿巴安圭的頭顱飛入天際，成為如今照亮夜晚的滿月。

### 星辰的起源

阿巴安圭的無頭身體倒在地上後，從他身體的每一個部位——手指、腳趾、耳朵、頭髮——飛出了無數細小的火花。這些火花升上天空，變成了星辰。身體本身則在大地上腐化，回歸 Nhanderuvuçú 所創造的原初大地。

兄弟的頭顱變成了新月與殘月——它較小、較暗，在月亮週期中處於陰暗的階段。阿巴安圭的頭顱則是滿月——圓滿而明亮。這就是爲什麼月亮會有盈虧的週期變化。

## 跨文化比較

| 文明 | 類似神話 | 與阿巴安圭的異同 |
|------|---------|----------------|
| 北歐 | Mímir 之首 | 頭顱被保存並提供智慧，而非變成天體 |
| 希臘 | 奧菲斯之首 | 頭顱漂流歌唱與預言，爲神諭來源 |
| 中國 | 刑天 | 被斬首後以乳爲目繼續戰鬥，頭顱本身不變天體 |
| 威爾斯 | Brân 之首 | 頭顱死後繼續說話護佑國土，埋於倫敦塔 |
| 印度 | 梵天之首（降臨故事） | 濕婆斬去梵天第五首，首級成爲其法器 |
| 波利尼西亞 | Māui 之死 | Māui 試圖穿越死亡女神而亡，非頭顱成天體 |
| 因紐特 | 月亮神 Igaluk | 月神爲凡人兄妹的故事，但非由斬首產生 |

## 神話類型：斬首昇天

阿巴安圭故事屬於「身體部分成爲天體」(Dismemberment Cosmogenesis) 的次類——專注於頭顱。這個母題在美洲尤其常見但各有變化：
- 墨西哥的 Coyolxauhqui 被月神 Huitzilopochtli 斬首後肢解，首級成爲月亮
- 祕魯的神話中也有類似的斬首造月敘事
- 但是只有圖皮-瓜拉尼版本以「兩兄弟交替作爲月亮本體」和「自願斬首」爲特色——這是全球同類神話中極罕見的

阿巴安圭的故事在世界神話中幾乎是獨一無二的：不僅頭顱變成月亮，而且連身體的細微部分都變成了星辰。這是一個徹底的「人體宇宙化」(Anthropocosmogenesis) 敘事——人類的身體成爲宇宙結構的直接組成部分。
