# 無邪之地：南美版的失落天堂 (Yvy Marãe'ỹ: The Land Without Evil)

- **文化：** 圖皮-瓜拉尼神話 (Tupi-Guarani Mythology)

## 故事背景

Yvy Marãe'ỹ（瓜拉尼語：yvy = 土地 + marã = 邪惡／破壞 + e'ỹ = 沒有——「無邪之地」）是圖皮-瓜拉尼神話中的核心理想——一片從未被人類觸及的純淨土地。這不是一個死後世界，而是一個在地理上真實存在（或在神話地理中可達）的樂園，位於東方海洋的彼岸。圖皮-瓜拉尼人對 Yvy Marãe'ỹ 的信仰強烈到在歷史上引發了多次大規模的集體遷徙——整個部落在先知（karai）的帶領下向東遷移以尋找無邪之地。

## 情節

### 第一章：Nhanderuvucu 的承諾

在創世的開端，Nhanderuvucu（至高神）創造了世界與人類。他將人類安置在 Yvy Pyau（新世界）中——這是一片美好的土地，但 Nhanderuvucu 知道人類最終會學會破壞與欺騙。

所以他在東方保留了一塊從未受過的區域——Yvy Marãe'ỹ。他對第一批人類說：「如果你們尋找我，不要往上看——我不在那裡。往東方走，越過山脈與海洋，那是我真正的家。」

### 第二章：人類的墮落

隨著人類數量增加，他們學會了戰爭、偷竊與土地爭奪。Nhanderuvucu 的預言實現了。首領們開始互相殺戮，祭司們開始以神的名義要求祭品。

一位名為 Karai（先知）的老人記得 Nhanderuvucu 的承諾。他站在部落中央宣布：「這裡的世界已經壞了。鳥不再歌唱、河水變苦、土地不再豐饒。這是 Yvy Pora（腐爛的世界）。我們必須離開這裡。」

### 第三章：向東方的遷徙

Karai 帶領他的族人向日落的反方向（東方）前進。他們穿越叢林、越過大河。一路上許多人放棄了——他們留下來，被森林吞噬。但 Karai 的步伐從未停止。

他告訴追隨者：「你們會問：土地在哪裡？我看不到。但我告訴你們——當你們到達時，你們不需要看到，你們會知道。」

在旅程中，Nhanderuvucu 賜予追隨者永生——他們不再衰老、不需要睡眠與進食。這是通往無邪之地的道路上最後的準備——放下肉體需求。

### 第四章：到達（永遠無法講述的結局）

故事的結局是開放式的：沒有一個從 Yvy Marãe'ỹ 返回的敘述者。那些到達的人不會回來——因為無邪之地是一條單行道。瓜拉尼口傳傳統說，無數的部落在不同的時間點穿越了看不見的邊界，進入了沒有破壞、沒有謊言、沒有飢餓的世界。

至今在某些瓜拉尼村莊中，老人們仍然在特定的月相凝視東方海平面，等待 Yvy Marãe'ỹ 再次向人類敞開大門。

## 跨文化平行

| 文化 | 對應樂園 | 比較 |
|------|---------|------|
| 希伯來 | 伊甸園（Gan Eden） | 人類被驅逐後不能回去；圖皮為可達的樂園 |
| 希臘 | 埃律西昂（Elysium，英雄死後之所） | 死後世界 vs 圖皮的生前可達 |
| 凱爾特 | 阿瓦隆（Avalon，蘋果島） | 西方海洋中的樂園島（與圖皮的東方完全相反） |
| 中國 | 蓬萊仙島 | 東海中的神仙居所、不死藥；方向一致但功能不同 |
| 印度 | Uttarakuru（北部福地） | 完美的土地、沒有疾病與痛苦 |
| 阿茲特克 | Tlalocan（雨神樂園） | 自然豐饒之地、為特殊死亡者保留 |
