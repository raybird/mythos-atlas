# 埃古茲基 (Eguzki / Eki / Ekhi)

- **文化：** 巴斯克神話 (Basque Mythology)
- **職掌：** 太陽神、光明、生命之力、守護人類免受邪惡侵害

## 概述

Eguzki（也作 Eki 或 Ekhi，巴斯克語意為「太陽」）是巴斯克神話中的太陽神靈，在基督教化前的巴斯克信仰中扮演核心角色。Eguzki 被視為 Mari（大地女神）之子或助手，在每日的運行中護送太陽穿越天空。與許多印歐神話（如希臘的 Helios、北歐的 Sól）不同——Eguzki 不僅是天空之神，更直接參與人類的道德世界：祂會懲罰那些在日落後仍然進行不潔活動（如紡織、洗衣、磨穀）的人類。

## 神話事蹟

Eguzki 最突出的特徵是祂是「觀看人類行為的眼睛」。巴斯克諺語道：「Etxean oihu eta kaldera, etxe barruan Eguzki behatzen du.」（在家中吵鬧與混亂，Eguzki 在觀看。）

Eguzki 與 Ilargi（月亮）的關係在各地傳說中不一：在一些地區他們是兄妹，在另一些地區則是夫妻。他們交替統治天空，但 Eguzki 的光芒遠強於 Ilargi。

在巴斯克傳統中，女性不能在日落後紡織——因為 Eguzki 不喜夜間的人類活動，違者會被 Eguzki 焚毀房屋或使亞麻布變黑。婚姻必須在白天舉行以獲得 Eguzki 的祝福。節日中人們跳起太陽之舞（Eguzki-dantza）以祈求豐收。

Eguzki 的聖所位於高山頂及石環（harrespil）——這些史前遺蹟被認為是用於向太陽祈禱的場所。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Helios（太陽神，每日駕車穿越天空） | 全觀之眼、護送陽光 |
| 北歐 | Sól（太陽女神） | 日夜交替，被狼追趕（與 Eguzki 的追獵者傳說相似） |
| 埃及 | Ra（太陽神，以太陽船穿越天空與地下） | 每日重生、正義之眼 |
| 日本 | 天照大神（Amaterasu，太陽女神） | 人類道德行為的監督者 |
| 印度 | Sūrya（太陽神） | 消除疾病、賜予長壽與豐收 |

## 相關神祇

- 母親／創造者：Mari（大地女神，Eguzki 自她體內升起）
- 對手：Ilargi（月亮之神或女神，Eguzki 的兄弟姐妹或配偶）
- 關聯：與 Urtzi（天空神）共享宇宙秩序
- 敵對者：覆盆子（Basajaun 的盟友，夜間出沒，被日光驅散）
- 巴斯克語星期一：astelehena（字面「白日之後」，與太陽無直接關聯，但 aste 意為「周」）
- 節日：San Juan 篝火節（Jainkoaren sua，上帝的火焰，基督教化後的太陽崇拜）
