# 烏爾齊 (Urtzi / Odei / Ortz / Ostri / Oskilli)

- **文化：** 巴斯克神話 (Basque Mythology)
- **職掌：** 天空之神、雷電、風暴、宇宙規律

## 概述

Urtzi（也作 Odei、Ortz、Ostri、Oskilli，巴斯克語中的「天空」或「雷電」）是巴斯克神話中掌管天空、雷電與風暴的神祇。他的地位複雜且在不同地區有所差異——在某些版本中 Urtzi 是至高神（天上之神），在其他版本中他僅為雷電之靈，低於 Mari（大地女神）的權力。Urtzi 是巴斯克神話中唯一可被歸類為「天父」（Sky Father）類型的神祇——但巴斯克宗教的核心為大地母神（Mari），Urtzi 通常被視為她的執法者而非平等的對手。

## 神話事蹟

Urtzi 被巴斯克人認為是雷電的創造者——他以雷電（ostargi，字面意為「天空之火」）懲罰人類與邪惡力量。在巴斯克傳統中，被閃電擊中的地方被視為聖地，禁止人畜停留。

Urtzi 與 Mari 的關係是巴斯克神話的關鍵動力：Mari 統治大地與人類命運，Urtzi 執行她的意思——他以雷電劈打不尊重 Mari 的人，或乾旱懲罰腐敗的社區。

一個著名的傳說講述 Urtzi 的嫉妒：他愛上了一個 Basajaun（森林野人）的女性後代，但對方選擇與凡人結婚。Urtzi 的憤怒化為雷暴，摧毀了整片區域。只有 Mari 出面以暴雨熄滅了雷電之火，平息了 Urtzi 的怒火。

在某些孤立地區，Urtzi 被賦予創世的地位——他是第一個點亮天空的存在，他將火從天空偷取並賜予人類（類似普羅米修斯角色，但此版本較邊緣）。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | 宙斯（Zeus，雷神，奧林帕斯之王） | 雷電為武器，懲罰不義者 |
| 北歐 | 索爾（Thor，雷神，擁有妙爾尼爾之錘） | 雷電作為攻擊武器；Urtzi 的神聖閃電也具備「返回持有者」的特性 |
| 印度 | 因陀羅（Indra，雷雨之神，以金剛杵擊敵） | 雷雨戰神、阻擋蛇怪、斬殺邪惡 |
| 中國 | 雷公（雷神，以雷劈擊） | 雷電為懲罰工具、正義之執行 |
| 埃及 | 賽特（Seth，風暴之神，混亂力量） | 風暴的毀滅性力量、宇宙整齊的對立面 |
| 斯拉夫 | 佩倫（Perun，雷神，以斧頭投擲） | 雷神打擊邪惡的結構極為相似 |

## 相關神祇

- 上級：Mari（大地女神，Urtzi 為其執行者）
- 伴侶（部分版本）：Ilargi（月亮神，藉由 Urtzi 的雷電獲得夜晚的光）
- 敵對者：Basajaun（森林之主，Urtzi 的雷電摧毀森林時與 Basajaun 衝突）
- 下屬：Odei（暴風雲團，Urtzi 的雲朵僕人）

## 詞源與語言學注記

Urtzi 的語意邊界模糊——同一個詞根在巴斯克語中衍生出多個與天空相關的詞：
- ortzi / urtzi：天空、打雷
- o (-a 為定冠詞)：雲、天空
- oskorri：黎明（字面「紅色的天空」）
- ortzadar：彩虹（字面「天空的角」）
- oskarbi：晴天（字面「純淨的天空」）

此語義擴散暗示 Urtzi 原本可能是巴斯克神話中一個更大的「天空原則」之神，但在基督教化過程中被簡化為雷電之神。
