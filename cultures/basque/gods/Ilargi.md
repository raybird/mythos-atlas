# 伊拉爾吉 (Ilargi / Ilazki / Iralei / Lare)

- **文化：** 巴斯克神話 (Basque Mythology)
- **職掌：** 月亮之神、夜晚與潮汐、女性生育、魔法與秘密

## 概述

Ilargi（也作 Ilazki、Iralei、Lare，巴斯克語原意為「亡者的光芒」）是巴斯克神話中的月亮之神。與 Eguzki（太陽神）相對，Ilargi 掌管夜晚、生育、魔法、潮汐，以及人類的潛意識世界。Ilargi 在某些地區被視為女神，在某些地區被視為男神——此性別二元性反映了月亮本身轉換的自然特徵。

Ilargi 有一個顯著特徵：她的臉上有永久性的傷痕或黑斑——在巴斯克傳說中，這是她與 Eguzki 爭吵時留下的印記。月亮上的陰影被解釋為 Ilargi 的臉龐的傷疤。

## 神話事蹟

Ilargi 與 Eguzki 的關係構成了巴斯克神話的核心二元結構：

**第一種敘事（兄妹型）：** Ilargi 與 Eguzki 為 Mari 所生的雙胞胎，負責交替照亮世界。但 Ilargi 總是嫉妒 Eguzki 的強光——當她試圖模仿 Eguzki 過度發光時，Eguzki 以泥土扔向她的臉，產生了月亮上的暗影。

**第二種敘事（夫妻型）：** Ilargi 原為 Eguzki 的妻子，但因與 Basajaun（森林野人）的曖昧關係被 Eguzki 發現，Eguzki 將她永遠驅逐到夜晚的領域。巴斯克人認為新月（ilberri）是 Ilargi 與 Eguzki 短暫和解的日子。

Ilargi 對女性的影響尤為強大——巴斯克婦女會在滿月時進行生育儀式，將白布鋪在月光下以獲得懷孕的力量。月亮也被視為死者靈魂的接收者——因此 Ilargi 的名字本身意為「亡者的光芒」，在葬禮上人們望向月亮為逝者祈禱。

月蝕（Ilargi-oste）被解釋為 Ilargi 被惡靈或蛇攻擊——巴斯克人同樣以敲擊金屬器皿的方式試圖拯救月亮，與菲律賓 Bakunawa 神話的驅趕儀式驚人地一致。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Selene（月神，駕著雙馬車穿越夜空） | 月光對女性的生殖力量、月相的規律 |
| 埃及 | 托特（Thoth，月神，智慧與書寫之神） | 月相與時間計算、夜間知識的來源 |
| 中國 | 嫦娥（月神，居住在月亮上的孤獨女性） | 月亮的女性關聯、孤獨與思念的象徵 |
| 菲律賓 | Mayari（月神，臉上有一隻受傷的眼睛） | 臉上受創後月相產生的平行結構 |
| 日本 | 月讀命（Tsukuyomi，月神，男性） | 性別的流動性；夜與日不可調和的對立 |
| 北歐 | Máni（月神，男性，被狼追趕） | 月亮被攻擊/傷害的神話動機 |

## 相關神祇

- 兄弟／配偶：Eguzki（太陽神，Ilargi 的對手與伴侶）
- 母親：Mari（大地女神，生育了 Ilargi 與 Eguzki）
- 潛在情人：Basajaun（森林之主，與 Ilargi 的禁忌關係）
- 象徵：柳樹（巴斯克傳統中柳樹吸收月光，用於魔法儀式）
- 節日：月圓之夜（Ilgora）為巴斯克女巫（sorginak）聚會之日
