# 澳洲原住民神話 (Aboriginal Australian Mythology)

## 創世神話

夢世紀 (Dreamtime / Tjukurpa) 不是「過去」而是永恆的時空——先祖存在穿行無形世界，創造山川河流、動植物與律法，後化為地貌或回歸地下。

## 洪水傳說

澳洲北部沿海部落保存海平面上升的集體記憶——大堡礁曾是陸地平原，被祖先稱為「洪水時代」。部分傳說描述彩虹蛇引發大水重塑地貌。

## 神系
