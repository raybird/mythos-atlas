# 伊南娜下冥界 (Inanna's Descent to the Underworld)

- **文化：** 美索不達米亞神話 (Mesopotamian Mythology)

## 故事大綱

伊南娜下冥界是蘇美文明最著名的神話之一，刻寫於公元前2000年左右的泥板，在考古發現中保存相對完整。此神話深刻影響了後世希臘(奧菲斯/歐律狄刻、Persephone)、迦南(巴力)、埃及(伊西斯/歐西里斯)甚至基督教(耶穌三日復活)的冥界旅程敘事。

## 情節

**出發：** 伊南娜(愛與戰爭女神，金星之神)決定前往冥界拜訪其姐——冥界女王埃蕾什基伽爾(Ereshkigal)。她囑咐守護者寧舒布爾(Ninshubur)：「若三日后我未歸，請至恩基與眾神處敲鼓哀悼求救。」

**七門剝奪：** 伊南娜抵達冥界入口，守門者根據埃蕾什基伽爾的指示讓她通過七道門——每過一門，即取下一件飾物/衣物：第一門取冠冕，第二門取天青石權杖，第三門取天青石頸鏈，第四門取胸前雙排扣，第五門取金戒指，第六門取胸針，第七門取皇家長袍——抵達冥界核心時伊南娜已全身赤裸，徹底卸除一切力量。

**審判與死亡：** 埃蕾什基伽爾與七位安努納奇(Anunnaki)審判官注視伊南娜——他們以死亡之眼注視她，她立即變成屍體，被掛在牆上的鉤子上。

**救援：** 三日后寧舒布爾如囑求助——恩利爾與南娜拒絕，恩基以智慧應對。恩基以指甲垢創造兩個無性別生物——庫爾加拉(Kurgarra)與加拉圖拉(Galatúrra)——授予他們「生命之食」與「生命之水」，指示他們進入冥界，在埃蕾什基伽爾因生產痛苦呻吟時模仿其痛苦，博取同情後換取伊南娜的屍體。

**復活與回歸：** 庫爾加拉與加拉圖拉成功以「啊！我的下體！啊！我的背！」的哀嚎與埃蕾什基伽爾建立共鳴，獲准帶走伊南娜的屍體。他們以生命之食撒六十次、生命之水澆六十次——伊南娜復活。但冥界法則規定「一人往，須一人代」——若無人替代，伊南娜必須重返冥界。

**替代者：** 伊南娜返回人間尋找替代者——見兩個城市的女神虔誠哀悼，放過她們。但回到家鄉庫拉布(Kulaba)時，發現丈夫杜姆茲(Dumuzi)非但未哀悼，反而盛裝坐在王座上享受權力。伊南娜憤怒，以「死亡之眼」注視杜姆茲——杜姆茲被七惡魔帶走。杜姆茲的姐妹葛絲提安娜(Geshtinanna)願意輪流替代他——從此杜姆茲半年在冥界，半年在人間，成爲植物生與死的象徵。

## 神話層次解析

| 層次 | 解釋 |
|------|------|
| **自然象徵** | 金星(伊南娜=晨星/昏星)在日落后隱入地平線(下冥界)，至晨復出(復活)——天文周期 |
| **農業週期** | 杜姆茲半年在冥界(夏季乾旱)半年在人間(冬季/春季生長)——季節循環 |
| **宗教儀式** | 神聖婚姻(伊南娜與杜姆茲)的死亡與復活儀式可能存在於蘇美各城邦 |
| **心理原型** | 女性力量的下降與回歸、死亡的心理穿透、英雄之旅的女性版本 |
| **宇宙秩序** | 上位世界(天/人間)與下位世界(冥界)的不可逆轉法則及其唯一例外 |

## 跨文化平行

- **希臘：** 奧菲斯(Orpheus)入冥界尋妻、Persephone被黑帝斯帶走半年重返
- **日本：** 伊邪那岐赴黃泉尋伊邪那美、逃離時以石堵門
- **埃及：** 伊西斯尋歐西里斯遺體、荷魯斯復仇
- **基督教：** 耶穌三日地下之旅後復活
