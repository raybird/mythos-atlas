# Donbettyr — 高加索水神

Donbettyr（頓貝蒂爾，奧塞提亞語意為「水之彼得」）是 Nart 史詩中的水域之主，掌管海洋、河流與地下水域。他是 Nart 英雄父親的典型——一位居住在海底水晶宮殿中的神靈，擁有數不盡的牛群與財富。

## 神格與屬性

- **領域：** 海洋、河流、水下世界、財富
- **形象：** 白鬚長者，有時以魚尾或水生特徵出現
- **居所：** 海底白色水晶宮殿
- **對應：** 希臘波塞冬、北歐埃吉爾、斯拉夫水神 Vodyanoy

## 神話與崇拜

### 海底的白色宮殿

在奧塞提亞 Nart 史詩中，Donbettyr 居住於海洋深處的水晶宮殿——牆壁由白色石頭砌成，宮殿周圍環繞著無盡的魚群與海怪。他有時被描繪為手持三叉戟的長者，性格既慷慨又易怒：以禮相求者可獲其幫助，傲慢無禮者則會被海潮吞噬。

### 女兒的海婚

Donbettyr 最著名的故事涉及他女兒的海婚。Nart 英雄 Xæmyc 在海邊邂逅了 Donbettyr 的女兒——她以天鵝或美女的形象出現。兩人相戀後，Xæmyc 必須前往海底宮殿向 Donbettyr 求親。水神設下三項考驗：從海底抓回特定的魚、在一天內建造宮殿、辨認偽裝成女兒的水中精靈。

Xæmyc 通過試煉後，Donbettyr 將女兒嫁給他，贈予一支能召喚海浪的號角。婚後這位水之女為 Nart 家族生下 Batraz——Narts 中最強大的英雄。此模式與希臘神話中的佩琉斯與忒提斯（Achilles 父母）的婚禮高度類似。

### 水下牛群

Donbettyr 被認為擁有海洋中最龐大的牛群——這些牛在海藻覆蓋的牧場上吃草，只有在水手遇難時才會浮出水面。夜晚海面上隱約傳來的牛鈴聲被認為是 Donbettyr 的牛群在穿越海峽。這一信仰與北歐神話中埃吉爾的啤酒鍋、希臘神話中海神波塞冬的馬群形成跨文化對應：水域之神通常是財富與牲畜的守護者。
