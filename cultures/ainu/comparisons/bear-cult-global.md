# 跨文化比較：熊崇拜 — 北半球狩獵民族的共同遺產

## 熊崇拜的分布

熊崇拜（Bear Cult）分布於北半球所有有棕熊分布的區域：北歐的薩米人、西伯利亞的諸多民族（鄂溫克、涅涅茨、雅庫特）、日本北部的阿伊努人、北美的因紐特與多個原住民族。此現象高度暗示熊崇拜起源於舊石器晚期的狩獵採集文化。

## 共通的宇宙觀前提

1. **動物自願獻身**：熊被殺不是人類的勝利而是熊神自願的贈與——因此必須以儀式感謝與道歉
2. **骨頭/頭骨的保存**：熊的骨骼代表其本體——妥善保存能使熊神以完整形態返回神界
3. **禁忌語言**：多數文化禁止在熊的儀式中使用日常語言——專用「熊語」反映熊的特殊神聖地位
4. **女性隔離**：女性通常被禁止參與熊的殺戮與相關儀式（阿伊努、薩米皆如此）

## 阿伊努 Iyomante 的獨特處

阿伊努的 Iyomante 是唯一將小熊飼養一至兩年後才進行儀式的熊崇拜——此等候期強化了熊與人類之間的擬親屬關係。小熊被稱為「神的子孫」，由人類母親以母乳餵養，模糊了人與神之間的界線。
