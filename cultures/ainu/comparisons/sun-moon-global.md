# 太陽與月亮：跨文化比較

## Sun and Moon Across Cultures: Comparative Analysis

阿伊努神話中，太陽為女性（Chup Kamuy）、月亮為男性（Kunnecip），與東亞多數文化（日本天照大神為女月男、中國羲和生日月神職各異）相近，但與印歐神話系統（太陽神男性、月神女性）恰好相反。此一分野具有深刻的比較神話學意義。

### 性別配置的跨文化分布

| 文化 | 太陽性別 | 月亮性別 | 備註 |
|------|---------|---------|------|
| **阿伊努** | 女（Chup Kamuy） | 男（Kunnecip） | 月亮以狗形態出現 |
| **日本** | 女（天照大神） | 男（月讀命） | 與阿伊努一致 |
| **中國上古** | 女（羲和） | 女（常羲） | 雙女神制，獨特 |
| **希臘** | 男（Helios） | 女（Selene） | 印歐典型配置 |
| **印度** | 男（Surya） | 女（Chandra） | Surya 為男神 |
| **北歐** | 女（Sól） | 男（Máni） | 日耳曼的獨特配置 |
| **埃及** | 男（Ra） | 男（Thoth） | 月神男性，非典型 |
| **馬雅** | 男（Kinich Ahau） | 女（Ix Chel） | 中美洲典型配置 |
| **波利尼西亞** | 女（Hine-titama） | 男（Rona） | 交錯配置 |
| **西伯利亞** | 女 | 男 | 泛西伯利亞共通 |
| **因紐特** | 女 | 男 | 兄妹追逐的神話 |

### 結構分析

1. **東亞—西伯利亞集群**（阿伊努、日本、西伯利亞、因紐特、部分美洲原住民）：太陽為女、月亮為男。這些文化多為狩獵—採集或初級農業社會，女性與家內火竈、養育的連結強化了太陽的女性象徵。

2. **印歐集群**（希臘、印度、羅馬等）：太陽為男、月亮為女。游牧民族的男性戰神傳統可能影響太陽的男性化。

3. **例外與交錯**：北歐的 Sól 為女性但太陽戰車由男性駕駛；波利尼西亞的太陽與月亮性別隨區域變化。

4. **追逐神話的普遍性**：阿伊努的 Chup Kamuy 被 Kunnecip 追逐、日本的伊邪那岐與伊邪那美（日月分離）、因紐特的太陽少女被其兄在黑暗中追逐——太陽—月亮追逐是跨文化最穩定的敘事模式之一。
