# Kim-un-kamuy — 熊神與送靈祭

Kim-un-kamuy（「山中的神」）是阿伊努萬神殿中最核心的動物神。熊的神聖地位決定了阿伊努宗教生活的最高儀式——熊送靈祭（Iyomante）。

## 神格與屬性

- **領域：** 山脈、森林、動物
- **動物形態：** 棕熊
- **真實形態：** 天界的友好神靈

## 神話與儀式

### 熊的降臨
Kim-un-kamuy 從天界降臨人間時披上熊的外衣。人類獵熊被認為是熊神自願獻身——將肉與毛皮作為禮物贈送。因此阿伊努獵人在殺死熊後會向熊神表達感謝與敬意。

### Iyomante 熊送靈祭
Iyomante 是阿伊努最莊嚴的儀式，歷時三天。流程：
1. **捕捉小熊**：在春天捕捉幼熊，以人類母乳飼養至一歲
2. **祭祀**：在儀式中對小熊說話，告知將送其回天界
3. **箭射與勒頸**：以象徵性的箭射殺小熊，以木棍勒頸確保不流血
4. **祈福**：將熊頭置於祭壇（Nusa），圍坐共食熊肉
5. **送返**：將熊頭骨安放於祭壇永久供奉——熊神以真實形態返回天界後向眾神報告人類的誠意
