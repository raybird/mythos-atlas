# 赫拉 (Hera)

- **文化：** 希臘神話 (Greek Mythology)
- **職掌：** 天后、婚姻、家庭、生育、女性保護

## 概述

赫拉是希臘奧林帕斯十二主神之一，為眾神之王宙斯（Zeus）的姊姊與正妻，亦是克羅諾斯（Cronus）與瑞亞（Rhea）所生的長女。她的羅馬對應神祇為朱諾（Juno），在伊特魯里亞神話中則稱 Uni。赫拉在天界掌管婚姻與家庭秩序，是生育婦女與合法妻子的守護神，但同時也以善妒與懲罰宙斯情人與私生子女的嚴厲形象著稱。她的聖物包括孔雀（其尾羽上有一百隻眼睛的守衛阿耳戈斯之眼）、杜鵑鳥、獅子與石榴。主要聖地位於薩摩斯島（Samos）的赫拉神廟（Heraion）與阿爾戈斯（Argos）的赫拉神廟——阿爾戈斯人稱她為「Hera Argeia」（阿爾戈斯的赫拉），並認為她與該城的命運緊密相連。赫拉的形象在古典藝術中通常表現為頭戴波羅斯（Polos）高冠或王冠，手持權杖或石榴的中年端莊女性。

## 神話事蹟

**1. 赫拉的出生與童年**
赫拉為克羅諾斯與瑞亞之女，出生後被克羅諾斯吞入腹中。宙斯迫使克羅諾斯吐出眾手足後，赫拉被母親瑞亞送至Oceanus（俄刻阿諾斯）與Tethys（特堤斯）夫婦處撫養長大。她在遠離奧林帕斯的寧靜童年中成長，直至成年後返回天界。

**2. 與宙斯的婚姻**
宙斯瘋狂追求赫拉，最初遭到拒絕。據Pausanias《希臘志》記載，宙斯變身成一隻被暴雨淋濕的杜鵑鳥，瑟縮在赫拉懷中求暖——赫拉憐憫地將鳥攏入懷中，宙斯隨即恢復原形強娶了她。眾神在赫拉與宙斯的婚禮上獻上盛大賀禮——大地女神蓋亞（Gaia）獻上一棵金蘋果樹，由赫斯佩里得斯姊妹（Hesperides）看守。赫拉與宙斯的婚姻象徵天與地的神聖結合，奧林帕斯從此以宙斯與赫拉為最高權力核心。

**3. 赫拉的嫉妒與懲罰**
赫拉最著名的神話主題是她對宙斯無數婚外情的反應。她懲罰宙斯的情人與私生子的手段極其嚴厲——伊娥（Io）被變為母牛，受牛虻驅趕流浪至埃及；塞墨勒（Semele）被赫拉慫恿要求宙斯顯現真身，結果在雷火中灰飛煙滅；阿爾克墨涅（Alcmene）的兒子赫拉克勒斯（Heracles）從嬰兒時期便遭到赫拉的迫害——赫拉派遣兩條毒蛇欲殺死搖籃中的嬰兒，卻被赫拉克勒斯徒手捏死。忒瑞西阿斯（Tiresias）因支持宙斯的論點說女性在性愛中享受更多，被赫拉奪去視力，但宙斯補償他預言的能力。赫拉的嫉妒在希臘神話中不僅是性格缺陷，更是推動大量英雄敘事的重要動力——她對赫拉克勒斯的敵意從側面塑造了這位英雄的十二項考驗故事。

**4. 特洛伊戰爭中的立場**
赫拉在特洛伊戰爭中全力支持希臘聯軍，因帕里斯（Paris）評判金蘋果時將「最美者」判給阿芙蘿黛蒂（Aphrodite）而非赫拉。她多次干預戰局：以美色引誘宙斯以分散其注意力，讓希臘人得以扭轉戰局；派遣睡眠神Hypnos使宙斯沉睡，使波賽頓得以率領希臘人衝鋒。她的憤怒成為荷馬史詩《伊利亞特》中推動情節的關鍵力量之一。

## 跨文化對應

| 文化 | 對應神祇 | 關係說明 |
|------|---------|---------|
| 羅馬 | Juno | 完全對應，每月初的Calends獻祭 |
| 伊特魯里亞 | Uni | 融合為三主神之一 |
| 埃及 | Isis | 作為天后與母神的角色相似 |
| 美索不達米亞 | Antu / Inanna | 天神Anu之配偶，天后的對應 |
| 北歐 | Frigg | 婚姻守護與預知未來的角色對應 |
| 印度 | Indrani (Sachi) | 因陀羅之妻，天后地位相似 |
| 中國 | 西王母 | 女仙之首、天后地位可比較 |
