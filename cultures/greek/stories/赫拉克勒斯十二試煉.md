# 赫拉克勒斯十二試煉 (The Twelve Labours of Heracles)

- **文化：** 希臘神話 (Greek Mythology)

## 故事背景

赫拉克勒斯(Heracles)爲宙斯與凡人阿爾克墨涅(Alcmene)之子。天后赫拉因丈夫出軌而終生憎恨赫拉克勒斯，在其年幼時派遣二蛇入搖籃，反被嬰兒赫拉克勒斯掐死。赫拉克勒斯成年後娶底比斯公主墨伽拉(Megara)爲妻，赫拉使其發狂，在瘋癲中殺害妻子與子女。清醒後悲痛欲絕，請求德爾斐神諭指示贖罪之路——神諭命他爲邁錫尼國王歐律斯修斯(Eurystheus)服役十二年，完成十項不可能任務(後增爲十二項)。

## 十二試煉

| 試煉 | 對手 | 象徵 |
|------|------|------|
| 1. 尼米亞獅子 | 刀槍不入的巨獅 | 不可戰勝的自然力量 |
| 2. 勒耳那水蛇 | 九頭毒蛇(Hydra)，砍一首生二首 | 生生不息的邪惡 |
| 3. 刻律涅亞牝鹿 | 金角銅蹄的神鹿，阿緹蜜絲聖獸 | 速度與神聖不可侵犯 |
| 4. 厄律曼托斯野豬 | 肆虐山野的巨豬 | 狂暴的破壞力 |
| 5. 奧格阿斯牛棚 | 一日內清掃三千牛糞的骯髒任務 | 以智慧(引河沖刷)解決羞辱性考驗 |
| 6. 斯廷法利斯湖怪鳥 | 銅翼鐵喙的食人怪鳥 | 空中的威脅 |
| 7. 克里特公牛 | 米諾斯王國的發狂公牛 | 海洋力量與馴服 |
| 8. 狄俄墨得斯牝馬 | 食人牝馬，主人爲戰神之子 | 以暴制暴 |
| 9. 希波呂忒的腰帶 | 亞馬遜女王的神奇腰帶 | 獲得異族文明的認可 |
| 10. 革律翁的牛羣 | 三體巨人的牛羣 | 穿越極限(直布羅陀海峽)的遠征 |
| 11. 赫斯佩里得斯金蘋果 | 巨神阿特拉斯看守的金蘋果 | 欺騙與智慧戰勝蠻力 |
| 12. 地獄三頭犬刻耳柏洛斯 | 活捉冥界守門犬 | 征服死亡本身的終極考驗 |

## 英雄之旅原型

赫拉克勒斯十二試煉是約瑟夫·坎貝爾(Joseph Campbell)「英雄之旅」單一神話的典範：

- **召喚：** 殺害家人後的贖罪需求
- **門檻：** 進入歐律斯修斯王宮服侍
- **考驗：** 十二試煉逐一克服
- **最深洞穴：** 活捉地獄三頭犬——穿越生死邊界
- **獎賞：** 不朽與奧林帕斯神籍
- **回歸：** 死後火化，升天成神，與赫拉和解

赫拉克勒斯的故事通過伊特魯里亞傳入羅馬(稱爲Hercules)，成爲羅馬最受歡迎的神祇之一。其十二試煉的主題在文藝復興後的西方藝術、文學、心理學(弗洛伊德/榮格)中被反覆詮釋，成爲西方文明中最持久的英雄原型。
