# 伊瑪的黃金時代與地下堡壘 (Yima's Golden Age and the Vara)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)

## 故事背景

伊瑪（Yima，中古波斯語 Jamshid / Jam）是波斯神話中人類的始祖與第一代王者。他是維萬格漢特（Vivanghant）之子，被阿胡拉·馬茲達選中肩負保存生物種子的使命。伊瑪的故事在《阿維斯塔》的〈萬迪達德〉（Vendidad）第二章中有最完整的記述，同時在菲爾多西的《王書》（Shahnameh）中以「賈姆希德王」的形象獲得史詩性擴展。他是人類黃金時代的統治者，也被稱為「擁有良好畜群者」，因為在他的統治下地球上的牲畜和人類從未減少。

伊瑪的文化地位極為特殊——他既非神也非先知，而是神與人之間的中介者。在瑣羅亞斯德教的宇宙史觀中，伊瑪屬於「第一紀元」（Ahura Mazda 以靈創造世界的階段最早期）的核心人物，他的故事銜接了創世神話與大洪水神話，構成了波斯宇宙循環論中的重要一環。

## 情節

### 第一幕：阿胡拉·馬茲達的委任

阿胡拉·馬茲達邀請伊瑪成為他的信仰傳播者，但伊瑪謙卑地拒絕了——他說自己不適合承擔這種使命。於是阿胡拉·馬茲達賜予伊瑪另一項任務：成為人類的保護者與牧者，統治世界並繁衍萬物。伊瑪接受了這個使命。阿胡拉·馬茲達將金色權杖與金戒指交給伊瑪，授予他統治人間的王權。

### 第二幕：黃金時代的三次擴張

在伊瑪的統治下，世界經歷了三次非凡的擴張。每一次地球的承載力達到極限時，伊瑪便以金戒指擊打大地，以金色權杖刺入土地——大地隨之擴張三分之一。人類與牲畜的數量因此得以增長，飢荒與疾病從未降臨。這個時期被稱為「伊瑪的黃金時代」——沒有寒冷、沒有炎熱、沒有衰老，人們以永恆的青春生活。

### 第三幕：伊瑪的墮落

《王書》中的版本更為戲劇化。賈姆希德（伊瑪）統治了七百年，發明了武器、紡織、建築、醫學，並以魔法製作了七環金杯以觀測宇宙。然而他的驕傲吞噬了他——他宣稱自己是神，命令臣民向他朝拜。這時阿胡拉·馬茲達收回了他的神聖榮耀（Xvarenah / Farr），伊瑪的權力瞬間崩潰。暴君佐哈克（Zahhak / 蛇王）趁機起兵推翻伊瑪，伊瑪被迫逃亡，最終在一場風暴中被鋸為兩半。

### 第四幕：嚴冬與 Vara

在伊瑪墮落之前，阿胡拉·馬茲達早已預見世界將因安格拉·曼紐的入侵而面臨毀滅性的嚴冬。阿胡拉·馬茲達指示伊瑪建造一座巨大的地下堡壘（Vara，意為「圍牆」）——方形結構，每邊長一箭之地（約數公里）。伊瑪在每層減小一圈的階梯式結構中存放了所有物種的種子、每種植物、所有人的家族（最完美、最強大者各一對），以及儲備的種子與食物。Vara 內的時間規則與外界不同——居住者以一年為一日，生命延長至一百五十年。嚴冬過後，Vara 中保存的生命種子重新繁衍於大地之上。

## 跨文化平行

伊瑪的 Vara 是人類神話中最深刻的洪水/災難避難敘事之一，與世界各地的滅世重生神話形成系統性平行：

| 文明 | 故事 | 關鍵對應 |
|------|------|----------|
| 美索不達米亞 | 烏特納匹什提 (Utnapishtim) 方舟 | 眾神以洪水毀滅人類，一位義人建造方舟保存生命 |
| 希伯來 | 諾亞方舟 (Noah's Ark) | 上帝降洪水，諾亞保存各類生物一對 |
| 希臘 | 杜卡利翁 (Deucalion) 的方舟 | 宙斯降洪水，杜卡利翁與皮拉向後丟石生人 |
| 印度 | 摩奴 (Manu) 與魚 | 毗濕奴化身大魚引領摩奴方舟，保存生命種子 |
| 中國 | 大禹治水 / 女媧補天 | 洪水治理（非避難），女媧止水補天 |
| 北歐 | 貝爾格爾密爾 (Bergelmir) | 霜巨人僅存者乘舟逃過伊米爾血肉洪水 |
| 瑪雅 | 《波波爾·伏》木人毀滅 | 眾神以洪水毀滅木製人類 |

伊瑪黃金時代中「大地三次擴張」的母題特別值得注意——這與中美洲神話中五太陽紀的人類擴張/毀滅循環在結構上驚人相似。《班達希申》中記載的宇宙一萬二千年循環也暗示了人類將經歷七個千紀的興衰。
