# 班達希申 — 波斯創世記

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)

## 故事背景

《班達希申》（Bundahishn，中古波斯語意為「原初創造」）是瑣羅亞斯德教最重要的宇宙起源文本，完成於九世紀的薩珊王朝後期，奠基於更古老的阿維斯塔經文與口傳傳統。它描述了宇宙從精神創造到物質創造的完整過程、安格拉·曼紐（Ahriman）的入侵、以及世界一萬二千年歷史的終末結局。《班達希申》是波斯神話中最接近聖經《創世紀》的文本，但其宇宙論規模遠大於聖經——它的時間尺度跨越一萬二千年，空間尺度涵蓋天界、物質世界與深淵三重結構。

## 情節

### 第一幕：精神創造（六千年的前奏）

在時間的開端，只有阿胡拉·馬茲達存在——他居住在無限光明之中，全知全能。在深淵的另一端，安格拉·曼紐居住在無盡黑暗之中。阿胡拉·馬茲達以靈體（menog）形式創造了第一個世界——精神世界。他創造了星辰、水火、草木的靈魂原型，以及六位大天使（Amesha Spenta）作為他的輔佐。精神世界是完美的——沒有疾病、沒有死亡、沒有時間的流動。

安格拉·曼紐因為無法看到光明而在黑暗中暴怒。他起身攻擊精神世界，但被阿胡拉·馬茲達以神聖禱詞（Ahuna Vairya）擊退。安格拉·曼紐退回深淵，昏迷了整整三千年。

### 第二幕：物質創造（阿胡拉·馬茲達七階段創造）

阿胡拉·馬茲達在安格拉·曼紐昏迷的三千年中，開始了物質世界（getig）的創造——這是精神世界的物質對應體。他分七個階段創造：

1. **天空**（第一日）— 以透明石材打造，如雞蛋般包圍世界，抵禦安格拉·曼紐的入侵
2. **水**（第二日）— 創造於世界下方，清潔而流動
3. **大地**（第三日）— 平坦而廣闊，位於水之上
4. **植物**（第四日）— 第一株植物在土地中央生長
5. **原始牛**（第五日）— Gav-aevo-data，獨一的牛，象徵動物世界的根源
6. **原始人**（第六日）— Gayomart，獨一的人，象徵人類的根源，光芒四射
7. **火**（第七日）— 阿胡拉·馬茲達自身存在的象徵，穿透並維持一切創造

### 第三幕：安格拉·曼紐的入侵

三千年結束，安格拉·曼紐從深淵甦醒。他召集了他所有的邪魔（Daevas），穿過天空的孔隙入侵物質世界：

- **對水的攻擊**：他使海水變鹹，創造了毒蛙與毒蛇污染水源
- **對大地的攻擊**：他創造了沙漠、山脈裂縫與毒蟲
- **對植物的攻擊**：他使植物枯萎，引入荊棘與毒草
- **對牛的攻擊**：他殺死原始牛——牛死後化為五穀、藥草與牲畜的靈魂從身體升起
- **對人的攻擊**：他殺死 Gayomart——Gayomart 死前將精子委託給大地。四十年後，一株大黃植物從土地中長出，從中生出第一對人類：Mashya（男）與 Mashyana（女）

阿胡拉·馬茲達介入——他將原始牛的精髓帶到天界，使之成為動物靈魂的守護者。Gayomart 的死也非徒然——從他的身體中，七種金屬從大地的七個方向流出。人類從此開始了在善惡混雜的物質世界中的生活。

### 第四幕：火的神聖使命

阿胡拉·馬茲達派遣了三類神聖之火守護世界：
- **Fire of Farnbag**（祭司之火）— 守護祭司階層與宗教傳承
- **Fire of Gushnasp**（戰士之火）— 守護武士階層與王權
- **Fire of Burzin-Mihr**（農民之火）— 守護農民與牧人，象徵生產與繁榮

這三把火在薩珊王朝時期被供奉在三大聖火廟中——至今仍有部分延續燃燒。

## 文化意義

《班達希申》的創世故事是理解波斯世界觀的基礎：

1. **善惡二元的本體論**：世界不是由單一神創造的完美產物，而是善惡兩股力量持續鬥爭的戰場
2. **人類的雙重責任**：每個人通過選擇善惡參與這場宇宙鬥爭
3. **物質世界不是邪惡的**：與諾斯替教不同，物質世界是阿胡拉·馬茲達對精神世界的完美對應，只是被安格拉·曼紐污染
4. **宇宙的有限性**：邪惡終將被消滅——歷史朝向一個確定的救贖終點前進

## 跨文化平行

| 元素 | 波斯 | 對應文化 |
|------|------|---------|
| 七階段創造 | 創世紀六天＋安息日 | 希伯來 |
| 原始人被殺後生出人類 | Purusha 被殺化身世界 | 印度 |
| 原始牛死後化為植物與藥草 | Ymir 的牛 Audhumla 與 Ymir 化世界 | 北歐 |
| 安格拉·曼紐入侵完美世界 | 弗栗多困鎖水、阿波菲斯攻擊太陽船 | 印度、埃及 |
| 三把聖火 | 希臘三女神 Hestia 的灶火／印度三火（Ahavaniya、Garhapatya、Dakshina） | 希臘、印度 |
