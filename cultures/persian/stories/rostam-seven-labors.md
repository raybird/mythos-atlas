# 羅斯坦七試煉 (Haft Khān-e Rostam / 羅斯坦的七重關)

- **文化：** 波斯神話／《王書》(Shahnameh)
- **時期：** 傳說中的凱揚王朝時期

## 故事背景

羅斯坦（Rostam）是波斯史詩《王書》（Shahnameh，菲爾多西著）中最偉大的英雄，地位相當於希臘神話的赫拉克勒斯或印度史詩的阿周那。他出身於薩姆家族（Sam / Nariman），是傳奇勇士扎爾（Zāl）之子，以超凡力量、忠誠與智慧聞名於世。

「七試煉」（Haft Khān）發生於凱·卡武斯王（Kay Kāvus）被馬贊德蘭（Māzandarān）的白魔（Div-e Sepid）俘虜之後。羅斯坦奉命穿越七重險境前往營救。每一道關卡不僅是物理上的考驗，更是對英雄毅力、智慧與神聖榮光（xvarenah）的極致挑戰。

## 情節（七重關）

**第一關：戰獅。** 羅斯坦在荒野中休息時，一頭巨獅襲擊他的戰馬拉克什（Rakhsh）。羅斯坦醒來徒手搏殺獅子，但拉克什早一步將獅子踩死。此關考驗英雄與坐騎的默契。

**第二關：穿越沙漠。** 羅斯坦穿越無邊焦渴沙漠，水源斷絕。他以薩滿式的冥想祈禱，獲得神祇指引，在絕望中發現清泉。此關象徵靈性匱乏中的信仰支撐。

**第三關：屠龍。** 一條巨大妖龍襲擊羅斯坦。在拉克什的警示下羅斯坦驚醒，以寶劍斬斷龍首。龍之血液污染大地七日不散——菲爾多西以此描繪黑暗力量被淨化前的最後掙扎。

**第四關：殺女巫。** 羅斯坦在一座華美宮殿中遇見一位絕世美女，實為馬贊德蘭女巫所化。他以對真神的祈禱識破幻象，以寶劍斬殺女巫。此關考驗辨別真偽的智慧，對應赫拉克勒斯辨別「歡愉之路」與「美德之路」的經典抉擇。

**第五關：擒盜。** 羅斯坦與馬贊德蘭強盜首領奧拉德（Aulād）交戰。羅斯坦擊敗奧拉德，並脅迫他為嚮導，引領通往白魔巢穴之路。這一關展現羅斯坦的謀略：以敵制敵。

**第六關：戰阿爾藏格魔。** 羅斯坦潛入馬贊德蘭城，於深夜將魔將阿爾藏格（Arzhang）一劍斬首。城中陷入混亂，羅斯坦趁亂向白魔大本營進發。普羅米修斯式夜襲戰術展現了英雄的隱忍與精準。

**第七關：戰白魔。** 最終對決——羅斯坦與巨魔白魔（Div-e Sepid，白色惡魔）展開血戰。白魔以巨石與黑暗魔法攻擊，羅斯坦先以寶劍削去其臂，再以匕首刺入其心臟。白魔死後從其胃中取出卡武斯王的雙眼（被白魔挖出），羅斯坦以鳳凰之羽醫治國王，恢復其視力。英雄以拯救君王完成使命，回歸秩序世界。

## 跨文化平行

七重試煉的敘事結構遍布世界英雄神話——約瑟夫·坎貝爾（Joseph Campbell）將其歸納為「英雄的千面之路」中的考驗階段：

| 羅斯坦七試煉 | 跨文化對應 | 文明 |
|---|---|---|
| 戰獅 | 涅墨亞獅子（赫拉克勒斯第一試煉） | 希臘神話 |
| 穿越沙漠 | 穿過冥界的伊南娜 | 美索不達米亞 |
| 屠龍 | 貝奧武夫戰格倫德爾之母 | 北歐／英格蘭 |
| 殺女巫（幻象考驗） | 奧德賽中的賽蓮女妖 | 希臘神話 |
| 擒敵為嚮導 | 赫拉克勒斯綁架特洛伊使者 | 希臘神話 |
| 夜襲魔將 | 奧丁潛入瓦夫斯魯德尼爾 | 北歐神話 |
| 終戰白魔救王 | 庫丘林戰三眼魔犬 | 凱爾特神話 |
| 眼復明（鳳凰之羽） | 荷魯斯之眼復原 | 埃及神話 |

此史詩最驚人的跨文化特徵在於：**七關的結構與赫拉克勒斯的十二試煉**呈現高度平行。學術界普遍認為，兩者共享了印歐英雄神話的原型（Indo-European Hero Cycle），並在亞歷山大東征後的希臘化交流中進一步相互影響。
