# 拉什努 (Rashnu)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 公正、審判、靈魂秤量、真理（Asha）的裁決者
- **象徵：** 金天平、權杖、光之審判座
- **別名：** Rashnu Razishta（最公正的拉什努）

## 概述

拉什努（阿維斯塔語 Rashnu，中古波斯語 Rashn）是瑣羅亞斯德教中負責審判靈魂的公正之神，其名直接意為「公正」或「審判」。在波斯萬神殿中，他與密特拉（Mithra）、斯勞沙（Sraosha）共同組成靈魂審判的三位一體——密特拉以千眼監視人間善惡，斯勞沙引導靈魂渡過欽瓦特橋，拉什努則在橋頭執掌天平，以絕對的精確性秤量每一個靈魂的善行與惡行。他是阿沙（Asha，真理與秩序）的人格化執法者，職能核心在於將宇宙道德律轉化為個人的最終裁決。

拉什努雖然不如阿胡拉·馬茲達或密特拉那樣廣為人知，但在瑣羅亞斯德教的宇宙論與末日論中佔據不可替代的位置。《阿維斯塔》的〈拉什努·耶什特〉（Rashnu Yasht，Yasht 12）是專門獻給他的頌歌，稱他為「最純淨的神聖存在」、「無可欺騙的審判者」。他的名字在波斯文化中演化為成語「以拉什努之名起誓」（Ba-Rashn），表示最嚴肅的誓言。

## 神話事蹟

拉什努的神話功能集中體現在欽瓦特橋（Chinvat Bridge）的靈魂審判場景中。根據《萬迪達德》（Vendidad）第十九章的描述，人的靈魂在死後第三天清晨抵達欽瓦特橋頭。此時拉什努以金天平出現，將生前的善行（以善思、善言、善行計算）置於一端，惡行（惡思、惡言、惡行）置於另一端。天平的精確度令人敬畏——沒有任何行為可以被隱藏，連最微小的念頭都將被秤量。如果善行一方重於惡行，橋樑便會變得寬闊平坦，靈魂在斯勞沙的引導下安然通過，前往永恆光明中的天堂（Garōdmāna）。如果惡行一方更重，橋樑將化為利刃般細窄，靈魂顫抖墜入深淵（Drujō-Demāna），承受懲罰。若善惡持平，靈魂則進入哈梅斯塔甘（Hamestagan），一個中陰狀態，等待末日的最終審判。

拉什努被認為擁有「不受任何欺騙的能力」——他不僅秤量外在行為，也秤量內在動機。一個人若表面行善但心懷惡意，拉什努的天平會如實反映。這種「內在動機審判」的概念在宗教史上是極為先進的倫理思想，遠遠超越了單純的「行為後果」評判。

在《班達希申》（Bundahishn）的末日論中，拉什努在宇宙年的終末階段再次登場。當救世主薩奧希揚特（Saoshyant）降臨、死者復活時，拉什努將主持最終的全面審判。這一次不再只是個別靈魂的秤量，而是全人類的集體道德裁決——正義者將在重建的世界中享受永恆幸福，邪惡者將被徹底淨化消滅。

## 跨文化對應

| 文化 | 對應神祇／存在 | 共通點 |
|------|---------------|--------|
| 埃及 | 瑪亞特（Ma'at） | 真理/正義女神，以羽毛秤量死者心臟——與拉什努的天平審判在儀式上幾乎一致 |
| 希臘 | 米諾斯（Minos）、拉達曼提斯（Rhadamanthys） | 冥界三法官之一，衡量靈魂歸處 |
| 印度 | 閻魔（Yama） | 死神兼公正審判者，以業力簿記載靈魂善惡 |
| 中國 | 泰山府君／閻羅王 | 冥界審判官，以善惡簿裁決靈魂去處 |
| 北歐 | 無特定審判者（奧丁挑選英靈） | 相較之下北歐缺乏系統性的道德秤量審判 |
| 基督教 | 聖米迦勒（Archangel Michael） | 在最後審判中秤量靈魂的天使長，形象直接受瑣羅亞斯德教影響 |

拉什努與埃及瑪亞特的平行尤其深刻——兩者皆以天平為核心象徵，皆以絕對真理（Asha / Ma'at）為秤量標準，皆強調審判的不可賄賂性。這顯示了古代文明對「死後正義」的普世渴望：在世間無法實現的公平，將在來世由一雙絕對公正的手來執行。

## 相關神祇

- **Ahura Mazda**：授予拉什努審判權力的至高源頭
- **Mithra**：契約與光明之神，擔任審判的監視者與見證者
- **Sraosha**：聽從之神，引導靈魂過橋的同行者
- **Vohu Manah**：善念聖者，在天堂入口迎接善良靈魂
- **Angra Mainyu**：惡靈，試圖在審判時搶奪靈魂的對手
