# 阿胡拉·馬茲達 (Ahura Mazda)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 至高神、創造主、智慧、光明、秩序（Asha）、善的化身
- **象徵：** 帶翼圓盤（Faravahar）、火焰、光明、白馬
- **對立面：** 安格拉·曼紐 (Angra Mainyu / Ahriman)，破壞之靈

## 概述

阿胡拉·馬茲達（意為「智慧之主」）是瑣羅亞斯德教（Zoroastrianism）的至高神，也是人類歷史上最早的一神論或多神論向一神論過渡的核心代表。他是創造主，以神聖之靈（Spenta Mainyu）創造了宇宙、星辰、水火、植物與人類。阿契美尼德王朝（Achaemenid Empire）的大流士一世在貝希斯敦銘文中宣稱：「阿胡拉·馬茲達賜我王權，助我征服天下。」自此這位神祇成為波斯帝國官方信仰的核心。

在瑣羅亞斯德教的宇宙觀中，阿胡拉·馬茲達居住於無限光明之中，以六位神聖存在（Amesha Spenta，意為「不朽聖者」）為輔佐——包括沃胡·瑪納（Vohu Manah，善念）、阿沙·瓦希什塔（Asha Vahishta，至真）、赫沙特拉·瓦伊里亞（Khshatra Vairya，理想王權）、斯彭塔·阿爾邁蒂（Spenta Armaiti，虔敬）、豪爾瓦塔特（Haurvatat，完好）與阿梅雷塔特（Ameretat，不朽）。這七位一體的聖者構成了善界的最高階層。

## 神話事蹟

阿胡拉·馬茲達的創世故事記載於《班達希申》（Bundahishn，意為「原初創造」）之中。他首先創造了天空（以石為材）、水、大地、植物、原始牛（Gavaevodata）與原始人（Gayomart），以及純淨之火。但安格拉·曼紐從深淵中衝出，殺死了原始牛與原始人，將疾病與死亡引入世界。然而阿胡拉·馬茲達的創造力並未因此終結——原始牛死後，其骨髓化為五穀與藥草；Gayomart 的精子被土吸收，四十年後長出一株大黃植物，從中誕生了第一對人類 Mashya 與 Mashyana。

阿胡拉·馬茲達也出現在《王書》（Shahnameh）的宇宙背景中，雖然菲爾多西的作品已帶有伊斯蘭色彩，但他仍然保留了「世界之主」（Khudavand-e Jahan）這一概念，作為正義與秩序的最終守護者。瑣羅亞斯德教的終末論中，阿胡拉·馬茲達將在宇宙年的最後階段派出生救世主（Saoshyant），使死者復活，對人類進行最終審判，徹底消滅邪惡。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 印度吠陀 | 阿修羅·伐樓那 (Asura Varuna) | 皆為契約與宇宙秩序的守護者，阿胡拉與阿修羅同源 |
| 希臘 | 宙斯 (Zeus) | 至高神、眾神之王、秩序維護者 |
| 北歐 | 奧丁 (Odin) | 智慧追求、犧牲精神、宇宙秩序的守護 |
| 迦南 | 埃爾 (El) | 至高神、創造主、眾神之父 |
| 中國 | 昊天上帝 | 天的化身，宇宙至高主宰，道德秩序的最終來源 |
| 埃及 | 拉 (Ra) | 太陽創造神，光明與秩序的化身，對抗混沌之蛇阿波菲斯 |

## 相關神祇

- Spenta Mainyu（聖靈）：阿胡拉·馬茲達的創造性面向，善的直屬執行者
- Angra Mainyu（惡靈）：永恆的對立面，混沌與毀滅的化身
- Mithra（密特拉）：契約與光明之神，阿胡拉·馬茲達的重要助手
- Anahita（阿娜希塔）：水與豐收女神
- Verethragna（勝利之神）：戰爭與勝利的化身
- Rashnu（公正之神）：靈魂渡橋時的審判者
