# 阿娜希塔 (Anahita)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 水、豐收、生育、醫治、智慧、戰神（特定地區）
- **象徵：** 流水、蓮花、海狸皮袍、黃金冠冕與項鍊、聖馬
- **全名：** Aredvi Sura Anahita（「濕潤、強壯、純淨」）

## 概述

阿娜希塔（阿維斯塔語 Aredvi Sura Anahita，中古波斯語 Ardwisur Anahid）是波斯萬神殿中最偉大的女神，掌管世間一切水流——從宇宙之河到溪流泉水。她是生命的給予者與淨化者，同時也具備強烈的戰士特質。她的地位在瑣羅亞斯德教中僅次於阿胡拉·馬茲達，與密特拉並列為最主要的神聖存在。

阿契美尼德王朝的國王阿爾塔薛西斯二世（Artaxerxes II，約西元前 404–359 年）在全帝國範圍內推動將阿娜希塔納入官方祭祀體系，在蘇薩、波斯波利斯、巴比倫等地為她建造了宏偉的聖殿。此後她在波斯民間信仰中的地位逐漸上升，影響力持續到薩珊王朝甚至伊斯蘭時代，她的身影出現在波斯細密畫與詩歌（如哈菲茲、菲爾多西的作品）中，轉化為愛情與美的象徵。

## 神話事蹟

在《阿維斯塔》的〈阿班·耶什特〉（Abān Yasht，Yasht 5）中，阿娜希塔被詳細描述為一位駕馭四匹白馬的絕美女神。她身披金色織錦斗篷，頭戴一百顆星辰的黃金冠冕，腰繫金絲帶。她居住在浩瀚宇宙之河（Aredvi）的上游源頭——這條河從哈拉（Hara）神山之巔奔流而下，分為一千條河川，灌溉大地萬物。

阿娜希塔的最重要神話功能是淨化與賦予生命力。她與人類的潔淨儀式有著根本聯繫——在瑣羅亞斯德教中，接觸死物後需要以流水或牛尿（nirang）進行九夜潔淨儀式，這些皆出自阿娜希塔的律法。她也負責賜予婦女順產、為母馬賦予乳汁、為戰士賜予勝利。波斯傳統中，戰士在出征前必向阿娜希塔禱告，祈求她以水洗去敵人的力量。

傳說中，伊朗傳說英雄伊瑪（Yima）、凱·霍斯魯（Kay Khosrow）、甚至眾先知 Zarathustra 本人皆曾向阿娜希塔獻祭，祈求她的庇護。雖然 Zarathustra 在祈禱中要求的是「使人類依正義行動」的祝福，而非物質財富，阿娜希塔仍然顯示了她的神聖青睞。在後世的波斯史詩《王書》中，阿娜希塔轉化為「伊朗之水（Āb-e Īrān）」的守護者——當伊朗陷入旱災時，她從星辰之間降下雨水，拯救王國。

## 跨文化對應

| 文化 | 對應女神 | 共通點 |
|------|----------|--------|
| 美索不達米亞 | 伊什塔爾 (Ishtar / Inanna) | 愛與戰女神，清晨之星，水之掌管者 |
| 希臘 | 阿芙洛狄忒 (Aphrodite) / 阿耳忒彌斯 (Artemis) | 愛與豐收；純潔狩獵與生育守護 |
| 印度吠陀 | 娑羅室伐底 (Saraswati) | 聖河女神，智慧與純淨的化身 |
| 埃及 | 伊西斯 (Isis) | 豐收、生育、醫治、魔法、聖水女神 |
| 羅馬 | 維納斯 (Venus) / 黛安娜 (Diana) | 愛神；生育與自然守護 |
| 北歐 | 芙蕾雅 (Freyja) | 愛、豐收、戰爭、魔法、生育女神 |

## 相關神祇

- Ahura Mazda（阿胡拉·馬茲達）：賦予阿娜希塔神聖地位的最高主神
- Mithra（密特拉）：太陽與契約之神，與阿娜希塔共享光明與水的綜合象徵
- Tishtrya（降雨之星）：天狼星之神，每夜與旱魔 Apaosha 戰鬥，帶來雨水
- Haurvatat（完好聖者）：Amesha Spenta 之一，守護水的聖者
- Vayu（風神）：風、空間、大氣之神，與阿娜希塔的流動之水互相呼應
