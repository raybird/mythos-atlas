# 斯彭塔·曼紐 (Spenta Mainyu)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 聖靈、阿胡拉·馬茲達的創造性面向、善的直接執行者
- **象徵：** 火焰、光明、白色公牛
- **對立面：** 安格拉·曼紐 (Angra Mainyu)，邪惡之靈

## 概述

斯彭塔·曼紐（Spenta Mainyu，阿維斯塔語意為「神聖的靈／豐饒之靈」）是阿胡拉·馬茲達的創造性靈體化身，在瑣羅亞斯德教神學中擔任至高神與物質世界之間的中介。他不是獨立于阿胡拉·馬茲達的獨立神祇，而是阿胡拉·馬茲達的內在屬性——如同太陽的光與熱之于太陽本身。在《阿維斯塔》的《耶斯納》（Yasna）第 30 章中被稱為「雙靈」（Mainyū）之一，與安格拉·曼紐形成永恆對立。斯彭塔·曼紐代表生命、真理、秩序（Asha）與創造，安格拉·曼紐則代表死亡、謊言、混沌（Druj）與毀滅。

在瑣羅亞斯德教的宇宙論中，斯彭塔·曼紐是七位阿梅沙·斯彭塔（Amesha Spenta，不朽聖者）之首，也是阿胡拉·馬茲達在物質世界中的直接代理人。他教導人類耕種、畜牧、建立秩序，並且是在終末之戰中率領善的力量最終擊敗邪惡的指揮官。學者 Mary Boyce 指出，斯彭塔·曼紐的概念可能深刻影響了後期猶太教、基督教與伊斯蘭教中的「聖靈」（Holy Spirit / Rūḥ al-Qudus）觀念。

## 神話事蹟

### 雙靈的選擇

《耶斯納》第 30 章記載了瑣羅亞斯德教最核心的神學文本——雙靈之歌（Yasna 30.3–5）：太初之時，斯彭塔·曼紐與安格拉·曼紐作為孿生雙靈並存於阿胡拉·馬茲達之中。二者各自做出了根本性的選擇——斯彭塔·曼紐選擇了真理（Asha）與生命，安格拉·曼紐選擇了謊言（Druj）與死亡。這一選擇既是宇宙性的（決定了世界史的方向），也是個人性的（每個人類必須複製這樣的選擇）。由此，世界成為善惡陣營的戰場，人類成為自由意志的承載者。

### 創世中的角色

在阿胡拉·馬茲達的創世行動中，斯彭塔·曼紐實際執行了創造的具體工作。《班達希申》描述阿胡拉·馬茲達以靈（Mēnōg）創造世界，斯彭塔·曼紐則將其轉化為物質（Gētīg）存在。他是將七階創造（天空、水、大地、植物、動物、人類、火）從潛在狀態實現為具體形式的推動力。當阿胡拉·馬茲達創造伽約馬特（Gayomart）與原始牛時，斯彭塔·曼紐賦予了他們靈性生命。

### 保護與復興

斯彭塔·曼紐在世界史中持續保護正義者，抵抗安格拉·曼紐的侵蝕。他在伊瑪的 Vara（地下堡壘）建造中提供了神聖指引，確保生物的種子在毀滅性嚴冬中得到保存。在終末論中，斯彭塔·曼紐將在三千年末的宇宙循環中注入火河（River of Fire）——熔化的金屬將流遍大地，正義者如沐溫水，邪惡者則承受燒灼痛苦，完成最後淨化。此後斯彭塔·曼紐將協助救世主（Saoshyant）使死者復活，重建完美的世界。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 基督教 | 聖靈 (Holy Spirit) | 神的創造性／使人活的靈，中介者功能 |
| 印度吠陀 | 普拉納 (Prāṇa) | 生命之氣，創造的推動力 |
| 希臘 | 普紐瑪 (Pneuma) | 世界之靈，賦予生命的呼吸 |
| 埃及 | 舒 (Shu) | 空氣之神，分離天地，生命之氣 |
| 北歐 | 奧丁的靈魂 (Önd) | 賦予人類靈魂的眾神之禮 |
| 佛教 | 法身 (Dharmakāya) | 佛的終極實相面向 |

斯彭塔·曼紐與聖靈之間的歷史關係尤為關鍵。學者普遍認為，猶太教第二聖殿時期（公元前 516–70 年）波斯統治的影響使得耶和華的「靈」（Ruah）開始被視為獨立位格，最終影響了基督教三位一體神學的形成。

## 相關神祇

- 阿胡拉·馬茲達：斯彭塔·曼紐的來源與歸宿，不可分割的一體
- 安格拉·曼紐：孿生對立者，邪惡之靈
- 阿梅沙·斯彭塔（Amesha Spenta）：六位不朽聖者，善界最高階層
- 阿沙·瓦希什塔（Asha Vahishta）：至真，斯彭塔·曼紐的親密夥伴
- 沃胡·瑪納（Vohu Manah）：善念，引導人類選擇真理
