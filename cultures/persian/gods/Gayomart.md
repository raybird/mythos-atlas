# 伽約馬特 (Gayomart)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 原人、人類始祖、有終之生命
- **象徵：** 大黃植物、大地、金屬
- **對立面：** 安格拉·曼紐 (Angra Mainyu) 所殺

## 概述

伽約馬特（Gayomart，阿維斯塔語 Gayō.marətan，意為「有終之生命」）是瑣羅亞斯德教創世神話中的第一個人類原型，對應於印度神話中的原人普魯沙（Purusha）、北歐神話中的伊米爾（Ymir）以及中國神話中的盤古。《班達希申》（Bundahishn）描述他由阿胡拉·馬茲達以土元素創造，身形巨大、光芒四射，如同太陽。他並非普通的人類祖先，而是宇宙原初犧牲的核心——他的死亡啟動了生物多樣性與人類種族的誕生。

伽約馬特居住在埃蘭維傑（Airyanem Vaejah，意為「雅利安人的原鄉」，神話地理中的世界中心），與原始牛（Gavaevodata）共同生活在純淨的世界中。此時世界尚處在阿胡拉·馬茲達的靈性創造階段（Mēnōg），尚未進入物質階段（Gētīg）。伽約馬特代表人類尚未墮落、尚未死亡的完美原初狀態。

## 神話事蹟

### 安格拉·曼紐的入侵

阿胡拉·馬茲達創造世界後，安格拉·曼紐（Angra Mainyu）從無盡黑暗中衝出，闖入光明的創造界。他首先攻擊伽約馬特與原始牛——但由於阿胡拉·馬茲達的保護，他無法直接殺死伽約馬特。他改為引誘原始牛走出——原始牛被毒害而亡，伽約馬特則在痛苦中存活了三十年。

### 原初犧牲與生命的起源

原始牛死後，其骨髓化為五十五種穀物與十二種藥草，其精子被淨化後送入月亮的保護之中，誕生了所有動物物種。伽約馬特在垂死之際，他的身體中的金屬元素散入大地——由此大地獲得了七種金屬礦藏，包括黃金、白銀、銅、鐵等。他倒下時，身體朝向南方，精子落入大地。四十年後，從精子所在之處長出一株大黃植物（rēwas，在《班達希申》中記載），其枝葉化為第一對人類伴侶——瑪什亞（Mashya）與瑪什亞娜（Mashyana）。

### 人類的起源與初始墮落

瑪什亞與瑪什亞娜最初是純善的，但安格拉·曼紐以謊言誘惑他們——使他們相信他是世界的創造者。他們開始崇拜惡神，從說謊開始逐步失去純真。阿胡拉·馬茲達因此使他們的壽命縮短，並將他們置於善惡之間的永恒鬥爭之中。從瑪什亞與瑪什亞娜繁衍出所有人類種族，人類從此成為善惡戰場上的核心參與者。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 北歐 | 伊米爾 (Ymir) | 巨人被殺，身體化為世界 |
| 印度 | 普魯沙 (Purusha) | 原人獻祭，化身宇宙萬物 |
| 中國 | 盤古 | 巨人開天闢地，死後化身萬物 |
| 伊朗—美索不達米亞 | 阿特拉哈西斯 (Atrahasis) | 人類由神犧牲創造 |
| 希臘 | 普羅米修斯 (Prometheus) | 以土造人，與神關聯 |
| 迦南 | 摩特 (Mot) | 死亡化身，犧牲創造的隱喻 |

伽約馬特的神話結構與印歐語系其他分支的「原人犧牲創世」母題（Proto-Indo-European *Manu-）高度同源。特別值得注意的是，波斯版本中伽約馬特的死亡不是被眾神犧牲（如普魯沙被眾神獻祭），而是被邪惡力量殺害——這賦予了原初犧牲一種全新的倫理維度：犧牲本身不是神聖的儀式，而是正邪對抗的結果。

## 相關神祇

- 原始牛（Gavaevodata）：與伽約馬特同時被造的原初生物，犧牲後化作植物與動物
- 阿胡拉·馬茲達：創造伽約馬特的至高神
- 安格拉·曼紐：殺死伽約馬特的邪惡之靈
- 瑪什亞與瑪什亞娜：從伽約馬特精子中誕生的第一對人類
- 斯彭塔·曼紐（Spenta Mainyu）：阿胡拉·馬茲達的創造性聖靈，保護伽約馬特的遺產
