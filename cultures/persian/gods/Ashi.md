# 阿希 (Ashi / Aši Vaŋuhī)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 命運、福報、財富、神聖回報
- **象徵：** 寶藏、光輝、豐收、公正報償
- **相關：** 密特拉 (Mithra)、拉什努 (Rashnu)、索羅什 (Sraosha)

## 概述

阿希（阿維斯陀語 Aši，意為「回報」「命運份額」），全名阿希·萬古希（Aši Vaŋuhī，意為「善報」），是瑣羅亞斯德教中最為重要的女性亞扎塔（Yazata，值得崇拜者）之一。她是掌管命運報償、財富福氣與神聖公正的化身。在《阿維斯塔》的〈亞什特〉（Yasht）頌歌中，阿希佔據了第17篇（Yasht 17）的全部篇幅——這本身就證明了她在大眾信仰中的崇高地位不同於一般次要神祇。

阿希不僅是物質財富的賜予者，更掌管著「依照個人之言行而應得的命運份額」——她不會賜福給說謊者、不義者與背約者，只有遵循阿沙（Asha，真理/秩序）之人才能獲得她的眷顧。這一特質使阿希從一名單純的「財富女神」昇華為道德因果律的神格化。

## 神話事蹟

### 出身與家族

〈亞什特〉17.16 記載了阿希的系譜：她是至高神阿胡拉·馬茲達之女，同時也是密特拉與拉什努的姊妹。這一神聖家族象徵著正義統治的完整循環——阿胡拉·馬茲達制定秩序（Asha），密特拉監督契約與盟誓的履行，拉什努衡量靈魂在欽瓦特橋（Chinvat Bridge）前的善惡比重，而阿希則根據裁決結果賜予每個人應得的命運份額。

她也與索羅什（Sraosha，聽從/順服之神）和帕蘭迪（Parandī，財富的先驅者）密切合作。帕蘭迪在阿希降臨之前先行，象徵著機會與前兆，阿希則隨後而至，將實質的福報帶給應得之人。

### 外貌與象徵

〈亞什特〉17 對阿希的描寫極具詩意：她駕著一輛由四匹強壯白馬拖曳的金色戰車，車輪鑲滿寶石，車身散發永恆光芒。她身穿金色飾服，面容姣好，雙臂雪白豐腴，身材高挑——這些描述不僅是審美意象，更暗示了她的本質：財富（金）、光明（光）、公正（白馬對應阿沙的純淨）。

阿希的權力以「三寶」象徵：**寶藏**（物質福報）、**光輝**（神聖恩寵）、**子孫**（血脈延續）。她也使牛群與羊群繁盛、使農田豐收、使家庭和睦——她是瑣羅亞斯德教徒在日常生活中最常祈求的亞扎塔之一。

### 與不義者的關係

阿希性格中最具特色的面向是她對道德純潔的要求。〈亞什特〉17.54–58 警告：「無論何人祭獻給她，若心中懷有謊言（Druj），阿希將避開他的祭品，如羊群避開狼群。」這段經文明確將物質繁榮與道德正義掛鉤——不受阿希眷顧的富人是矛盾的、暫存的，其財富終將消散。

## 跨文化對應

| 文化 | 對應神祇/概念 | 共通點 |
|------|--------------|--------|
| 印度吠陀 | 頗哩提毗 / 羅什彌 (Ṛtam / Lakṣmī) | 掌管財富與命運，拉克什米為吉祥天女，與阿希皆為道德財富之神 |
| 希臘 | 堤喀 (Tyche) 與 涅墨西斯 (Nemesis) | 堤喀掌管機遇財富，涅墨西斯執行報應——阿希一人兼具二者 |
| 北歐 | 芙蕾雅 (Freyja) / 凡蒂 (Vant) | 豐收與黃金之神系，半由命運分配 |
| 羅馬 | 福圖娜 (Fortuna) | 命運輪盤的掌舵者 |
| 中國 | 福神 / 祿神 | 賜福與官祿之神，但缺乏道德因果的直接綁定 |
| 埃及 | 瑪亞特 (Maat) | 真理與公正之神，決定死者命運的審判標準——與阿希同為「報償道德化」的化身 |

值得特別指出的是，阿希在功能上與印度的 **Ṛta**（宇宙秩序）和 **Lakṣmī**（吉祥財富）有深層的印歐共同根源：都將「應得之報」與「宇宙真理」綁定。但在波斯，這個概念被更強烈地個人化與倫理化，成為每一個教徒日常行為的激勵機制。

## 相關神祇

- 阿胡拉·馬茲達（Ahura Mazda）：父親，至高智慧主
- 密特拉（Mithra）：兄長，契約與光明之神
- 拉什努（Rashnu）：兄長，公正之神，靈魂的秤量者
- 索羅什（Sraosha）：戰友，聽從之神，冥界引路人
- 帕蘭迪（Parandī）：財富的前驅者，先於阿希到來的吉兆
- 沃胡·瑪納（Vohu Manah）：善念，決定命運的道德層面
