# 韋雷特拉格納 (Verethragna / Wahrām / Bahram)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 勝利、戰爭、力量、野豬、屠龍、賦予英雄戰力
- **象徵：** 野豬、駱駝、鷹、風、公馬、羚羊（十種化身）
- **對應：** 印度因陀羅 (Indra) 的戰鬥面向，米底—波斯戰士階層的守護神

## 概述

韋雷特拉格納（阿維斯塔語 Verethragna，中古波斯語 Wahrām，波斯語 Bahram）是波斯萬神殿中最具戰鬥力的神祇，其名直譯為「擊破障礙者」或「殺戮抵抗者」。他在瑣羅亞斯德教的神聖階層中屬於 Yazata（可敬拜者）的較高層級，與密特拉（Mithra）共同構成戰士階層的雙重守護——密特拉掌管契約與戰場忠誠，韋雷特拉格納則賜予實際的戰鬥力量與勝利。在薩珊王朝時期，他的地位進一步上升——巴赫拉姆（Bahram）成為了多任國王的名字，也是火星的波斯語名稱。

## 神話事蹟

韋雷特拉格納的神話主要記載於《阿維斯塔》的〈巴赫拉姆·耶什特〉（Bahram Yasht，Yasht 14）。這篇頌歌詳細描述了韋雷特拉格納的十種化身，每一種對應一種動物或自然力量。當他降臨於瑣羅亞斯德面前時，依次化為：（1）疾風——阿胡拉·馬茲達創造的強風；（2）金角公牛；（3）白耳金甲駿馬；（4）發情期駱駝；（5）尖牙野豬——最著名的形象，象徵不可阻擋的攻擊；（6）十八歲青年；（7）展翅金爪巨鷹——翼展遮蔽群山；（8）彎角公羊；（9）尖角戰羚；（10）飾金戰士。這十種化身不僅展現了他的多面向力量，也為波斯藝術提供了豐富的圖像學基礎——薩珊銀盤上常見的野豬頭盔戰士即是韋雷特拉格納的象徵。

韋雷特拉格納最關鍵的神話功能是「賦予英雄戰力」。在伊朗史詩傳統中，傳奇英雄如魯斯塔姆（Rostam）、伊斯凡迪亞爾（Esfandiyār）、甚至凱·霍斯魯（Kay Khosrow）在關鍵戰役前都會向韋雷特拉格納祈禱。他賜予英雄的力量並非一般性的增強，而是特定的「勝利靈光」（Xvarenah），使英雄在戰場上獲得超自然的敏捷、耐力與威懾力。他也負責擊退惡魔勢力——尤其與旱魔 Apaosha 及邪魔 Druj 作戰，維護世界的正義秩序。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 印度吠陀 | 因陀羅 (Indra) Vritrahan | 語源一致（Vritrahan「殺弗栗多者」= Verethragna），同為戰神 |
| 希臘 | 阿瑞斯 (Ares)／赫拉克勒斯 (Heracles) | 戰爭與力量；英雄式戰功 |
| 北歐 | 索爾 (Thor) | 雷神屠巨蛇，守護人類與眾神 |
| 美索不達米亞 | 尼努爾塔 (Ninurta) | 戰神屠十一怪獸，擁有野豬象徵 |
| 羅馬 | 馬爾斯 (Mars) | 戰神，農業與戰爭雙重職能 |
| 中國 | 蚩尤／關羽 | 戰神崇拜，死後被神化的戰力化身 |

韋雷特拉格納最特別之處在於他的動物化身系統——十種化身體現了他對自然界不同力量（風、野豬、鷹、駱駝等）的控制力，這也是波斯薩滿傳統與軍事貴族文化交織的結果。

## 相關神祇

- **Ahura Mazda**：至高之主，韋雷特拉格納的力量最終來源
- **Mithra**：契約與光明之神，共享戰士信仰
- **Sraosha**：聽從之神，韋雷特拉格納在靈魂審判中的同伴
- **Tishtrya**：天狼星之神，同為「對抗旱魔」的戰鬥者
- **Anahita**：水神，戰士出征前的祈禱對象
