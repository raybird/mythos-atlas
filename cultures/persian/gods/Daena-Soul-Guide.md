# 達埃納 (Daēnā) — 靈魂的引導者與內在良知

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 良知、靈魂的審判引導、死後在欽瓦特橋迎接靈魂的少女
- **象徵：** 美麗聖潔的少女（對義人），或醜陋老嫗（對惡人）

## 概述

達埃納（Daēnā，中古波斯語 Den / Din，意為「良知」或「宗教」）是瑣羅亞斯德教中極具特色的神聖存在——她同時是一個人內在良知的化身、死後靈魂的指引者、以及在欽瓦特橋（Chinvat Bridge）前迎接靈魂的少女形象。Daēnā 的概念融合了心理學中的「良知」（道德判斷）、宗教中的「異象」（死後的審判）以及神話中的「靈魂伴侶」。她是個人一生中思想、言語與行為的總和——在死後化為具體的形象迎接靈魂。

Daēnā 與印度吠陀中的 Dhénā（靈感、思想）同源——兩者共享印歐語系詞根 *dhei-（看、思想、宗教直覺）。在阿維斯塔語中，Daēnā 不僅指個人的良知，也指整個瑣羅亞斯德教的宗教體系——由此衍生出中古波斯語 dēn（宗教）與現代波斯語 din（信仰）。

## 神話事蹟

### 死後的相遇（欽瓦特橋的迎者）

瑣羅亞斯德教對死後旅程的敘述中，靈魂在死後第四天清晨抵達欽瓦特橋（「審判之橋」）。在此時，一陣芬芳的風從南方吹來——靈魂看見一位美麗的少女從風中走出。她以世間最美麗的少女形象出現——十五歲的年紀、容光煥發、身穿白衣、手持花環。靈魂問：「你是誰？為何如此美麗？」少女回答：「我是你的行動——Daēnā。你是誰？你愛我，你就見到了我。你在世上的善念、善言、善行——那就是我。」

對於生前作惡的靈魂，Daēnā 以截然不同的面貌出現：一位醜陋的老嫗，身體腐爛，散發惡臭。靈魂恐懼地問她是誰——她回答：「我是你的行動。你在世上的惡念、惡言、惡行——那就是我。」此敘述與佛教《中陰聞教得度》（Bardo Thödol）中臨終時所見的景象驚人地相似，二者的道德審判邏輯具有深層結構的同源關係。

### 人間的良知

Daēnā 不僅是死後的審判者——她在人間即已存在於每個人的心中。她引導人選擇善的道路，在思想中低語正確的決定。使 Daēnā「增長」或「減弱」是一生的任務——每個善行使她的形象更加美麗，每個惡行使她的形象趨向腐朽。在《阿維斯塔》〈哈多赫特·納斯克〉Hadokht Nask 第二章中，這一名義上的「天使」被進一步擬人化為帶領靈魂進入天堂（Garothman，光明之域）的引導者。

### Daēnā 與 Amesha Spenta

Daēnā 與阿胡拉·馬茲達的六位大天使（Amesha Spenta）關係密切。她經常與位於左側的 Armaiti（Spenta Armaiti，神聖虔敬）並列。在部分晚期文獻中，Daēnā 被視為在阿沙·瓦希什塔（Asha Vahishta，至真）與沃胡·瑪納（Vohu Manah，善念）之間的橋樑——她將善念轉化為實際行動，再將行動回歸真理。

## 跨文化對應

| 波斯 | 跨文化對應 | 文化 |
|------|-----------|------|
| Daēnā（死後面見的少女） | Mādhyamaka 中的業力影像 | 佛教 |
| Daēnā = 個人行動的化身 | 《埃及死者之書》中「心臟的重量」審判——心臟即一生行為的總和 | 埃及 |
| Daēnā（良知引導者） | 守護天使／良知神聖聲音 | 基督教 |
| Daēnā 在欽瓦特橋迎接 | 赫爾墨斯引導靈魂至冥界 | 希臘 |
| 善人見到美少女／惡人見到老嫗 | 中陰階段所見佛尊與恐怖相 | 藏傳佛教 |
| Daēnā 為宗教本身的化身 | 索菲亞（Sophia，智慧） | 諾斯替教 |
| Daēnā = 個人思想言語行為的總和 | 業力（Karma） | 印度 |

## 相關神祇

- **Rashnu（公正之神）**：在欽瓦特橋共同審判靈魂
- **Mithra（密特拉）**：監視靈魂的審判過程
- **Sraosha（聽從之神）**：守護靈魂通過欽瓦特橋
- **Amesha Spenta（六位大天使）**：Daēnā 與之共同引導人類接近善
