# 克雷薩斯帕 (Keresaspa)

- **文化：** 波斯神話 (Persian / Zoroastrian Mythology)
- **職掌：** 英雄、戰士、龍蛇屠殺者、不死者
- **象徵：** 狼牙棒、公牛之力、金色盔甲
- **族系：** 薩馬家族（Sāma clan），斯里塔（Thrita）之子

## 概述

克雷薩斯帕（Keresaspa，阿維斯塔語 Kərəsāspa，中古波斯語 Karshāsp / 薩珊波斯語 Garshāsp）是波斯神話中最強大的英雄戰士之一，地位僅次於伊瑪（Yima / Jamshid）與扎哈克（Zahhak）傳說中的英雄。他是薩馬家族的核心人物，以其屠龍事蹟與超自然的戰鬥能力聞名。在《阿維斯塔》的〈耶什特〉（Yasht）與〈萬迪達德〉以及《王書》（Shahnameh）中均有重要地位。《王書》中他的名字出現為加爾沙斯普（Garshāsp），偶爾與薩姆（Sām）混淆或融合。

克雷薩斯帕在瑣羅亞斯德教神學中佔據一個獨特位置——他雖然是史前英雄，但由於他的驕傲與疏忽，他未能幫助瑣羅亞斯德傳播信仰。然而阿胡拉·馬茲達預言他將在終末時代復活，協助救世主索什揚特（Saoshyant）擊敗最後的邪惡。相對於伊瑪的王者類型，克雷薩斯帕代表戰士階層（Rathaëštar）的完美典型——力量、勇氣、榮譽與悲劇缺陷的混合。

## 神話事蹟

### 屠蛇英雄

克雷薩斯帕最著名的事蹟是殺死巨蛇斯魯瓦拉（Srūvara，或作 Sruvar）。《阿維斯塔》的〈耶什特〉第 19 章中描述這條巨蛇吞食馬匹與人類，身軀龐大足以覆蓋整個山谷。克雷薩斯帕以沸水灌入蛇穴或利用熔化的黃銅（青銅）注入蛇口將其殺死。這一母題與印度神話中因陀羅殺死弗栗多（Vritra）、北歐神話中索爾殺死耶夢加得（Jörmungandr）以及希臘神話中赫拉克勒斯殺死九頭蛇（Hydra）極為相似。

另一個版本記載他殺死了巨龍阿日達哈卡（Azhdahāka，蛇王扎哈克的惡魔本體）。在後期《王書》傳統中，加爾沙斯普的故事與蛇王扎哈克傳說交織——他是最終制服扎哈克的英雄之一，將蛇王囚禁於德馬萬德山（Mount Damavand）中。

### 漫長的沉睡

克雷薩斯帕在冒險過程中因不慎食用了被安格拉·曼紐污染的烤肉（或觸碰了禁忌之物），陷入深沉的永恆睡眠。他的靈魂進入了類似冥界的中間狀態（Hamestagān）。然而阿胡拉·馬茲達的預言顯示，他並未真正死亡——他的力量在等待最後的喚醒。民間傳說中描述他沉睡於迷霧山谷中，千年不醒，周圍環繞著守護他的火焰。

### 終末的覺醒

在瑣羅亞斯德教的終末論中，克雷薩斯帕將在宇宙年的最後一個千紀被救世主索什揚特喚醒。他將舉起金色狼牙棒，屠殺最後的惡魔大軍，見證火河淨化世界。他與另一位復活的英雄圖沙·阿特拉（Tusa Athra）並肩作戰，共同消滅安格拉·曼紐在人間最後的追隨者。克雷薩斯帕的復活象徵戰士美德在終末勝利的不可或缺性——正義需要力量來保護。

## 跨文化對應

| 文化 | 對應人物 | 共通點 |
|------|----------|--------|
| 希臘 | 赫拉克勒斯 (Heracles) | 屠蛇、十二苦役、半神英雄、死後成神 |
| 北歐 | 索爾 (Thor) | 雷神屠蛇、世界蛇耶夢加得、終末之戰 |
| 印度 | 因陀羅 (Indra) | 殺死弗栗多（蛇龍）、解放水源、戰士之神 |
| 美索不達米亞 | 吉爾伽美什 (Gilgamesh) | 英雄冒險、殺死天牛、求永生不得 |
| 日耳曼 | 齊格菲 (Siegfried) | 屠龍英雄、不死之身、英雄的悲劇 |
| 中國 | 后羿 | 屠蛇（斬修蛇）、射日、英雄悲劇結局 |

克雷薩斯帕的英雄循環深刻地影響了中亞與伊朗語系民族的史詩傳統。《王書》中的英雄薩姆（Sām）與魯斯塔姆（Rostam）明顯繼承了他的母題——尤其是戰士與怪獸的搏鬥、英雄沉睡與喚醒的敘事公式。通過薩珊時期與伊斯蘭初期的口傳傳統，克雷薩斯帕的故事甚至可能影響了歐洲中世紀的屠龍英雄傳說。
