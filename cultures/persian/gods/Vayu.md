# 瓦尤 (Vayu / Vāyu)

- **文化：** 波斯神話／瑣羅亞斯德教 (Persian / Zoroastrian Mythology)
- **職掌：** 風、大氣、空間、生命氣息、靈魂渡引
- **其他名稱：** Vāta（風的具體現象）、Rambha Vāyu（至福之風）

## 概述

瓦尤（Vāyu）是波斯神話中最古老且最矛盾的神祇之一。在《阿維斯塔》中，瓦尤被稱為「在天地之間運行者」，掌管大氣層與中間空間（frashefta）。不同於大多數神祇明確歸屬於善或惡陣營，瓦尤具有雙重本性——他既是斯彭塔·曼紐（Spenta Mainyu，聖靈）的盟友，也與安格拉·曼紐（Angra Mainyu，惡靈）合作。這種道德模糊性使瓦尤成為瑣羅亞斯德教二元宇宙觀中最獨特的存在。

在亞什特（Yasht）第十五篇中，瓦尤被描述為一位強勁的戰士神，以金色鎧甲與閃耀武器武裝，駕馭風之戰車橫掃天際。他的呼吸即為風——溫和時賜予生命與豐收，暴怒時以颶風毀滅一切。瓦尤同時是靈魂的渡引者：人在死亡後的前三天，靈魂依附於瓦尤的風域，隨後方抵達欽瓦特橋（Chinvat Bridge）接受審判。

## 神話事蹟

瓦尤最關鍵的角色出現在世界末日論（eschatology）中。根據《班達希申》（Bundahishn），在最終更新（Frashokereti）之時，瓦尤將吹響神聖之風，喚醒所有亡者的骨骸，使其重生接受最終審判。這個「復活之風」的意象對後來的亞伯拉罕宗教產生深遠影響。

在更早期的雅利安共祖神話中，瓦尤被視為宇宙的第一口氣息——創世之初，神聖之風吹過無限汪洋，水面泛起漣漪，從而誕生了物質世界。這個「風生水起」的創世母題同樣出現在埃及（阿圖姆以風創造舒）、希伯來（Elohim之靈運行於水面上）以及希臘（厄羅斯與風之卵）神話中。

## 跨文化對應

| 瓦尤的角色 | 其他文化對應神祇 | 文明 |
|---|---|---|
| 風與大氣之神 | 舒（Shu） | 埃及神話 |
| 生命氣息 | 普紐瑪（Pneuma）／阿涅摩伊 | 希臘神話 |
| 雙重本性（善惡之間） | 洛基（Loki） | 北歐神話 |
| 風神 | 風伯（Feng Bo）／飛廉 | 中國神話 |
| 戰士風神 | 伊亞波（Ehecatl） | 阿茲特克神話 |
| 風與靈魂 | 風神（Fūjin） | 日本神話 |
| 復活之風 | 聖靈／Ruach Elohim | 希伯來聖經 |
| 宇宙風／空間 | 伐由（Vāyu） | 印度神話（吠陀） |

瓦尤與吠陀神話的伐由（Vāyu）共享同一印歐根源，但其演化路徑截然不同：在印度傳統中，伐由始終是善神，與因陀羅密切合作；在波斯傳統中，瓦尤則發展出更為複雜的道德雙重性，反映了瑣羅亞斯德教對宇宙善惡拉鋸的深刻思考。

## 相關神祇

- **斯彭塔·曼紐（Spenta Mainyu）**——聖靈，瓦尤的善性面向的盟友
- **安格拉·曼紐（Angra Mainyu）**——惡靈，瓦尤的破壞性面向的夥伴
- **密特拉（Mithra）**——契約與光之神，瓦尤常與之共同守護欽瓦特橋
- **蒂爾（Tishtrya）**——雨星神，與瓦尤共同掌管天氣變化
- **拉什努（Rashnu）**——公正之神，與瓦尤協同進行靈魂審判
