# Ilmatar（伊爾瑪塔）

- **文化：** 芬蘭-烏戈爾神話 (Finno-Ugric Mythology)
- **職掌：** 空氣、水、創世、天空之母

## 概述

Ilmatar（芬蘭語「ilma」意為空氣＋「-tar」女性後綴）是芬蘭神話中的空氣女神與水之母，為《卡勒瓦拉》創世序章的核心人物。她也被稱為 Luonnotar（自然之母），是萬物之母與創世原力的化身。Ilmatar 在無邊原海上漂浮了七百年，孤獨地在汪洋中孕育生命——一隻潛水鳥（sotka）飛來在她膝上築巢孵蛋，蛋跌落破裂後碎片化為世界。Ilmatar 本身就是創世的容器與母體——她的身體化為大地，她的呼吸為風，她的淚水為水。她象徵宇宙尚未分化前的原初狀態——無邊的海洋與天空，充滿潛能等待分化與創造。

## 神話事蹟

Ilmatar 的神話主要出現在《卡勒瓦拉》開篇。在時間開始之前，Ilmatar 降臨無邊原海——她獨自漂浮數百年，孤獨至極。一隻潛水鳥飛來，以為 Ilmatar 的膝蓋是陸地，在其上築巢並孵育六枚金蛋與一枚鐵蛋。Ilmatar 膝蓋發熱使蛋滾落破裂——蛋的碎片化為宇宙：下殼為大地、上殼為天空、蛋黃為太陽、蛋白為月亮、蛋斑為星辰。創造完成後，Ilmatar 繼續塑造海岸、海灣與島嶼。她後來生下 Väinämöinen——他在她體內孕育三十年後出生，一出生即成年。Ilmatar 在兒子出生後逐漸退居幕後，成為宇宙運行的背景力量。部分芬蘭咒語中召喚 Ilmatar 治療疾病或保護產婦。

| 埃及神話 | Nut（努特） | 天空女神／從原水升起 |
| 希臘神話 | Gaia（蓋亞） | 大地之母、萬物起源 |
| 印度神話 | Aditi（阿底提） | 無限之母、眾神之母 |

| 波利尼西亞神話 | Papatūānuku（地母） | 大地母親、創世源頭 |
