# Kuu（庫）

- **文化：** 芬蘭-烏戈爾神話 (Finno-Ugric Mythology)
- **職掌：** 月亮、夜晚、狩獵、潮汐、人類命運

## 概述

Kuu（芬蘭語「kuu」意即月亮）是芬蘭-烏戈爾神話中的月神，為太陽女神 Päivätär 的兄弟。他的名字在卡累利阿與愛沙尼亞方言中分別為「Kuu」與「Kuun」。不同於北歐 Máni（僅為驅動月車的存在）或希臘 Selene（被動的美麗形象），芬蘭的 Kuu 是一個活躍的神話角色——他駕駛銀色月車橫跨夜空，同時主宰人類的命運、狩獵的成敗與水手的平安。Kuu 的形象是位披著銀色斗篷的年輕男子——他的斗篷內襯以星星裝飾，象徵夜空披風。Kuu 的職能涵蓋：（1）夜間照明；（2）決定孩童的出生時刻與性格（滿月出生的孩子被認為有靈視力）；（3）監管狩獵（月光下的獵物更容易被追蹤）；（4）控制潮汐與女性的生理週期。在芬蘭語中，月相變化（kuun vaiheet）直接與人類情緒詞彙（mieliala 心情）形成語言學關聯——月亮與心靈的連結深植於文化之中。

## 神話事蹟

Kuu 最著名的神話為他對 Päivätär（太陽女神）的永恆追逐。根據《卡勒瓦拉》的口傳旁支，Kuu 與 Päivätär 曾訂下婚約——但 Kuu 在婚禮前夕愛上了黎明女神（Aamurusko），因此永遠失去 Päivätär。從此 Kuu 駕著月車日夜追逐 Päivätär——但永遠無法追上，因為天亮時 Kuu 便消失在地平線下。當 Kuu 在滿月時最接近太陽——那是他距離「和解」最近的時刻。芬蘭民間傳說中 Kuu 與熊（Karhu）有關——熊在芬蘭語中稱為「otso」或「mesikämmen」，但獵人使用代名詞「Metsänomena」（森林之蘋果）以避免直呼其名，因為熊是 Kuu 的神聖動物。Kuu 同時掌管命運——孩子在滿月下出生將成為薩滿，在新月下出生將成為戰士。水手在波羅的海航行時向 Kuu 祈禱——月光在水面上的倒影被稱為「Kuu 的橋樑」。Kuu 有時降臨人間——他以銀色斗篷遮面，化為英俊青年引誘人類女子，產下的半神後代具有預言能力。部分芬蘭咒語也召喚 Kuu 治療眼疾——因為月亮的銀光被認為具有恢復視力的功效。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Selene（塞勒涅） | 月神、駕車穿越夜空 |
| 北歐神話 | Máni（馬尼） | 月車、被狼追逐 |
| 埃及神話 | Thoth / Khonsu（托特／孔蘇） | 月神、時間計算、命運 |
| 美索不達米亞 | Nanna／Sin（南納／辛） | 月神、智慧、牧羊 |
| 日本神話 | 月讀命（Tsukuyomi） | 月神、夜晚統治 |
| 中國神話 | 太陰星君／嫦娥 | 月宮、不死藥 |

## 相關神祇

- **Päivätär**（太陽女神）——Kuu 追逐的對象
- **Ukko**（雷神）——天空秩序的共同維護者
- **Aamurusko**（黎明女神）——Kuu 的婚外感情對象
- **Ilmatar**（空氣之母）——賦予 Kuu 天空中的位置
