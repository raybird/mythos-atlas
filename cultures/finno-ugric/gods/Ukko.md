# Ukko（烏科）

- **文化：** 芬蘭-烏戈爾神話 (Finno-Ugric Mythology)
- **職掌：** 天空、雷電、天氣、豐收、眾神之王

## 概述

Ukko（芬蘭語，意為「老者/祖父」）是芬蘭-烏戈爾神話中的至高神，掌管天空、雷電、氣候與豐收。他的名字可能與芬蘭語「ukkonen」（雷暴）同源。Ukko 在卡累利阿與芬蘭地區被尊為眾神之父——他住在天界的第九重天，手持雷錘或雷斧，駕馭雲中戰車。雷聲是 Ukko 駕車駛過天穹的聲音，閃電來自他投擲的雷鎚。他的妻子是 Akka（地母），二人共同為大地帶來生命與豐饒。Ukko 的象徵物包括雷鎚、閃電箭與橡樹。在芬蘭民間信仰中，農民在春耕時節向 Ukko 祈禱降雨——他的名字出現在許多古老的咒語與祈禱文中，如「Ukko 的雷聲驅散邪靈」。基督教傳入後 Ukko 的角色部分被轉移至聖烏爾霍（Saint Ulrich），但民間仍保留 Ukko 的節日與祭祀傳統。

## 神話事蹟

Ukko 最著名的功能是掌控天氣——當他揮動雷鎚，天空降下雨水滋養農作物。在《卡勒瓦拉》中，Ukko 派遣雷雲與閃電協助 Väinämöinen 完成艱鉅任務。芬蘭神話中 Ukko 以閃電箭（Ukko 的矛）擊殺邪靈與水怪——這些箭矢落地後化為「雷石」（ukonvaaja），被當作聖物收藏。芬蘭民間咒語中，Ukko 被召喚來治療疾病、保護牲畜與驅除邪祟。他曾在世界之初協助孵育世界蛋——部分神話版本中 Ukko 以膝蓋協助潛水鳥孵蛋，蛋破裂後形成宇宙。Ukko 與 Akka 的婚姻象徵天地結合——他們的後代包含所有自然神靈。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 北歐神話 | Odin / Thor | 眾神之父 / 雷神 |
| 希臘神話 | Zeus（宙斯） | 雷電、天父、降雨 |
| 印度神話 | Indra（因陀羅） | 雷神、降雨、戰蛇 |
| 斯拉夫神話 | Perun（佩倫） | 雷電、天父、戰斧 |
| 中國神話 | 雷公 / 昊天上帝 | 天氣掌控 / 至高神 |

## 相關神祇

- **Akka**（地母）——Ukko 的妻子，大地豐饒的象徵
- **Ilmatar**（空氣之母）——世界蛋的孵化者
- **Väinämöinen**（吟唱智者）——Ukko 的凡人／半神後代
- **Seppo Ilmarinen**（鐵匠）——鍛造天空穹頂者
