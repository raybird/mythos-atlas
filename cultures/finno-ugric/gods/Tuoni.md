# Tuoni（圖奧尼）與 Tuonetar（圖奧內塔爾）

- **文化：** 芬蘭-烏戈爾神話 (Finno-Ugric Mythology)
- **職掌：** 冥界之王／冥界之后、死亡、疾病

## 概述

Tuoni 是芬蘭神話中的冥界之王，掌管 Tuonela（冥界／死人之國）——位於最北端的一處黑暗陰冷之地。他的名字與芬蘭語「tuoni」（死亡）同源，字面意義即為「死亡本身」。他的配偶是 Tuonetar（冥界之后），二人共同統治死者國度。Tuonela 被描繪為被黑水分流的島嶼——亡者必須渡過黑水河（Tuonelan joki），河上盤旋著天鵝（Tuonelan joutsen）。Tuonela 的意象充滿芬蘭北部自然景觀的投影——永恆的暮色、冰冷的河流、陰鬱的森林。Tuoni 的形象不是邪惡的——他不懲罰亡者，只是冷淡地接引所有進入冥界的靈魂。Tuoni 被描繪為戴著鐵手套的老者，他的手指具有將人化為灰燼的力量。Tuonetar 則釀造「遺忘之啤酒」(muistojen olut)——亡者飲用後忘記生前一切。

## 神話事蹟

Tuoni 與 Tuonetar 在《卡勒瓦拉》中最重要的登場是 Väinämöinen 的冥界之旅。Väinämöinen 為了獲取失傳的咒語，決定親赴 Tuonela。他建造鐵船渡過黑水河——Tuonetar 以遺忘之啤酒款待他，但 Väinämöinen 拒絕飲用，因為他知道飲後將無法返回人間。Tuoni 試圖以鐵網與鐵矛困住 Väinämöinen——但 Väinämöinen 化身為蛇逃脫。離去前 Tuoni 詛咒他永遠無法離開——Väinämöinen 以吟唱的力量在冰上滑行逃回人間。另一段著名的冥界之旅是 Lemminkäinen 進入 Tuonela 射殺天鵝——Tuoni 以鐵網擒住他，將他的身體砍成碎片投入黑水河中。他的母親前往 Tuonela 以鐵耙打撈碎片，以魔法將他復活。Tuoni 對闖入冥界的生者既無惡意也無善意——他冷漠地執行自然法則。

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Hades（哈迪斯） | 冥界之王，冷淡而公正 |
| 北歐神話 | Hel（赫爾） | 冥界女神，死者國度 |
| 埃及神話 | Osiris（歐西里斯） | 冥界審判之神 |
| 中國神話 | 酆都大帝 | 冥界主宰 |
| 日本神話 | 伊邪那美 | 黃泉國之主 |
| 斯拉夫神話 | Veles（維列斯） | 冥界之神，與雷神對立 |

## 相關神祇

- **Tuonetar**（冥界之后）——Tuoni 的妻子，遺忘啤酒的釀造者
- **Väinämöinen**——曾進入 Tuonela 尋求咒語
- **Lemminkäinen**——在 Tuonela 被殺後由母親復活
- **Akka**（地母）——與 Tuoni 相對的生命與死亡循環
