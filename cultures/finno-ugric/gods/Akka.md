# Akka（阿卡）

- **文化：** 芬蘭-烏戈爾神話 (Finno-Ugric Mythology)
- **職掌：** 大地、豐收、婚姻、婦女與生育

## 概述

Akka（芬蘭語意為「老婦／祖母」）是芬蘭-烏戈爾神話中的大地女神與豐收之神，為至高神 Ukko 的妻子。她是芬蘭民間信仰中最受愛戴的女神之一——農民在春耕時向她祈禱豐收，婦女在生產時祈求她的庇護。Akka 的名字與芬蘭語「akka」（妻子／老婦）同源，在卡累利阿地區她被稱為「Maan Akka」（大地阿卡）或「Raudun Akka」。她的神聖動物包括母牛與綿羊——春天第一個犢牛的出生被認為是 Akka 的祝福。Akka 與 Ukko 的婚姻象徵天地的神聖結合——Ukko 從天降雨，Akka 從地孕育萬物。她有時也被視為命運女神，掌管人類壽命的長短。在基督教傳入後 Akka 的角色被聖母瑪利亞部分取代，但芬蘭民間仍保留「Akka 的祈禱」這一傳統咒語。

## 神話事蹟

Akka 的主要神話功能是與 Ukko 共同維持宇宙秩序與人間豐饒。在春耕節慶中，農民舉行「Akka 的婚禮」(Akkaan häät) 儀式——將最後一把種子獻給 Akka，並以奶油與麵包祭祀。若遇乾旱，婦女會到田間呼喊 Akka 的名字，相信她能說服 Ukko 降雨。Akka 也被視為嬰兒的守護神——新生兒出生後，產婆會以 Akka 之名誦唸祝福咒語，驅除邪靈。在《卡勒瓦拉》中 Akka 的登場次數有限，但她在芬蘭民間咒語中的重要性遠高於史詩中的篇幅——超過兩百首古老的咒語提到 Akka 的名字，涵蓋農業、婚姻與醫療領域。在芬蘭葬禮傳統中，Akka 被喚醒來護送亡者靈魂到 Tuonela（冥界），顯示她作為生死循環女神的另一面向。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Demeter（狄蜜特） | 豐收女神、大地母親 |
| 北歐神話 | Frigg（芙麗格） | 婚姻、家庭、眾神之母 |
| 中國神話 | 后土娘娘 | 大地之神、生育護佑 |
| 印度神話 | Prithvi（普利提維） | 地母、豐饒之神 |
| 埃及神話 | Geb（蓋布） | 大地之神（豐收相關） |
| 斯拉夫神話 | Mokosh（莫科什） | 土地、生育、紡織女神 |

## 相關神祇

- **Ukko**（雷神）——Akka 的丈夫，天地結合
- **Ilmatar**（空氣之母）——創世女神，與 Akka 共享大地母性
- **Väinämöinen**——接受 Akka 祝福的文化英雄
- **Tuoni / Tuonetar**——冥界之神，與 Akka 的生死循環功能相對
