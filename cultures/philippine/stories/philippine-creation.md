# 菲律賓創世神話：世界從何而來 (The Creation of the Philippines)

- **文化：** 菲律賓神話 (Philippine Mythology)

## 故事背景

菲律賓的創世神話反映了這個七千島嶼國家的地理特徵——世界從海洋中誕生、從竹子中生出人類、從鳥類的飛行中產生了島嶼。以下是菲律賓創世神話的核心敘事，綜合了他加祿與米沙鄢地區的多種版本。

## 情節

### 第一章：天空與海洋

在時間開始之前，只有無邊無際的天空與同樣無邊無際的海洋。沒有人類、動物、植物——只有水與天空的永恆對望。一隻鳥（有時被稱為「世界鳥」或「神聖之鳥」）在兩者之間無盡地飛行，找不到休息的地方。

這隻鳥對天空說：「為何我們之間沒有土地？為什麼我要永遠地飛？」

天空回答：「沒有人類的腳步，我無法低下。」

海洋回答：「沒有河床，我無法後退。」

鳥兒飛得更低，翅膀幾乎觸及水面。就在那一刻，天與海被鳥的飛行感動——天空降下了第一場雨，海洋回應以退潮，露出了第一塊岩石。

### 第二章：島嶼的誕生

從這塊岩石開始，更多的石子與泥土從海底升起——鳥將它們從深水中拾起，投放到陽光下。這些石頭變成了菲律賓的七千多個島嶼。但最初的島嶼是荒蕪的——沒有樹木、生命或人類。

鳥向天空呼喊：「我們有了大地，但誰住在那裡？」

### 第三章：竹子中的人類

至高神 Bathala 聽到了鳥的呼喊。他看到一個人從海中漂浮——那是一根巨大的竹子（Kawayan），被海浪推到了岸邊。一根在風中搖曳的竹枝碰到了岸邊——竹子裂開，從中走出了第一對人類：

Malakas（「強者」，男性）與 Maganda（「美麗者」，女性）。Bathala 將他們命名為人類的祖先。

Malakas 與 Maganda 赤腳踏上沙灘，Bathala 對他們說：「這片土地是你們的家。我將讓竹子為你們建造房屋、讓鳥類教你們歌唱、讓大海餵養你們。」

### 第四章：人類的擴散

Malakas 與 Maganda 的後代很快遍布了島嶼。但語言的分歧隨之而來——Bathala 對人類的驕傲不滿，於是讓人類說不同的語言。這就是為何菲律賓群島上存在數百種不同的語言。

在大洪水後，部分人類逃到山頂存活了下來。他們的血統成為今日菲律賓不同民族的祖先——他加祿人、米沙鄢人、伊洛卡諾人。

## 跨文化平行

| 文化 | 對應故事 | 比較 |
|------|---------|------|
| 馬來 | 人類從檳榔樹中誕生 | 植物作為人類容器——南島語系文化的共通起源敘事 |
| 日本 | 伊邪那岐與伊邪那美以矛攪拌海水造島 | 島嶼從海洋中創造的相同動機 |
| 波利尼西亞 | Māui 以魚鉤釣起島嶼 | 英雄從海洋中拉出陸地的神話 |
| 中國 | 盤古創世→身體化為世界的各部分 | 身體-宇宙對應的創世結構 |
| 印度 | 梵天以蓮花創造世界 | 創世從水中浮現的共通模式 |
| 聖經 | 諾亞洪水後人類重新擴散 | 洪水後以同一家族開始新文明的結構 |
