# 迪安·瑪薩蘭塔（Dian Masalanta）

- **文化：** 菲律賓神話（他加祿 Tagalog）
- **職掌：** 愛情、受孕、分娩、家庭守護
- **別名：** Diyan Masalanta、Dayang Masalanta、Maria Makiling（民間衍生）

## 概述

迪安·瑪薩蘭塔（Dian Masalanta），又稱 Diyan Masalanta，是他加祿神話中的愛情女神，掌管受孕、分娩並保護戀人。她是阿納戈萊（Anagolay，失落事物女神）與杜馬庫萊姆（Dumakulem，狩獵之神）的女兒，太陽神阿波拉基（Apolaki）的妹妹，也是巴塔拉（Bathala）神系中最年輕的神祇之一。她的名字在古他加祿語中意為「不可觸及的純潔」，象徵愛情的神聖與不可侵犯。

西班牙殖民者將其比擬為希臘的阿佛洛狄忒，但在前殖民神話中，她的形象更具母性特質，高度重視家庭倫理。她的象徵物是「人類心臟」與「胎兒」，代表愛情的真誠與生命傳承。據文獻記載，他加祿人向她獻祭以確保順利受孕與平安生產。

瑪薩蘭塔同時與民間信仰中的馬里亞·馬基林（Maria Makiling）有著深層關聯。馬基林山（Mount Makiling）的守護女神瑪基林被認為是瑪薩蘭塔在民間傳說中的延續，掌管山林、湖泊與豐饒，並保護前來尋求幫助的居民。西班牙人將「瑪基林」加上「馬里亞」之名，試圖將其天主教化為聖母瑪利亞的形象（Jocano 1969）。

她的死敵是瑪尼西拉特（Manisilat），即破壞家庭的惡靈。兩者的對立反映了神話中家庭凝聚力與離散力之間的永恆鬥爭，也體現了他加祿社會以大家族（extended family）為核心的社群價值觀。

## 跨文化對應

| 文化 | 對應神祇 | 對比 |
|------|---------|------|
| 希臘 | 阿佛洛狄忒（Aphrodite）↔ 狄安娜（Artemis） | 兼具愛神與生育守護，但阿佛洛狄忒偏浪漫愛，瑪薩蘭塔更重家庭；生育面接近 Artemis 的助產職能 |
| 羅馬 | 維納斯（Venus）↔ 盧西娜（Lucina） | 維納斯僅限愛情，瑪薩蘭塔的護產職能更接近朱諾·盧西娜 |
| 印度 | 帕爾瓦蒂（Parvati） | 皆為溫柔母神，但帕爾瓦蒂是雪山神女，瑪薩蘭塔不具毀滅性面向 |
| 北歐 | 弗蕾亞（Freyja） | 同為愛神具生育力，但弗蕾亞有戰爭面，瑪薩蘭塔純為守護 |
| 埃及 | 哈索爾（Hathor） | 哈索爾亦兼愛神、母神與守護，結構相似度最高 |
| 日本 | 木花之佐久夜毘売（Konohana-Sakuya-Hime） | 皆為婚姻與生產女神，富士山傳說與馬基林山傳說有平行結構 |

## 相關神祇

- 父：杜馬庫萊姆（Dumakulem）——狩獵之神
- 母：阿納戈萊（Anagolay）——失落事物女神
- 兄：阿波拉基（Apolaki）——太陽神與戰士守護神
- 敵：瑪尼西拉特（Manisilat）——家庭破壞者
- 關聯：伊卡帕蒂（Ikapati / Lakapati）——農業與豐饒女神，負責人類生育的第二階段
