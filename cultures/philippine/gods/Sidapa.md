# 西達帕 (Sidapa / Sidapa)

- **文化：** 菲律賓神話 (Philippine Mythology)
- **職掌：** 死亡之神、冥界之主、生命長度的裁定者

## 概述

Sidapa（米沙鄢地區的死亡之神）是菲律賓神話中掌管人類壽命與死亡的神祇。與大多數文明的死亡之神不同——Sidapa 不住在地下或陰暗的冥界，而是居住在一棵巨大的聖樹（Balanak 樹或特定種類的大樹）上，從樹頂觀察人類的誕生。當一個人出生時，Sidapa 在樹上放置一個標記；當他從樹上取下這個標記時，那個人就會死亡。

Sidapa 的神話體系與 Bathala 的創世權力形成了有趣的互補——Bathala 創造人類並賦予生命，Sidapa 決定人類何時死亡。

## 神話事蹟

Sidapa 最獨特的神話特徵是他不是邪惡的死亡之神——他不懲罰亡靈，不審判人類的行為。他的角色是「生命定時器」：他準確記錄每一個人類的壽命，從不延遲也從不提前。

在一個著名的神話中，Sidapa 與月亮女神 Mayari 之間的禁忌之戀：Sidapa 爱上 Mayari 的美麗，每次在滿月時試圖靠近她。但太陽神 Apolaki（Mayari 的兄弟）發現此事後，確保 Sidapa 只能生活在白天看不見月亮的月份。此神話連結了特定的月相周期與雨/旱季節的交替。

Sidapa 的聖樹被認為具有強大的神聖力量——在部分地區，人們向 Sidapa 獻祭以祈求延長壽命，或在樹下進行儀式以治癒重病患者。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Thanatos（死神，非邪惡的死亡人格化） | 非懲罰性的死亡使者，僅執行生命的終結 |
| 埃及 | 阿努比斯（Anubis，死亡引導者） | 引導而非審判死亡的過程 |
| 印度 | Yama（冥界之王，人類第一個死者） | 決定人類壽命的長度，但更具審判性 |
| 日本 | 伊邪那美（黃泉津大神） | 死亡的根源，但 Sidapa 不因憤怒而行動 |
| 北歐 | Hel（冥界女神） | 統治死者領域，同樣不涉及善惡審判 |
| 中國 | 閻羅王（地獄之主，審判者） | 決定生死簿——與 Sidapa 的樹上標記功能類似 |

## 相關神祇

- 伴侶/暗戀對象：Mayari（月神，禁忌之戀的對象）
- 敵對者：Apolaki（太陽神，阻止 Sidapa 靠近 Mayari）
- 上級：Bathala（至高神，賦予 Sidapa 管理死亡的權力）
- 動物象徵：貓頭鷹（Sidapa 的鳥類，死亡的信使）
- 他加祿對應：Tigmamanukin（他加祿神話中類似的生命之鳥角色，也棲息在樹上決定人類壽命）

## 出現在

- 米沙鄢群島 Visayan 口傳傳統
- 19 世紀西班牙民族誌學者記錄的米沙鄢神話
- 部分菲律賓傳統治療儀式（向 Sidapa 祈求患者壽命延長）
