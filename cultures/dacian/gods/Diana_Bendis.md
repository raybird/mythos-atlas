# Diana（戴安娜-本狄斯）

- **文化：** 達基亞/羅馬尼亞神話 (Dacian/Romanian Mythology)
- **職掌：** 月神、狩獵、森林、女性守護、冥界引渡

## 概述

戴安娜（Diana）在達基亞文化中是羅馬月神 Diana 與本土色雷斯-達基亞月神 Bendis 深度融合的結果。Bendis 原為色雷斯人與達基亞人共同崇拜的月神與狩獵女神——她的名字在色雷斯語中可能意為「連結者」（指連結人間與冥界的橋樑）。公元前五世紀，雅典城內建立了 Bendis 的官方祭所（Bendideion），並於每年 Thargelion 月（約五月）舉行盛大的 Bendideia 火炬夜遊節，足見此神在希臘化世界的影響力。羅馬征服達基亞後，達基亞人將原有的 Bendis 信仰與羅馬的 Diana 合併——新的合成女神稱為 Diana Bendis 或 Regina Daciarum（達基亞之后）。她的雙重名字——拉丁名 Diana 與達基亞名 Bendis——銘刻於巴爾幹半島數十處還願石碑上。

Diana-Bendis 的形象融合了兩大傳統：手中持弓與箭（Diana 的狩獵特徵），肩披新月斗篷（Bendis 的月神特徵），腰間繫著火炬（Bendis 的冥界引渡象徵——火炬照亮死者的冥界之路）。她腳邊常伴兩隻動物：紅色雌鹿（狩獵）與黑色母狼（冥界）。她的雙重性——生者世界的狩獵女神同時也是死者世界的引導者——使她成為達基亞神系中唯一貫穿三界（天、地、冥）的女神。

## 神話事蹟

**Bendis 的降臨雅典：** 色雷斯-達基亞的 Bendis 崇拜在公元前五世紀傳入雅典，成為雅典官方認可的第一個「外來神秘崇拜」。柏拉圖的《理想國》開篇即以 Bendis 節為背景——蘇格拉底前往 Piraeus 港口參加 Bendideia 火炬賽馬與夜間遊行。這是古希臘宗教史上外來神被雅典完全接納的罕見案例——說明了 Bendis 崇拜在古典世界的強大吸引力。雅典的 Bendideia 節慶包括午夜火炬傳遞、騎兵競賽與獻祭儀式——此「火炬之夜」可能影響了後來厄琉息斯密儀的夜間元素。

**戴安娜的達基亞神廟：** 羅馬時期達基亞行省（今羅馬尼亞）出土了大量獻給 Diana 的祭壇，其中以 Sarmizegetusa Ulpia Traiana（達基亞行省首府）的 Diana 神廟最為著名。神廟內發現的雕像同時刻有拉丁文「Dianae」與達基亞語「Bendidi」——這是宗教融合的物質證據。考古學家在神廟周圍發現了大量小火炬形陶器，推測為 Bendis 冥界引渡儀式的遺物——信徒以火炬引導逝者的靈魂。

## 跨文化對應

| 文化 | 對應女神 | 共通點 |
|------|----------|--------|
| 希臘神話 | Artemis（阿緹蜜絲） | 狩獵、月亮、野獸與純潔 |
| 羅馬神話 | Diana | 狩獵、月亮、森林保護 |
| 埃及神話 | Bastet（芭絲特） | 月光、女性守護、貓與狩獵 |
| 日本神話 | 月讀命（Tsukuyomi） | 月亮神祇、夜的主宰 |
| 蘇美神話 | Inanna（伊南娜） | 戰爭、愛慾、冥界女王（雙重性） |
| 凱爾特神話 | Morrigan | 三方女神、預言、死亡與戰爭 |
| 中國神話 | 嫦娥（Chang'e） | 月宮居住、永生象徵、月光女神 |

## 相關神祇

- **Derzelas**（健康/冥界神）—— Bendis 在冥界的合作者，共同引渡靈魂
- **Zalmoxis**（冥界之主）—— Diana-Bendis 將靈魂送至 Zalmoxis 的永恆盛宴
- **Kotys**（大地之母）—— Bendis 作為月神與 Kotys 的豐收節令交織
- **Diana 的獵犬軍團**——達基亞傳說中 Diana 率領一群月光獵犬巡視喀爾巴阡山森林
