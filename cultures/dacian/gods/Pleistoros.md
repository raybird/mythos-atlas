# Pleistoros（普萊斯托羅斯）

- **文化：** 達基亞/羅馬尼亞神話 (Dacian/Romanian Mythology)
- **職掌：** 戰爭、戰鬥狂怒、戰略智慧、族羣保護、邊界衝突

## 概述

Pleistoros 是達基亞神話中的戰神，為達基亞勇士在戰場上的精神支柱。他的名字僅在古希臘—羅馬文獻中的數處銘文中被紀錄——希臘地理學家斯特拉波在《地理學》中提及達基亞人「崇拜戰神 Pleistoros」，並描述達基亞戰士在戰鬥前以鮮血祭祀此神以獲得「戰鬥狂怒」。Pleistoros 的名字可能源自達基亞語中與「打擊」或「衝鋒」相關的詞根。不同於希臘的 Ares（暴力之戰）或雅典娜（戰略之戰）的區分——Pleistoros 同時涵蓋戰鬥的狂熱與策略智慧，類似於愛爾蘭的 Morrigan（戰爭與命運女神）或北歐的 Odin（智慧與戰死之神）。達基亞人在出征前於聖林中向 Pleistoros 獻祭動物（經常是公羊或戰馬），並由祭司在戰爭期間攜帶 Pleistoros 的聖物（一把裝飾著鷹羽的鐵劍或一面青銅盾牌）隨軍出征。達基亞人以其勇猛善戰著稱——羅馬人將達基亞戰爭（公元 87-106 年）視為帝國面臨的最艱苦挑戰之一，而 Pleistoros 的信仰正是達基亞戰士精神的核心支柱。

## 神話事蹟

**祭戰與戰前儀式：** 達基亞人對 Pleistoros 的祭祀極為隆重——戰前選定一人作為「Pleistoros 的使者」，此人身披全副武裝徒步走向敵陣，挑釁敵方最強者出戰。若使者戰死則被認為 Pleistoros 接受了祭品，達基亞全軍將帶有神聖的怒氣發動總攻。若使者得勝歸來，則象徵 Pleistoros 不願在此戰中流血——達基亞人將以其他方式（圍困或外交）解決衝突。此習俗與凱爾特神話中的單挑儀式（Champion's Single Combat）以及印度史詩《摩訶婆羅多》中的榮譽決鬥構成明顯平行。希羅多德在描述色雷斯人時記載：「他們以戰死為榮，以老死為恥——因為戰死者由 Pleistoros 引領前往英雄的永恆國度。」

**邊界的守護者：** Pleistoros 的雕像經常被放置在達基亞王國邊界的要塞與關隘處。考古學家在達基亞王都 Sarmizegetusa Regia 周圍的防禦工事中發現多處 Pleistoros 的祭祀祭壇——以石塊壘成圓形平台，中央插有鐵劍。達基亞守軍相信 Pleistoros 的劍鋒會切斷入侵者的攻勢。在羅馬征服達基亞後，羅馬軍團吸收了一部分 Pleistoros 的信仰——達基亞輔助士兵將 Pleistoros 與羅馬的 Mars（戰神）融合，稱之為「Mars Pleistoros」，此名出現在二世紀的數處軍事銘文中。

**戰後淨化與和解：** 與許多文化中的戰神不同，Pleistoros 也掌管戰後的和解儀式——戰鬥結束後雙方（即使為敵對部落）在 Pleistoros 的聖林中以同飲葡萄酒完成和解。此習俗呼應達基亞人相信「所有戰士死後都在 Zalmoxis 的盛宴中共飲」的宇宙觀——敵人與朋友在死亡面前沒有區別。達基亞人對戰死者的屍體不施以凌辱——他們相信 Pleistoros 已在死者的戰鬥中獲得了榮耀，不需要進一步的暴力。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Ares（阿瑞斯） | 戰爭之神、戰鬥狂怒、戰犬聖物 |
| 希臘神話 | Athena（雅典娜） | 戰略智慧、城市保護、勝利女神 |
| 羅馬神話 | Mars（馬爾斯） | 戰神、農業守護、建國先祖信仰 |
| 北歐神話 | Odin（奧丁） | 智慧之神、戰爭主導、英靈殿 |
| 凱爾特神話 | Morrigan（莫瑞甘） | 戰爭女神、命運、戰士保護 |
| 印度神話 | Kartikeya（迦絺吉夜） | 戰神、天軍統帥、智慧與戰略 |
| 日本神話 | 建御雷神（Takemikazuchi） | 戰神、刀劍之神、雷與勝利 |

## 相關神祇

- **Zalmoxis**（冥王）——戰死英雄的最終歸宿
- **Heros**（騎馬英雄神）——Pleistoros 的凡人表現面向
- **Gebeleizis**（雷神）——戰神的雷電力量
- **Kotys**（大地之母）——戰前祭祀在 Kotys 的聖林中進行
- **Dacian Mars**（羅馬化戰神）——融合後的名稱
