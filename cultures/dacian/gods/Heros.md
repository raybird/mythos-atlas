# Heros（赫羅斯）

- **文化：** 達基亞/羅馬尼亞神話 (Dacian/Romanian Mythology)
- **職掌：** 騎馬英雄神、獵人守護、冥界伴隨、王權象徵、邊境保護

## 概述

Heros（亦稱 Heros Equitans 或「色雷斯騎手」The Thracian Horseman）是達基亞與色雷斯地區最常見的神祇形象之一，以騎馬獵人的姿態出現在數以千計的考古遺物中——包括還願浮雕、墓碑、金銀飾品與聖所壁畫。他的形象標準化為一位蓄鬚的騎馬男子，右手持矛（或長棍）刺向跪伏在地的野豬、野兔或獅子，左手有時握著酒杯或火炬。Heros 的崇拜從達基亞王國到羅馬化的色雷斯（公元前 4 世紀至公元 4 世紀）跨越了超過七百年，顯示他在民眾信仰中的持久地位。Heros 不僅是戰士與獵人的守護神——他更承載了達基亞人的英雄理想：勇敢、榮譽、與死亡的和諧共處。羅馬時期 Heros 被等同於希臘的 Heracles 和羅馬的 Silvanus（森林神），但他的本質仍保留達基亞特色——他是凡人英雄死後的昇華，象徵人類可以通過榮譽的死亡進入 Zalmoxis 的永恆國度。

## 神話事蹟

**騎手獵人原型：** Heros 最常見的雕像是他騎著駿馬、持矛刺殺一頭野豬。野豬在達基亞神話中象徵著森林的混沌破壞力——每年秋季野豬從喀爾巴阡山區湧出，破壞田園與威脅村民。Heros 因此被視為文明對抗混沌的保護者。達基亞獵人在狩獵前向 Heros 獻祭葡萄酒與麵包——若祭祀被接受，Heros 會在夢中指示獵物位置。部分浮雕顯示 Heros 身旁伴隨著一條蛇或一隻狗——狗代表忠誠與追蹤，蛇代表冥界的智慧與守護。Heros 的馬匹同樣具有神聖性——他被描繪為一匹強壯的白色戰馬，象徵王權與勝利。

**英雄死後的昇華：** Heros 最重要的神話功能是作為人類英雄進入 Zalmoxis 冥界的示範。達基亞人相信勇敢的戰士或智慧的祭司在死後將轉變為 Heros——他們騎著戰馬進入 Zalmoxis 的永恆之宴。此信仰直接影響了達基亞的戰士文化：戰士在出征前宣誓「我將在 Zalmoxis 的盛宴中飲酒」——戰死沙場被視為至高榮譽。斯特拉波記載達基亞人「歡送他們的國王死亡」——國王通過死亡回歸 Zalmoxis，而 Heros 正是這條回歸之路的引導者。墓葬浮雕中 Heros 的頻繁出現（超過 3000 件已知浮雕）證明了此信仰在達基亞社會的普遍性。

**Heros 與羅馬化的融合：** 羅馬征服達基亞（公元 106 年）後，Heros 的崇拜未被消滅而是以「Interpretatio Romana」的方式延續——羅馬士兵將 Heros 等同於 Silvanus（森林神）、Heracles（英雄神）與 Mithras（密特拉/契約與太陽神）。在羅馬化時期的達基亞（今羅馬尼亞地區），Heros 的浮雕從未間斷——他成為文明與邊境防禦的象徵。部分 Heros 浮雕在基督教化後被改刻為聖喬治屠龍——這可能是歐洲最晚近的 Heros 崇拜轉化案例。羅馬尼亞民間傳說中的「Făt-Frumos」（英俊王子/英雄騎手）被學者認為是 Heros 神話在民間文學中的直接繼承者。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Heracles（赫拉克勒斯） | 英雄死後成神、十二試煉、普通人出身 |
| 羅馬神話 | Silvanus（西爾瓦努斯） | 森林守護、邊界保護、獵人守護 |
| 羅馬神話 | Mithras（密特拉） | 騎馬鬥牛、士兵信仰、密儀宗教 |
| 凱爾特神話 | Nuada（努阿達） | 英雄王權、戰神、不死傳說 |
| 波斯神話 | Mithra（密特拉） | 契約/光之神、騎馬、邊境保護者 |
| 北歐神話 | Einherjar（英靈戰士） | 死後在 Valhalla 繼續戰鬥慶宴 |
| 中國神話 | 關公 | 武將成神、死後受崇拜、戰神/保護者 |

## 相關神祇

- **Zalmoxis**（冥王）——英雄死後前往 Zalmoxis 的盛宴
- **Derzelas**（冥界守門）——Heros 協助 Derzelas 引導英雄靈魂
- **Pleistoros**（戰神）——Heros 的戰士面向
- **Kotys**（大地之母）——Heros 保護 Kotys 的森林與田野
- **Dacian Mars**（達基亞戰神）——Heros 在戰時的特殊顯現
