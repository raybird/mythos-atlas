# 達基亞 Căluşari 儀式與跨文化治療舞蹈比較 (The Căluşari Ritual: A Cross-Cultural Analysis of Healing Dances)

- **文化：** 達基亞/羅馬尼亞神話 (Dacian/Romanian Mythology)

## 概述

Căluşari（音近「克魯沙里」）是羅馬尼亞南部與保加利亞北部地區流傳超過兩千年的儀式性舞蹈傳統，為歐洲延續時間最長的民間儀式之一。此舞蹈由名為「Căluşari」的男性團體表演——他們身穿白色服裝、頭戴彩色絲帶、腳繫鈴鐺、手持木棍（象徵武器與馬匹），以高難度的跳躍與交叉步伐重現古代戰士的戰鬥與治療儀式。Căluşari 的起源可直接追溯至達基亞時代的戰士舞與 Kotys（大地之母/狂喜女神）的密儀祭祀——它以舞蹈的形式將達基亞宗教的核心元素（治療、戰鬥、狂喜、死亡與重生）保存至今。2008 年，Căluşari 傳統被聯合國教科文組織列為「人類非物質文化遺產」。本比較分析將 Căluşari 置於全球治療舞蹈的脈絡中，比較其與西伯利亞薩滿儀式、北美太陽舞、韓國巫堂舞、希臘狂喜祭典與非洲靈媒舞蹈的異同。

## 一、Căluşari 的歷史脈絡與神話根源

Căluşari 的名稱可能源自羅馬尼亞語的「cal」（馬），暗示其與達基亞騎馬戰士（Heros）的關聯。達基亞戰士在戰鬥前以舞蹈祭祀 Pleistoros（戰神）與 Kotys——舞者模仿騎馬衝鋒的步伐，以鈴鐺聲象徵馬匹奔騰。儀式中包含「治癒被 Căluşari（超自然女性精靈）附身的人」的環節——舞者以跳躍驅趕精靈，以木棍擊打地面引入大地母親（Kotys）的治療力。此治療功能與達基亞 Derzelas 的健康信仰密不可分。基督教化後，Căluşari 的祭祀對象被轉移至聖母瑪利亞與聖喬治，但其治療—舞蹈—狂喜的結構從未中斷。

Căluşari 的核心神話敘事：在一年中的特定時期（主要是復活節後到五旬節間），「Căluşari」（在空中飛舞的女性精靈）會從樹梢、洞穴與溪流中出現——她們具有極強的力量，可以「佔據」人類的身體，導致麻痺、失語或精神異常。Căluşari 舞者的使命即是透過舞蹈——以完美的節奏、跳躍與棍棒交叉動作——抵禦這些精靈，治癒被附身者，確保社群的健康與豐收。這個結構本質上是一場「象徵性戰爭」——舞者作為 Heros（騎馬英雄神）的代理人與混沌力量鬥爭。

## 二、與西伯利亞薩滿治療的比較

西伯利亞薩滿（Noaidi / Saman）與 Căluşari 共享以下特徵：

| 特徵 | Căluşari（達基亞） | 西伯利亞薩滿（通古斯/雅庫特） |
|------|---------------------|-----------------------------|
| **靈體附身** | Căluşari 精靈佔據病人 | 惡靈佔據病人身體 |
| **薩滿/治療者** | Căluşari 舞團（七/九人） | 薩滿（個人，通常由靈召喚） |
| **治療工具** | 木棍、鈴鐺、口哨 | 鼓、腰鈴、神杖 |
| **治療方式** | 舞蹈 + 跳躍 + 棍擊 | 鼓聲 + 吟唱 + 靈魂旅程 |
| **狂喜狀態** | 舞者進入出神（trance） | 薩滿進入昏迷（ecstasy） |
| **靈界旅行** | 象徵性：舞者「飛躍」 | 實際：薩滿靈魂前往靈界 |
| **社會功能** | 保護社群、治療疾病 | 治療、占卜、祭祀 |

關鍵差異在於：Căluşari 的治療是集體性的（整團舞者象徵社群對抗混沌），而西伯利亞薩滿是個體性的。Căluşari 的舞者不以「被靈附身」進行治療——他們以舞蹈技術與紀律「指揮」靈力，而非被動接受附身。此外，Căluşari 的木棍戰舞具有更強的戰鬥象徵，體現達基亞戰士文化對「征服疾病」的隱喻。

## 三、與韓國巫堂（Mudang）舞的比較

韓國巫堂（Mudang 무당）與 Căluşari 均屬於東亞—東歐非基督教化民間宗教的殘留，二者具有驚人的結構平行：

| 特徵 | Căluşari（達基亞） | 韓國巫堂（Mudang） |
|------|---------------------|-------------------|
| **參與者** | 男性 | 多為女性（巫女） |
| **儀式時間** | 春季（復活節—五旬節） | 各種節氣與特定日期 |
| **舞蹈特徵** | 跳躍、旋轉、交叉步伐 | 緩慢旋轉、抖動、蹲起 |
| **音樂** | 笛子（fluier）+ 鼓、鈴鐺 | 鼓（janggu）、鑼、笛子 |
| **治療功能** | 驅逐 Căluşari 精靈 | 治療靈魂疾病、引導亡魂 |
| **靈界旅程** | 集體防禦性舞蹈 | 巫堂個人附身 + 旅程 |
| **顏色象徵** | 紅、白、藍（達基亞三色） | 五方色（青、赤、黃、白、黑） |
| **神祇關聯** | Kotys / Derzelas 治療面向 | 巴里公主（Bari）冥界女神 |

Căluşari 的春季週期性與韓國巫堂的年度祭祀（gut）共享「時間是循環的」宇宙觀。兩者治療疾病的方式均不仰賴藥草——而是透過舞蹈調整人體的「能量節奏」，使病人重新與宇宙的和諧步伐同步。Căluşari 的「排除精靈」邏輯與巫堂的「安撫精靈」邏輯形成了「對抗 vs 協商」的對比——前者是戰士式的（達基亞文化的遺產），後者是母系式的（韓國無母系薩滿傳統的遺產）。

## 四、與希臘 Dionysus 狂喜祭典的比較

Căluşari 在古典時期即被希臘—羅馬觀察者指出與 Dionysus（狄奧尼索斯）的狂喜祭典極為相似：

| 特徵 | Căluşari | Dionysus 狂喜祭（Maenads / Bacchae） |
|------|----------|--------------------------------------|
| **出神狀態** | 有紀律的集體出神 | 個體失控式狂喜 |
| **性別** | 男性舞團 | 多為女性（Maenads） |
| **動物象徵** | 馬、蛇 | 山羊、豹、蛇 |
| **武器** | 木棍 | Thyrsus（茴香杖）、撕裂動物 |
| **音樂** | 笛子 + 鼓 | 雙管笛（aulos）+ 鼓 |
| **意識狀態** | 高度控制的技術性舞蹈 | 迷幻醉態 + 山區奔跑 |
| **治療目標** | 他人（病人康復） | 自我（靈魂解放與 catharsis） |

Căluşari 與 Dionysus 祭典共享「透過節奏與運動解除日常意識」的技術——但 Căluşari 的紀律性更接近一種「戰士瑜伽」，而非 Dionysus 式的迷狂。學者（如 Mircea Eliade）認為 Kotys 的崇拜是 Dionysus 在巴爾幹半島的前身之一——達基亞/色雷斯的狂喜元素可能通過 Kotys 的崇拜影響了希臘 Dionysus 宗教的形成。

## 五、Căluşari 的當代意義與傳承

Căluşari 的生存至今超過兩千年——它跨越了達基亞王國、羅馬帝國、中世紀、共產主義與現代化。它的持續存在有幾個原因：(1) 治療功能提供實際社會價值——在缺乏正式醫療的傳統農村，Căluşari 是唯一可得的「醫療服務」。(2) 社區凝聚——Căluşari 舞團的組織方式為村莊提供了年度性的社交與合作契機。(3) 文化的韌性——由於表演不依賴文字而是身體記憶與口傳，Căluşari 能夠在不同的政治與宗教環境中靈活轉化。21 世紀的 Căluşari 已經從純粹的醫療儀式轉變為文化展演——但在羅馬尼亞南部的 Oltenia 地區，仍有部分 Căluşari 舞團被村民秘密邀請進行真正的治療儀式。Căluşari 的持續存在證明達基亞宗教並非只是考古學上的化石——它作為活的傳統，仍在喀爾巴阡山腳下的村莊中呼吸與跳動。

## 六、結論

Căluşari 達基亞治療舞蹈是歐洲本土宗教傳統中「活的化石」。它不僅保存了達基亞戰士文化的動作語彙——它以舞蹈為媒介，將達基亞的宇宙觀（秩序 vs 混沌、集體 vs 個人、信仰 vs 治療）在每一次表演中重新激活。透過與西伯利亞薩滿、韓國巫堂與希臘 Dionysus 祭典的比較，我們可以看到 Căluşari 屬於一個更廣泛的「節奏性治療舞蹈」家族——它們共享透過身體運動改變意識狀態以對抗疾病的核心機制，而 Căluşari 以長達兩千年的連續傳承，成為這個家族中最古老的活躍成員之一。
