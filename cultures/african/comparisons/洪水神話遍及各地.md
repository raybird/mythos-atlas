# 洪水神話遍及各地 — 跨文化比較

## 比較：非洲諸神話 vs 其他文化

### 非洲洪水神話概述

非洲大陸幅員廣大，包含數百個語言文化群體，洪水神話在不同區域呈現多樣面貌：

1. **約魯巴（Yoruba）**：海洋女神 Yemaya 因人類驕傲而降怒，以巨浪淹沒大地。Obatala 奉 Olorun 之命以金鍊垂降，以白色祭品（山羊、布匹）說服 Yemaya 退水，象徵人類與自然力量的契約。
2. **祖魯（Zulu）**：宇宙巨蛇（Inkanyamba）引發洪水懲罰人類的不敬行為。祖魯巫醫（Sangoma）透過祖先通靈儀式探知洪水消退時機，保留旱季與雨季更替的集體記憶。
3. **多貢（Dogon）**：多貢天文知識中保存了天狼星 B 的運行週期與對應的地球洪水週期——Nommo（水靈）從天界降臨帶來秩序，洪水消退後文明建立。
4. **東非班圖（Bantu）**：洪水為祖先的憤怒——人類違反禁忌（如亂倫、不祭祀），降雨之神降下不間斷的豪雨，唯有透過頭人的祈雨儀式方可停止。
5. **阿散蒂（Ashanti）**：天神 Nyame 以洪水隔離人類與天界——天梯因此斷裂，人神從此分離。
