# Olorun / Olódùmarè

- **文化：** 非洲諸神話 (African Mythologies) — 約魯巴 (Yoruba)
- **職掌：** 至高神、天空之主、萬物的創造者與維持者

## 概述

Olorun（字面意為「天空之主」）或 Olódùmarè（「至高全能者」）是約魯巴神話中的至尊神祇，居於天空（Orun），掌管宇宙萬物的運行。不同於希臘或北歐神話中積極介入人間事務的天神，Olorun 將管理物質世界的權力委託給眾多 Orishas（神靈），自身保持超越與超然的地位。約魯巴人認為 Olorun 是全知全能卻不可直接崇拜的存在，人類不建造專門祭祀 Olorun 的神廟，而是通過 Orishas 向其祈禱。此一神學特徵與非洲多神信仰中「高天上帝遠離人間」的普遍母題完全一致。

## 神話事蹟

在約魯巴創世神話中，世界最初只有天空與汪洋。Olorun 命 Obatala 帶領眾神以金鍊垂降至原水之上，用沙土與白雞創造陸地。Obatala 醉酒後造出殘疾人，Olorun 因此禁止他飲酒，並由 Oduduwa 完成陸地創造。Olorun 亦派遣 Eshu（信使神）作為人間與天界的溝通橋樑。每個人死後靈魂返回 Orun，接受 Olorun 的審判，善者歸於祖先行列，惡者遭受懲罰。在 Ifá 占卜系統中，所有 Odu（經文）最終皆指向 Olorun 的至高權威。

## 跨文化對應

- **昊天上帝（中國）**：同為至高神而鮮少直接干預人間
- **El／埃爾（迦南）**：退居幕後的眾神之父
- **Anu／安（蘇美）**：遠居天界的天神，權力委託予下級神
- **Olodumare → 基督教上帝**：在奴隸貿易時期，約魯巴人將 Olorun 與基督教上帝概念同化，形成巴西 Candomblé 與古巴 Santería 中的 Supreme God 觀念

## 相關神祇

- Obatala（受命造地的純潔之神）
- Oduduwa（約魯巴人的先祖與陸地完成者）
- Eshu（信使，溝通人神）

## 出現在
