# Saulė (紹萊)

- **文化：** 波羅的神話 (Baltic (Lithuanian/Latvian) Mythology)
- **職掌：** 太陽女神、生命之源、豐收

## 概述

Saulė（立陶宛語）／Saule（拉脫維亞語）是波羅的海神話的太陽女神，而非太陽神——這在印歐神話中較為特殊（多數印歐傳統以男性太陽神為主，如希臘 Helios、印度 Sūrya）。Saulė 以金色馬車（或戰車）每日穿越天際，傍晚落入波羅的海，乘船沿海底回到東方。她以美麗、慷慨與溫暖著稱，是萬物生命的源泉。Saulė 被描繪為金冠、金袍的貴婦，手中握有金色蘋果與玫瑰花。她的馬車由 Ašvieniai（神馬雙子）或兩匹白馬拉行。在拉脫維亞 Dainas 民謠中，Saulė 是最常被提及的神祇之一——她的名字出現頻率僅次於 Laima。Saulė 的女兒們稱為 Saules meitas（太陽少女），常在民謠中與人類互動。太陽被視為神聖不可直視的存在，人們以日出日落為日常儀式的節奏。

## 神話事蹟

Saulė 的神話核心在於她與 Mēness（月神）的婚姻關係。Saulė 嫁給 Mēness，但 Mēness 不忠，愛上 Aušrinė（晨星／黎明女神）。Perkūnas 基於正義制裁 Mēness，以雷斧劈裂其臉——自此人類永見月亮的傷疤。Saulė 與 Mēness 離異後獨自運行天際。另一版本：Saulė 與 Mēness 的女兒 Žemyna 化身為地母神。Saulė 的婚禮是波羅的海文學重要的主題——在立陶宛民謠中，Saulė 嫁給 Mēness 的盛大婚禮由宇宙萬物參與，橡樹、花草、鳥獸皆為賓客。Saulė 的日常旅程組成世界秩序：她早上從東方的「太陽之門」升起，正午在高天照耀，傍晚穿過「海之門」進入海底通道，沿九海（Jūra）返回東方。在冬至（Kūčios）與夏至（Rasa／Joninės）時，波羅的海民族舉行盛大節日以迎接或告別 Saulė。她在冬至後重生，象徵光明的回歸。Saulė 也保護孤兒——民間歌謠中孤兒向 Saulė 訴苦，她以金線織衣溫暖他們。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Helios（赫利奧斯） | 太陽馬車穿越天際、每日循環 |
| 埃及神話 | Ra（拉） | 太陽船晝夜航行、傍晚入冥界 |
| 中國神話 | 羲和 | 駕六龍，御太陽 |
| 日本神話 | 天照大神（Amaterasu） | 太陽女神、光明之源 |
| 印度神話 | Sūrya（蘇利耶） | 太陽戰車、驅散黑暗 |
| 北歐神話 | Sól（太陽女神） | 少數女性太陽神之一 |

## 相關神祇

- **Mēness**（月神）——Saulė 的前夫
- **Perkūnas**（雷神）——Saulė 的護衛與正義執行者
- **Ašvieniai**（神馬雙子）——拉行 Saulė 馬車的天馬
- **Aušrinė**（晨星女神）——Mēness 的外遇對象
- **Vakarinė**（暮星）——Saulė 的侍女或女兒
- **Žemyna**（地母神）——Saulė 的女兒
