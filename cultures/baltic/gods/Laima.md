# Laima (萊瑪)

- **文化：** 波羅的神話 (Baltic (Lithuanian/Latvian) Mythology)
- **職掌：** 命運女神、生育、婚姻、好運

## 概述

Laima（立陶宛語）／Laime（拉脫維亞語，意為「幸福」）是波羅的海神話中最重要的命運女神，掌管人類從出生到死亡的一切關鍵轉折——出生、命名、婚姻與死亡。她的名字直接意為「好運」或「命運」。Laima 被描繪為年輕的婦女，有時與另一位命運女神 Laimė（立陶宛）或 Kārta（拉脫維亞）並列，類似希臘神話的 Moirai（命運三女神）或北歐神話的 Norns。在拉脫維亞 Dainas 民謠中，Laima 是出現頻率最高的神祇，超過 Perkūnas 與 Saulė。民間信仰中 Laima 在嬰兒出生時到訪，決定其一生的命運——富貴、貧窮、健康、疾病、婚姻對象。她在立陶宛傳統中被視為「設定道路者」（kelių tiesėja）。Laima 的象徵物包括亞麻布、紡車與織布機（命運之線），以及樺樹與菩提樹。她的節日與生育和婚禮相關，新娘在婚禮上向 Laima 祈求祝福。

## 神話事蹟

Laima 最顯著的神話角色是作為新生兒命運的決定者。在出生後第三天（或第七天），Laima 降臨，在搖籃旁決定嬰兒的一生。如果 Laima 歡喜，嬰兒一生幸福；若她不悅，則災禍連連。人類不得窺視 Laima 的決定。第二個神話：Laima 與其他女神的競賽——在拉脫維亞民謠中，Laima 與 Dēkla（行為女神）和 Kārta（命運分配者）共同決定人類命運。她們有時意見不一致，象徵命運的不可預測性。第三，Laima 是婚姻的保護者。新娘在婚禮前需獲 Laima 的祝福——否則婚姻將不幸。婚禮儀式中新娘繞行 Laima 的聖樹（樺樹或菩提樹）三圈祈求好運。Laima 也介入神話中的婚姻：當 Saulė 嫁給 Mēness 時，Laima 主持婚禮，但其後 Mēness 不忠，Laima 也無力阻止——顯示命運女神雖預見一切但無法改變既定的軌跡。Laima 與地母神 Žemyna 關係密切——二者共同守護婦女生育與農業豐收。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Moirai（命運三女神） | 紡織命運之線 |
| 北歐神話 | Norns（諾恩三女神） | 決定諸神與人類命運 |
| 羅馬神話 | Parcae（巴爾開） | 出生、婚姻、死亡三階段 |
| 斯拉夫神話 | Rozhanitsy（生育女神們） | 出生時決定命運 |
| 印度神話 | Lakṣmī（吉祥天女） | 好運、幸福、豐盛 |
| 日本神話 | 弁才天 | 幸運、音樂、水邊女神 |

## 相關神祇

- **Dēkla**（行為女神）——Laima 的姊妹，掌管行爲與德性
- **Kārta**（命運分配者）——Laima 的助手，分配具體命運
- **Žemyna**（地母神）——共同守護生育與豐收
- **Saulė**（太陽女神）——Laima 參與其婚禮
- **Māra**（身體守護神）——與 Laima 共同保護婦女
