# Žemyna (熱米娜)

- **文化：** 波羅的神話 (Baltic (Lithuanian/Latvian) Mythology)
- **職掌：** 地母神、豐收、生育、大地

## 概述

Žemyna（立陶宛語）／Zemes māte（拉脫維亞語「大地之母」）是波羅的海神話的地母神，象徵大地作為生命之源的永恆女性力量。她的名字來自立陶宛語 žemė（土地），與斯拉夫神話的 Mat Zemlya 同源。Žemyna 被視為孕育所有植物、動物與人類的母體。她在波羅的海三重宇宙結構中代表地界（下界），與天父 Dievas（天界）和冥神 Velnias（冥界）共同構成宇宙的三層結構。Žemyna 的祭儀以農業週期為核心——春耕、秋收與冬藏皆向她獻祭。立陶宛農民在耕作前將第一把種子獻給 Žemyna，並祈禱豐收。她的聖物為麵包——波羅的海民族對麵包的敬意極為深厚，麵包掉在地上需撿起親吻，因為那是 Žemyna 的賜予。蛇（žaltys）是 Žemyna 的聖獸，被視為家中守護靈，不得傷害。

## 神話事蹟

Žemyna 的神話集中於她作為母親與養育者的角色。她是 Saulė（太陽女神）的女兒，溫暖的陽光讓大地孕育生命。在創世神話中，Dievas 創造大地後，Žemyna 賦予大地生命力——她以綠色覆蓋大地，讓樹木生長、河流流淌。人類從大地中獲得食物、住所與藥材。第二，Žemyna 與 Laima 合作守護婦女生育——人類的誕生由 Laima 決定命運、Žemyna 提供肉體，二者缺一不可。在婚禮儀式中，新娘觸碰大地以祈求生育能力。第三，Žemyna 為死者接納者——人類死後回歸她的懷抱。立陶宛葬禮習俗中，遺體需直接接觸地面或覆以故鄉的泥土，象徵回歸 Žemyna。部分 Dainas 民謠描述魂魄在 Žemyna 的子宮中等待轉世或進入永恆安息。Žemyna 也維護真理——立陶宛人在爭執時手握泥土立誓，相信 Žemyna 會懲罰偽誓者。她的殿是開闊的自然——森林、田野與河岸，而非人造建築。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Gaia（蓋亞） | 大地之母、萬物之源 |
| 羅馬神話 | Tellus / Terra Mater | 大地腹地、豐收女神 |
| 埃及神話 | Geb（蓋布） | 大地之神（互補性別） |
| 中國神話 | 后土 | 大地女神、社稷 |
| 印度神話 | Bhūmi / Pṛthvī | 地母、萬物承載者 |
| 印加神話 | Pachamama | 大地之母、農業祭儀 |
| 斯拉夫神話 | Mat Zemlya 濕土之母 | 大地為母、立誓傳統 |

## 相關神祇

- **Saulė**（太陽女神）——Žemyna 的母親，陽光使大地生長
- **Dievas**（天父）——Žemyna 的創造者或配偶
- **Perkūnas**（雷神）——以雷雨滋潤大地
- **Laima**（命運女神）——與 Žemyna 共同守護生育
- **Māra**（身體守護神）——Žemyna 的助手之一
- **Velnias**（冥神）——地下世界的主宰
