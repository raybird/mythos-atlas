# Dievas (迪耶瓦斯)

- **文化：** 波羅的神話 (Baltic (Lithuanian/Latvian) Mythology)
- **職掌：** 天父、造物主、宇宙秩序

## 概述

Dievas（立陶宛語）／Dievs（拉脫維亞語、古普魯士語 Deyvs）是波羅的海神話的至高天父與創造之神。他的名字直接源自原始印歐語 \*Dyēus（「光明之天」），與希臘 Zeus、羅馬 Jupiter（< \*Dyēus ph₂tēr）、北歐 Týr、印度 Dyáuṣ Pitṛ́ 完全同源，屬於印歐民族最古老的神祇層級。Dievas 被視為宇宙的創造者與秩序的維持者，居住在遙遠的「天國」，以金冠與白袍的形象出現。他不同於 Perkūnas 的暴烈——Dievas 代表冷靜、睿智與父親般的慈悲。在波羅的海民間傳統中，Dievas 的意志透過自然顯現，人類的行為需符合 Dievas 所立的道德法則。Dievas 的象徵物包括橡樹、聖火與白山。他的名字在立陶宛語中直接成為「神」的通用詞（dievas）。

## 神話事蹟

Dievas 最核心的神話是與 Velnias 共同創造世界。最初只有無邊汪洋——Dievas 與 Velnias 為兄弟，Dievas 命 Velnias 潛入海底取泥。Velnias 取回泥土，Dievas 將其鋪在水面上形成大地。但 Velnias 口中藏泥，吐出後形成山脈、湖泊與石頭。這個「大地潛水者」（Earth-Diver）母題普遍存在於中亞、北歐與北美原住民神話中，顯示波羅的海神話與北亞薩滿傳統的深層聯繫。人類由 Dievas 行走時滴落的唾液或水滴形成——這與北歐神話中奧丁三兄弟以木頭造人的故事有相似結構。Dievas 也在民間故事中以白鬚老人形象下凡，考驗人類的道德。當人類違背 Dievas 的誡命時，他會授權 Perkūnas 以雷電懲罰。在波羅的海三重奏神系中，Dievas 象徵天父的靜態智慧，Perkūnas 代表動態力量，而 Velnias 代表混沌與物質世界。

## 跨文化對應

| 文化 | 對應神祇 | 共通點 |
|------|----------|--------|
| 希臘神話 | Zeus（宙斯） | 同源 \*Dyēus、天父、眾神之王 |
| 羅馬神話 | Jupiter（朱比特） | 同源 \*Dyēus ph₂tēr |
| 北歐神話 | Týr（提爾） | 同源 \*Dyēus、正義之神 |
| 印度神話 | Dyáuṣ Pitṛ́（天父） | 同源、吠陀天父 |
| 斯拉夫神話 | Svarog（斯瓦羅格） | 天火、造物主功能對應 |
| 中國神話 | 昊天上帝 | 至高天神、道德秩序 |

## 相關神祇

- **Perkūnas**（雷神）——Dievas 的力量執行者
- **Velnias**（冥神）——Dievas 的兄弟與對手
- **Saulė**（太陽女神）——Dievas 的女兒或創造物
- **Laima**（命運女神）——在 Dievas 之下決定人類命運
- **Žemyna**（地母神）——Dievas 的地面對應
