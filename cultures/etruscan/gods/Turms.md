# 圖姆斯 (Turms)

- **文化：** 伊特魯里亞神話 (Etruscan Mythology)
- **職掌：** 神使、亡靈引導者、商業與旅行之神、界域穿越者

## 概述

Turms（圖姆斯）是伊特魯里亞萬神殿中的神使與邊界穿越者，對應希臘的 Hermes 與羅馬的 Mercurius。Turms 在伊特魯里亞藝術中以年輕人形象出現——通常戴著帶翼的帽子（petasos）或頭盔、手持 caduceus（雙蛇杖）、腳踝有翼。Turms 的角色比希臘對應者更側重於引導死者前往冥界——他是三大冥界引導者（與 Charun、Vanth 並列）之一，在伊特魯里亞墓穴壁畫中頻繁出現，伴隨死者靈魂前往阿伊塔（Aita）與佩爾西普奈（Phersipnai）的國度。

## 神話事蹟

Turms 在伊特魯里亞神話中的核心職責是 psychopompos（靈魂引導者）。在塔奎尼亞（Tarquinia）與武爾奇（Vulci）等地的墓穴壁畫中，Turms 被描繪為帶領一列靈魂前往冥界的引路人。與希臘藝術中短暫陪伴死者的 Hermes 不同，伊特魯里亞的 Turms 全程參與死後旅程——他手持火炬（而非 caduceus）照亮通往冥界的黑暗路徑。

Turms 也出現在特洛伊戰爭相關的伊特魯里亞壁畫中——作為仲裁者與訊息傳遞者。伊特魯里亞人將 Turms 視為凡人之間溝通的中介者，特別是涉外貿易（伊特魯里亞是地中海主要的貿易強權）中，商人會向 Turms 祈禱以確保路途中順利。

Turms 在伊特魯里亞宗教中的特殊地位體現在他被認為是向人類揭示命運的使者——通過夢境或自然徵兆，Turms 將未來事件的信息從神界帶到人間。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Hermes（神使、亡靈引導者、盜賊之神） | 帶翼帽與蛇杖、邊界穿越、商業守護 |
| 羅馬 | Mercurius（Mercury，商業之神與神使） | 從 Etruscan Turms 繼承大量 iconography |
| 埃及 | Anubis（胡狼頭神，木乃伊化與引導亡靈） | 兩者皆為冥界引導者，但 Hermes/Turms 另有商業面向 |
| 北歐 | Odin（神王，但也引導戰死者前往瓦爾哈拉） | 旅途中的智慧之神、邊界穿越者 |
| 印度 | Nārada（天界使者、神與人間溝通者） | 神界到人間的訊息傳遞，神的使者 |
| 日本 | 惠比壽（Ebisu，商業與福神） | 商業守護神的面向 |

## 相關神祇

- 冥界同事：Charun（冥界守門人）、Vanth（冥界女神，攜帶火炬與卷軸）
- 冥界君主：Aita（冥界之王）與 Phersipnai（冥后）
- 父親／上級：Tinia（眾神之王，授予 Turms 使者職權）
- 對應者：羅馬人將 Turms 完全等同於 Mercurius，但 Etruscan 的 Turms 帶有更強烈的冥界色彩
- 聖物：Caduceus（雙蛇杖）與火炬（冥界版本）
- 動物：公羊（Turms 在部分描述的象徵動物）
