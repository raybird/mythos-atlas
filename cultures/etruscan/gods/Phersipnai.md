# 佩爾西普奈 (Phersipnai / Persipnei)

- **文化：** 伊特魯里亞神話 (Etruscan Mythology)
- **職掌：** 冥后、地下世界的女主人、春季與重生的象徵

## 概述

Phersipnai（也作 Persipnei）是伊特魯里亞神話中的冥后，對應希臘的 Persephone 與羅馬的 Proserpina。她的名字顯然是希臘 Persephone 的伊特魯里亞化形式。Phersipnai 在伊特魯里亞信仰中被描述為 Aita 的妻子與冥界的共同統治者。她一年之中部分時間生活在地下世界，部分時間返回人間——此季節循環對應了農作物的生長週期。Phersipnai 在伊特魯里亞藝術中通常頭戴冠冕或面紗，手持石榴或卷軸。

## 神話事蹟

Phersipnai 的故事在伊特魯里亞版本中與希臘版本有重要差異：在希臘神話中，Persephone 是被 Hades 強行綁架的受害者；而在伊特魯里亞版本中，Phersipnai 與 Aita 的婚姻被表現為一種神聖的契約——Phersipnai 自願在冬季返回地下，因為她知道她的回歸（春季）將使大地恢復生機。

伊特魯里亞墓穴壁畫（如塔奎尼亞的 Tomb of the Orcus）描繪了 Phersipnai 與 Aita 並肩坐在王座上迎接靈魂的場景——她手持一個石榴或蛋（象徵生命與再生），面容平靜而非哀傷。此圖像與希臘 Persephone 從冥界返回時欣喜的表情形成對比：伊特魯里亞的 Phersipnai 是一位接受自己雙重身份的女神。

在伊特魯里亞的葬禮儀式中，祭司會吟誦 Phersipnai 的讚美詩以祈求死者平安抵達冥界。Phersipnai 被相信是唯一可以說服 Aita 允許靈魂偶爾返回人間進行短暫拜訪的女神——因此伊特魯里亞人在葬禮上向她獻祭黑羊或紅花（象徵經由紅色血液與白色靈性的平衡）。

## 跨文化對應

| 文化 | 對應神祇 | 共享特徵 |
|------|---------|---------|
| 希臘 | Persephone（冥后，Demeter 之女） | 冥界婚姻、季節循環、石榴象徵 |
| 羅馬 | Proserpina（豐收與冥后） | 從希臘 Persephone 與 Etruscan Phersipnai 融合而成 |
| 美索不達米亞 | Ereshkigal（冥界女王） | 地下世界的女性統治者、不可逆轉的死亡領域 |
| 埃及 | Isis（復活與魔法女神，Osiris 之妻） | 穿越生死邊界的女神、死者保護者 |
| 日本 | 伊邪那美（黃泉津大神） | 配偶為死亡世界的共同統治者、死亡與生育的雙重象徵 |
| 印度 | Pārvatī（雪山神女，濕婆之妻） | 既是溫柔的妻子也是毀滅的女神——雙重性與 Phersipnai 類似 |

## 相關神祇

- 配偶：Aita（冥界之王，與 Phersipnai 共同統治 Aiz）
- 下屬：Vanth（冥界女神，可能為 Phersipnai 的使者）
- 象徵物：石榴（維持冥界的聯繫）、紅花（獻祭）、蛋（重生）
- 節日：伊特魯里亞的春季節日慶祝 Phersipnai 從冥界返回人間
- 羅馬對應：Proserpina 的崇拜與 ludi saeculares（百年節）有強烈的 Phersipnai 遺留
