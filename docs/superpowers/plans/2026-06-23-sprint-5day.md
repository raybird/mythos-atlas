1. **Day 1–2**：波斯 → 約魯巴 → 薩米 → 蒙古 → 高加索（5 個 7–8 頁文化 → 12–14 頁）
2. **Day 3–4**：越南 → 圖皮-瓜拉尼 → 腓尼基 → 菲律賓 → 印加 → 巴斯克（6 個 9–10 頁文化 → 12–14 頁）
3. **Day 5**：薩米 → 蒙古 → 高加索 → 馬普切 → 韓國 → 日本（6 個 8–11 頁文化 → 12–14 頁）
